# 機能 — フルツアー

ようこそ。cMindは非常に多くの機能を1つのアプリに凝縮しています。これがマップです。各機能は独自のディープダイブドキュメントを持っています — 知りたいことをクリックしてください。

## コピートレーディング

皇冠。マスターアカウントを多数にミラーし、インターネットが不安定でも同期を維持します。

- **[コピートレーディング](./copy-trading.md)** — コア：ミラーリング、order types、SL/TP、スリッページ、非同期/再同期。
- **[実行透明性](./copy-execution-transparency.md)** — 何がいつ、なぜコピーされたかを正確に確認。
- **[パフォーマンス手数料](./copy-performance-fees.md)** — あなたのシグナルに対してウォーターマークスタイルの有料。
- **[プロバイダーマーケットプレイス](./copy-provider-marketplace.md)** — トレーダーがプロバイダーを発見してフォロー。
- **[通知](./copy-notifications.md)** — 必要なものがあるときに通知を受ける。
- **[AIコピーレコメンダー](./ai-copy-recommender.md)** — AIに誰をコピーするかを提案させる。
- **[Open APIトークンライフサイクル](./token-lifecycle.md)** — cMindがcIDごとに正確に1つの有効なトークンを維持する方法。

## ホームベース

- **[ダッシュボード](./dashboard.md)** — ライブ、モバイルファーストのコマンドセンター：スパークライン付きKPI、アクティビティチャート、ステータスリング、ライブフィード、（管理者向け）クラスター健康。自動更新。

## AIコア

サイドにボルトオンされたチャットボックスではありません — 実際に*作業を行う*AI。

- **[AIアシスタント、エージェント、リスクガード&アラート](./ai.md)** — 戦略生成、セルフ修復ビルド、背景リスクガード（ボットの自動停止が可能）、スマートアラート。

## ビルド＆実行

- **[cBotのビルド＆バックテスト](./build-and-backtest.md)** — ブラウザ内Monaco IDE、C#/Pythonテンプレート、沙箱ビルド、ライブエクイティカーブ。
- **[MCPサーバー](./mcp.md)** — HTTP + SSEでcMindツールを露出してAIクライアントが駆動できるように。

## ビジネスとして運営

- **[ホワイトラベル/ブランディング](./white-label.md)** — 設定ですべての表面を再ブランド化。
- **[Prop-firmチャレンジシミュレーション](./prop-firm.md)** — ライブエクイティでデイリー損失、ドローダウン、ターゲットルールを強制。
- **[フィーチャートグル](./feature-toggles.md)** — 各デプロイメント/テナントに表示するものを決定。
- **[コンプライアンス/法務](./compliance.md)** — 監査証跡と法的表面。

## 体験

- **[インストール可能なアプリ（PWA）](./pwa.md)** — モバイルファースト、オフラインシェル、追加-to-home-screen。
- **[UIデザインシステム＆モバイルファースト](../ui-guidelines.md)** — 外観の背後にあるデザイントークンとルール。

## 舞台裏

一切を実行し続ける運用ビット：

- **[ノードフリート＆ディスカバリー](../operations/node-discovery.md)** — ノードが自己登録して回復する方法。
- **[水平スケール](../deployment/scaling.md)** — レプリカを追加、外部コーディネーター不要。
- **[ログ記録＆監査](../operations/logging.md)** — 構造化ログ + OpenTelemetry。
- **[デプロイメント](../deployment/local.md)** — どこでも実行。
