# Tính năng Mở rộng

cMind mở rộng theo từng tầng:

- **Stateless (Web + MCP):** tự động chia tỷ lệ sau load; không có trạng thái.
- **Database:** Postgres được quản lý, tuning và sao lưu.
- **Nodes:** Thêm agent HTTP hoặc tăng concurrency backtest.

Xem hướng dẫn chi tiết để biết các điểm chặn hiệu năng và cách tuning.
