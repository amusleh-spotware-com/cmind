# الميزات — الجولة الكاملة

مرحباً بك في الجولة الكبرى. cMind يحزم *الكثير* في تطبيق واحد، لذا إليك الخريطة. كل قدرة لديها الخاصة بها وثيقة dive عميقة - اضغط على أي حك كنت الخدش.

## نسخ التداول

الجوهرة التاج. مرآة حساب رئيسي على العديد، وإبقاء عليهم متزامنة حتى عندما يتصرف الإنترنت بسوء.

- **[نسخ التداول](./copy-trading.md)** — الأساس: المرايا، أنواع الأوامر، SL/TP، الانزلاق، desync/resync.
- **[شفافية التنفيذ](./copy-execution-transparency.md)** — انظر بالضبط ما تم نسخه، متى، ولماذا.
- **[رسوم الأداء](./copy-performance-fees.md)** — رسوم إشارتك، نمط أعلى علامة مائية.
- **[سوق الموفر](./copy-provider-marketplace.md)** — دع المتاجرين اكتشف واتبع الموفرين.
- **[الإخطارات](./copy-notifications.md)** — أخبر عندما يحتاج شيء إلى.
- **[موصي نسخ AI](./ai-copy-recommender.md)** — دع AI يقترح من نسخ.
- **[Open API token lifecycle](./token-lifecycle.md)** — كيف cMind تبقي بالضبط رمز واحد صحيح لكل cID.

## لوحة التحكم

- **[لوحة التحكم](./dashboard.md)** — المركز القيادي المباشر والهاتف أولاً: KPIs مع sparklines وخريطة نشاط وحلقة الحالة وتغذية حية و (للمسؤولين) صحة الكتلة. يحدث نفسه.

## نواة AI

ليس صندوق دردشة مرتبط على الجانب - AI الذي يفعل فعلاً *العمل*.

- **[مساعد AI وعميل ووكيل حماية المخاطر والتنبيهات](./ai.md)** — توليد الإستراتيجية والبناء ذاتي الإصلاح وحارس مخاطر خلفية يمكنه إيقاف الروبوتات وتنبيهات ذكية.

## البناء والتشغيل

- **[بناء واختبار cBots](./build-and-backtest.md)** — محرر Monaco في المتصفح وقوالب C#/Python والبناء في الرمل ومنحنيات الإنصاف الحية.
- **[خادم MCP](./mcp.md)** — عرض أدوات cMind على HTTP + SSE حتى عملاء AI يمكنهم قيادتها.

## تشغيله كعمل

- **[White-label / Branding](./white-label.md)** — إعادة تسمية كل سطح عبر الإعدادات.
- **[محاكاة تحدي Prop-firm](./prop-firm.md)** — فرض قواعس الخسارة اليومية والانخفاض والهدف مع الإنصاف الحي.
- **[مفاتيح الميزات](./feature-toggles.md)** — قرر ما يرى كل نشر/عميل.

- **[نظام تصميم UI والهاتف أولاً](../ui-guidelines.md)** — رموز التصميم والقواعس خلف المظهر.
