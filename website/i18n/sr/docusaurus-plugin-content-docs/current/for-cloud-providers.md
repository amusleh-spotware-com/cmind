# cMind за облачне и VPS добављаче 🖥️

Већ даш рачуни. cMind је спремна, отворена-кода производ који могаш da обавијеш тај рачуни
око: **нуди управљано cMind смештање** и земљи висок-вредност, лепљив, рачуни-гладна оптерећење —
алгоритамски трговци, брокери, prop фирме, и трговачке заједнице који желе платформу трчећа
без постајања ops тима сам.

:::tip TL;DR
Покрени stateless слој + Postgres + флота чворова; даш клијентима марку URL. Монетизирај
претплату, рачуни, белоетиетна, и AI. → [Развој на облак](./deployment/cloud.md)
:::

## Зашто нуди управљано cMind

- **Без грађевне цене.** То је отворена кода, MIT-лиценцирана, и већ документирана, тестирана, и контејнеризована.
  Паковаш и управљаш га — не грађиш га.
- **Диференцирана производ за лукратаван нише.** Algo трговина је рачуни-гладна: тестови и
  живи чворови палимо CPU, који је *наплатљив употреба* што већ даш.
- **Лепљив клијенти.** Трговци који граде и покрећу стратегије унутар платформе не утичу случајно.
- **Претвори превој у уписни улаз.** cMind је самосмећена по дизајну — за клијенте који "не желе да
  буду ops тим," *ти* си одговор.

## Ко купи управљано cMind од тебе

- **Појединачни квант & трговци** који га желе смештају. → [За трговце](./for-traders.md)
- **cTrader брокери** трчећа белоетиетна за њихове клијенте. → [За брокере](./for-brokers.md)
- **Prop фирме & копирање-трговања бизниси** који морају брендирана, аудитна инфраструктура.

## Шта "управљано cMind" означава за покретање

Управљаш три слоја; клијент добија брендирана веб URL:

| Слој | Шта је то | Где трчи |
|---|---|---|
| Stateless (Web + MCP) | Апликација + API + MCP сервер | Било какав контејнер платформа, autoscaled |
| База | PostgreSQL | Управљано Postgres (RDS / Flexible Server / твој сопствени) |
| Флота чворова | Граде и покрећу cTrader контејнере | **VM или Kubernetes — потребан привилегован Docker** |

:::warning Једна ствар за утврди напред
Агенти чворова граде и покрећу cTrader контејнере, тако да им морају привилегован Docker. То искључује
serverless контејнер окруженја (Azure Container Apps, AWS Fargate) *за агенте* — покрени те на
[Kubernetes](./deployment/kubernetes.md), VM, или EC2. Stateless слој трчи било где.
:::
