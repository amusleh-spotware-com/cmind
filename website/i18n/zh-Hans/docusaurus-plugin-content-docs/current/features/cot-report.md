# 交易商承诺报告 (COT)

cMind 配备了一份内置的**交易商承诺**报告——美国期货市场的每周 CFTC 分析，涵盖
谁持有多头和空头（商业套期保值者、大型投机者、基金），具有交互式历史图表、标准化的 **COT 指数**、
用于 cBots 的经过身份验证的 REST API 以及用于 AI 客户的 MCP 工具。数据直接来自 **CFTC 公开 Socrata 数据集**——
无需 API 密钥，无需聚合。像经济日历一样，它是一个可以禁用而不影响交易核心的分离模块。

## 它提供什么

- **所有三个报告系列，仅期货和期货+期权组合:**
  - **传统版** — 非商业性（大型投机者）、商业性（套期保值者）、不可报告性。
  - **分类版** — 生产者/商人、掉期交易商、管理资金、其他应报告方。
  - **金融期货交易商 (TFF)** — 交易商、资产管理人、杠杆基金、其他应报告方。
- **精选市场目录** — FX 主对、黄金/白银/铜、原油和天然气、国债、股票
  指数、加密货币以及主要谷物/软商品——每个都映射到其稳定的 CFTC 合约代码，
  在明确的地方，映射到可交易符号（例如欧元 FX → `EURUSD`、黄金 → `XAUUSD`）。
- **COT 指数 (0–100)** — 当前投机者净头寸在其历史范围内的位置
  （默认约 3 年回溯）。极端值附近的读数标记拥挤的头寸，通常先于
  反转；报告标记**多头极端**（≥80）或**空头极端**（≤20）。
- **时间点正确性。** 每周报告在周二计量，但只在
  次周五才向公众公开；每次读取都遵守该发布时刻，所以回测定位信号
  永远不会在报告公开前看到它（无前瞻偏差）。

## 使用页面

从左侧导航打开**交易商承诺**。选择一个**市场**、一个**报告类型**（传统版 /
分类版 / 金融版），并切换**期货 + 期权**在仅期货和组合变体之间切换。页面显示：

- **随时间变化的净头寸** — 每个交易者类别的净头寸（多头 − 空头）
  在整个历史窗口内的交互式折线图。
- **COT 指数** — 0–100 指数的折线图，显示最新读数及其极端标签。
- **最新快照** — 每个交易者类别的多头 / 空头 / 净头 / % 开放权益表，
  加上总开放权益和报告日期。

每个图表都配备了**放大/缩小**（和重置）工具栏按钮，您可以拖动时间轴进行缩放。**导出 CSV** 下载所选市场和报告类型的完整周历史记录作为电子表格就绪文件。使用**比较市场**在单个图表上叠加多个市场——比较图表并排绘制每个选定市场的投机者净头寸和 COT 指数，因此您可以一目了然地阅读跨市场头寸。

## 数据流动方式

数据库是缓存。每周的摄取工作程序拉取追踪市场的六个 CFTC 数据集，更新插入市场目录并幂等地追加每份新报告（重新运行永不复制快照）。此外，数据是**按需加载的**：首次请求市场时，它从 CFTC 源获取并存储，随后的每个请求都直接从数据库提供。缓存**在发布新的每周报告时刷新**——一旦存储的最新报告已超过一周，下一个请求会透明地拉取并追加最新数据（限制以防止源被过度访问）。第一次加载回填数年历史；源中断会降级为提供最佳缓存数据。一切开箱即用，无需密钥；可选的 Socrata 应用令牌仅提高速率限制。

## 配置

所有密钥都位于 `App:Cot` 下（请参阅[功能切换](./feature-toggles.md)和
[白标所有者设置](./white-label-owner-settings.md)）：

| 密钥 | 默认值 | 目的 |
|-----|---------|---------|
| `IngestionEnabled` | `true` | 每周摄取工作程序是否运行。 |
| `PollInterval` | `6h` | 工作程序轮询 CFTC 数据集的频率。 |
| `BackfillYears` | `5` | 第一次运行时拉取的历史年数。 |
| `ReconcileLookbackWeeks` | `4` | 每个周期重新同步的最近几周以捕获修订。 |
| `SocrataAppToken` | — | 可选令牌，提高匿名速率限制。 |
| `CotIndexLookbackWeeks` | `156` | 用作 COT 指数范围的每周报告（约 3 年）。 |

## 门控

可见性是一个两级门控，与经济日历相同：白标硬门控
`App:Branding:EnableCot`（构建级别）**和**运行时功能切换 `App:Features:Cot`。禁用其中任何一个时，
导航链接、页面、REST API 和 MCP 工具都会消失（API 返回 `404`）。由于数据
源无需密钥，没有数据源密钥门控——启用意味着可见。

## 适用于开发者

- 领域：`Core.Cot` — `CotMarket` 和 `CotReport` 聚合、`CotPositions` 值对象、
  `CotIndexCalculator` 领域服务和 `ICotReports` / `ICotSource` 端口。
- 基础设施：`Infrastructure.Cot` — `CftcSocrataSource` 反腐败解析器、速率门控、
  仅追加写入服务、读端和每周摄取工作程序（EF `cot` 架构）。
- cBot 和 AI 访问：[COT cBot API](./cot-cbot-api.md)（REST、`market:read` JWT）
  和 MCP 工具 `CotMarkets`、`CotLatest`、`CotHistory`、`CotHealth`。
