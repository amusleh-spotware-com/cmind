# cMind για cTrader κομίστρες 🏦

Εσείς τρέχουν α cTrader κομίστρα. Δικό σας κλιέντες μπορώ ήδη έμπορο — αλλά έτσι μπορώ κάθε άλλο κομίστρα κλιέντες. **cMind αφήνει σας χέρι δικό σας έμπορα α πλήρη AI-στρατηγικό διαπραγμάτευσης ενέργειες πλατφόρμα, branded ως
δικό σας**, έτσι τους κτίριο, backtest, τρέχουν, αντίγραφο, και παρακολουθούν στρατηγικές εσωτερικό *δικό σας* οικοσύστημα
αντί ολίσθηση σε α τρίτη-κόμμα εργαλείο. Αυτό είναι κολλώδη κλιέντες, περισσότερος όγκος, και α πραγματικό άκρο πάνω
κομίστρες προσφέρει τίποτα αλλά ένα τερματικό.

:::tip[TL;DR]
Τρέχουν α λευκό-ετικέτα cMind για δικό σας κλιέντες. Περιορισμός λογαριασμούς σε **δικό σας** κομίστρα, διακοπή σε AI και
αντιγραφή διαπραγμάτευσης, και πλοίο αυτό κάτω δικό σας brand. → [Λευκό-ετικέτα για επιχείρηση](./white-label-for-business.md)
:::

## Το άκρο είστε παίρνουν πάνω άλλος κομίστρες

- **Διαφοροποίηση σε εργαλείο, δεν ακριβώς σχύσιμο.** Δώρο κλιέντες AI cBot δημιουργία, backtesting σε α
  διαχειριζόμενο σύμπλεγμα, αντιγραφή διαπραγμάτευσης, και prop-firm προκλήσεις — δυνατότητες περισσότερα κομίστρες απλά δεν
  προσφορά.
- **Κρατήστε κλιέντες σε δικό σας οικοσύστημα.** Όταν έμπορα κτίριο και τρέχουν τα στρατηγικές εσωτερικό δικό σας branded
  πλατφόρμα, τους παραμένει. Κράτηση είναι ολόκληρος παιχνίδι.
- **Κάτω δικό σας brand, σε δικό σας τομέα.** Όνομα, λογότυπο, χρώματα, favicon, ακόμη ο εγκαταστάσιμη τηλέφωνο εφαρμογή —
  όλα δικό σας. Κανείς δείχνει "cMind." → [Λευκό-ετικέτα δυνατότητα](./features/white-label.md)

## Εξυπηρετούν μόνο δικό σας λογαριασμούς (κομίστρα allowlist)

Τρέχοντας α λευκό-ετικέτα για *δικό σας* κλιέντες; Περιορισμός ποιο κομίστρες διαπραγμάτευσης λογαριασμούς χρήστες μπορώ προσθήκη έτσι
δικό σας ανάπτυξη μόνο ποτέ εξυπηρετούμενο δικό σας βιβλίο:

```json
{
  "App": {
    "Accounts": {
      "AllowedBrokers": ["Δικό σας Κομίστρα Όνομα"]
    }
  }
}
```

Όταν το allowlist έχει ορίστε, cMind έλεγχο κάθε λογαριασμό ένας χρήστης προσπαθεί προσθήκη — αμφότερα μέσα cTrader Open
API και μέσα χειροκίνητα cID σύνδεση (επαληθευμένο κατά ανάγνωση λογαριασμό πραγματικό κομίστρα όνομα) — και απορρίπτει κάθε
λογαριασμό το οποίο δεν έχουν σε δικό σας κατάλογο. Αφήστε αυτό κενό και κάθε κομίστρα επιτρεπτό (το προεπιλογή). Δείτε το
[Λευκό-ετικέτα δυνατότητα doc](./features/white-label.md#κομίστρα-allowlist) για πλήρης μηχανικά.

## Πλοίο ένα Open API εφαρμογή για όλα δικό σας χρήστες

Παραλήψεις ο ανά-χρήστη ταλαιπωρία: παρέχουν **ένα cTrader Open API εφαρμογή** και κάθε κλιέντης εξουσιοδοσία
τα λογαριασμούς μέσα αυτό — χωρίς κλιέντης ποτέ εγγραφή δικό τους. Εγγραφή ένα ενιαίο ανακατευθύνει URL, ρίχτε
το διαπιστευτήρια σε ρύθμιση ή ιδιοκτήτης ρυθμίσεις, και κοινή-τρόπος τρέχει σε για όλα. Διαπραγματευόμαστε α
υψηλότερο cTrader μήνυμα όριο; Tune το **ανά-μήνυμα-τύπο κλιέντη ποσοστό όρια** (ή αποκλείουν ρυθμό).
→ [Κοινή Open API εφαρμογή & ποσοστό όρια](./features/open-api-shared-app.md)

## Νέος τρόποι σε μετατροπή

- **AI, με μηδέν τριβή για κλιέντες.** Παρέχουν α προεπιλογή AI παροχέα κλειδί σε το ανάπτυξης σταθμό και
  κάθε κλιέντης παίρνει AI δυνατότητες αμέσως — χωρίς υπογραφή αλλού. Σημάδι αυτό προς τα πάνω, ή δεσμευμένο αυτό σε premium
  σειρών. Κλιέντες μπορώ ακόμη φέρνουν δικό τους κλειδί. → [AI δυνατότητα](./features/ai.md)
- **Prop-firm προκλήσεις.** Τρέχουν χρηματοδότηση-έμπορα προκλήσεις με ζωντανό ίδια κεφάλαια ιχνηλάτηση και επιβάλλονται κανόνες,
  και χρέωση για καταχωρίσεις. → [Prop-firm κανόνα](./features/prop-firm.md)
- **Αντιγραφή-διαπραγμάτευσης επιχείρηση.** Επιδόσεων τέλη και α παροχέα αγορά τρόπο αντιγραφή διαπραγμάτευσης σε
  έσοδο. → [Επιδόσεων τέλη](./features/copy-performance-fees.md) ·
  [Παροχέα αγορά](./features/copy-provider-marketplace.md)
- **Δυνατότητα σειρών.** Απόφασης ποιο δυνατότητες κάθε κλιέντης τμήμα δείχνει με
  [δυνατότητα toggles](./features/feature-toggles.md).

## Ρυθμιστικά, ελεγκτική, multi-ενοικιαζόμενο

- **[Συμμόρφωση](./features/compliance.md)** καταγραφή δώρο σας ο ελεγκτική λογιστική δικό σας ρυθμιστής θα ζητάω για.
- **[Δύο-παράγοντας τυλίγματος](./features/two-factor-auth.md)** μπορώ να κάνουν υποχρεωτικό ανά ανάπτυξη.
- **Ανά-κλιέντη κοσμήματος** — τρέχουν α χωριστό branded παράδειγμα ανά τμήμα, στρατηγική από δικό σας δικό
  ελεγχος επίπεδο. → [Multi-ενοικιαζόμενο κοσμήματος](./white-label-for-business.md#multi-ενοικιαζόμενο-ανά-κλιέντη-κοσμήματος)

## Πώς να ξεκινήσει

1. Διαβάστε [Λευκό-ετικέτα για επιχείρηση](./white-label-for-business.md) για το 60-δευτερόλεπτο rebrand.
2. Θέστε `App:Accounts:AllowedBrokers` σε δικό σας κομίστρα και επιλέξτε δικό σας [δυνατότητα σύνολο](./features/feature-toggles.md).
3. [Ανάπτυξη](./deployment/cloud.md) αυτό — Docker, Kubernetes, Azure, ή AWS.

Μην θέλει να τρέχουν υποδομή ο εαυτός σας; Α φιλοξενείες παροχέα μπορώ να λειτουργούν α διαχειριζόμενο cMind για σας
— σημείο τους σε [Για σύννεφο & VPS προμηθευτές](./for-cloud-providers.md).

## Σχήμα ο χάρτης

cMind είναι ανοιχτή πηγή. Κομίστρες ποιο κτίριο σε αυτό παίρνουν ένα outsized λέγε σε όπου αυτό πάει — αίτημα το
ενσωματώσεις και έλεγχος είστε απαιτούν, και συμβάλει τους πίσω μέσα το
[Συμβάλλω οδηγό](./contributing.md).
