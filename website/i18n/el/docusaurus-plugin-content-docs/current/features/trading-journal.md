# Trading Journal & Coach

Η νέα γνήσια χρήσιμη κατηγορία AI-for-trading δεν είναι η πρόβλεψη της αγοράς — είναι η ανάλυση
*της δικής σας* συμπεριφοράς. Το Trading Journal μετατρέπει το ιστορικό των runs και backtests σας σε ειλικρινή ανατροφοδότηση ώστε
να μπορείτε να βελτιώσετε τη στρατηγική που ήδη έχετε.

Ανοίξτε **AI → Trading Journal** (`/journal`).

## Τι φέρνει στην επιφάνεια

Από τα instances σας (runs και backtests) υπολογίζει, ντετερμινιστικά:

- **Μετρήσεις νίκης/απώλειας/αποτυχίας και win rate** στα backtests σας;
- **Συμπεριφορικές γνώσεις** — τις διαρροές που σιωπηλά κοστίζουν στους retail traders:
  - **Υπερσυγκέντρωση** — το μεγαλύτερο μέρος της δραστηριότητάς σας είναι σε ένα σύμβολο;
  - **Επαναλαμβανόμενες αποτυχίες** — υψηλό ποσοστό runs απέτυχαν να χτίσουν ή να διαμορφώσουν;
  - **Χαμένη προκατάληψη** — περισσότερα χαμένα από κερδισμένα backtests (με μια ώθηση να τρέξετε το Integrity Lab και
    να ελέγξετε αν το πλεονέκτημα είναι πραγματικό);
  - ένα καθαρό πιστοποιητικό υγείας όταν δεν ισχύει τίποτα από τα παραπάνω.

```http
GET /api/journal
```

## Γιατί είναι αξιόπιστο

Η συμπεριφορική ανάλυση είναι καθαρός, ντετερμινιστικός domain code (`Core.Journal`) χωρίς εξάρτηση υποδομής — unit-tested για την υπερσυγκέντρωση, τις επαναλαμβανόμενες αποτυχίες, τη χαμένη προκατάληψη, την ισορροπημένη περίπτωση και τον κενό λογαριασμό. Τα γεγονότα πρώτα· ο AI coach (Portfolio Digest) είναι ένα προαιρετικό αφηγηματικό επίπεδο από πάνω, gated στο Anthropic API key, ώστε το journal να λειτουργεί πλήρως χωρίς AI.
