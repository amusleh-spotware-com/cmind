# 时区

应用显示的每个时间都以您自己的时区呈现，而非服务器时区。您的选择会保存到个人资料，并在各设备间跟随您。

首次访问时，应用会自动采用您浏览器的时区。您可随时在“设置 → 时区”中更改；部署默认值为白标选项 App:Branding:DefaultTimeZone（默认 UTC）。时间始终以 UTC 存储并由 API 返回——仅显示会被转换。

- 解析顺序：个人资料时区，然后 Cookie，然后部署默认值，然后 UTC。
- 检测只运行一次，绝不覆盖您已选择的时区。
- 格式遵循您的语言；诸如“2 分钟前”之类的相对标签不受影响。
