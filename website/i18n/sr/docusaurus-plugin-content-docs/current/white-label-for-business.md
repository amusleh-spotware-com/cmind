# Белоетиетна cMind за твој бизнис 🏢

Покрени prop фирму, браборски стол, или копирање-трговања сервис? cMind је саграђен од дана једна да буде
**препродањ као твој сопствени производ**. Свака површина — име, логотип, favicon, боје, чак и
инсталабилна телефонска апликација — саглас твојој марки. Твоје клијенти виде *твој* компанија. Без кодних промена,
без fork, само конфигурација.

:::tip TL;DR
Укажи `App:Branding` на твој име, боје, и логотип. Рестарт. Урађено. Пунина техничка референца живи
у [Белоетиетна функција док](./features/white-label.md).
:::

## Шта можеш прерадити

| Површина | Шта се мења |
|---|---|
| **Име производа** | App bar текст + карта прегледача наслов |
| **Логотип & favicon** | Твоје знакове свуда, укључујући карту прегледача |
| **Боје** | Пунина палета — примарна, површине, статус боје — тече кроз цео UI *и* апликација је своја CSS преко дизајна жетона |
| **Инсталабилна апликација (PWA)** | Додај-за-домаћа-екран име, икона, и блиц користе твоју марку |
| **Meta / SEO** | Опис и подршка URL су твоја |
| **Прилагођена CSS** | Инјектирај твој сопствени полиш за последња 5% |

Све подразумевано на залиха cMind идентитет, тако да само преправи шта те брине.

## 60-секундна прерадити

Постави ово на твој развој (JSON конфигурација или променљиве окружења):

```json
{
  "App": {
    "Branding": {
      "ProductName": "AcmeFX",
      "CompanyName": "Acme Markets Ltd",
      "SupportUrl": "https://support.acme.example",
      "LogoUrl": "/branding/acme-logo.svg",
      "FaviconUrl": "/branding/acme.ico",
      "PrimaryColor": "#2D7FF9",
      "SecondaryColor": "#1E63C8",
      "ShowSiteLink": false
    }
  }
