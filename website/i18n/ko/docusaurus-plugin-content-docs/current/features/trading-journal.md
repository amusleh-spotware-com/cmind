# 거래 저널 및 코치

AI-for-tr레이딩의 진정으로 유용한 새로운 범주는 시장을 예측하는 것이 아닙니다 — *자신의 행동*을 분석하는 것입니다. 거래 저널은 실행 및 백테스트의 이력을 솔직한 피드백으로 변환하여 이미 가지고 있는 전략을 개선할 수 있습니다.

**AI → Trading Journal** (`/journal`)을 엽니다.

## 무엇이 표면화되는가

인스턴스(실행 및 백테스트)에서 결정론적으로 계산합니다:

- **승/패/실패 횟수 및 승률** 백테스트 전반;
- **행동적 인사이트** — 조용히 소매 거래자에게 비용이 드는 leak:
  - **과도한 집중** — 활동 대부분이 하나의 심볼에集中;
  - **반복적 실패** — 실행 또는 구성에 실패한 실행 비율이 높음;
  - **손실 편향** — 패한 백테스트가 이긴 것보다 많음 (Integrity Lab 실행 및 에지가 진짜인지 확인하도록 nudge와 함께);
  - 위에 해당하지 않으면 건강한 상태로 발표.

```http
GET /api/journal
```

## 왜 신뢰할 수 있는가

행동 분석은 인프라 의존성이 없는 순수 결정론적 도메인 코드 (`Core.Journal`)입니다 — 과도한 집중, 반복적 실패, 손실 편향, 균형 경우 및 빈 계정에 대해 단위 테스트됩니다. 사실이 먼저입니다; AI 코치 (Portfolio Digest)는 Anthropic API 키에 게이트된 선택적 내러티브 레이어이므로 AI가 구성되지 않아도 저널이 완전히 작동합니다.
