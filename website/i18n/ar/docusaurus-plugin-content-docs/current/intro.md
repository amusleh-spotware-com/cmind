# مرحبًا بك في cMind 👋

إذًا تريد بناء روبوتات تداول، واختبارها رجعيًا دون أن يذوب حاسوبك المحمول، وتشغيلها عبر عدّة أجهزة،
ونسخ الصفقات إلى عشرات الحسابات، وترك الذكاء الاصطناعي يراقب المخاطر بينما تنام. **أنت في المكان الصحيح
تمامًا.**

cMind هي **منصّة عمليات تداول مفتوحة المصدر وقابلة للاستضافة الذاتية لمنصّة cTrader**. تخيّلها كمكتب
التداول الخاص بك بالكامل — التأليف والتنفيذ وأسطول حوسبة ونسخ التداول ونواة ذكاء اصطناعي — مجمّعة في
تطبيق هادئ وداكن ومناسب للجوّال تملكه من البداية إلى النهاية.

:::tip في جملة واحدة
ابنِ ← اختبر رجعيًا ← شغّل ← انسخ استراتيجيات cTrader الخاصة بك على نطاق واسع، مع ذكاء اصطناعي مدمج،
على خوادمك الخاصة وتحت علامتك التجارية الخاصة.
:::

## مسار الخمس دقائق ⏱️

إذا كان لديك Docker وخمس دقائق، يمكنك أن تعبث بنسخة cMind حقيقية الآن:

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

ثم افتح **<http://localhost:8080>** وسجّل الدخول، وقد انطلقت. الدليل الكامل (مع حلّ المشكلات لحين يكون
لدى Docker رأي حتمًا) موجود في **[تشغيله محليًا](./deployment/local.md)**.

## كلمة سريعة عن المال 💸

يحرّك cMind **رأس مال حقيقي**. نأخذ ذلك على محمل الجدّ — كل تغيير يُشحن مع اختبارات وحدة وتكامل ومن طرف
إلى طرف، بما في ذلك مسارات الفشل (انقطاع الاتصالات، رفض الأوامر، تعطّل العُقد). ينبغي أن تأخذه على محمل
الجدّ أيضًا: **اختبر أولًا على حساب تجريبي**، واقرأ [ملاحظات الامتثال](./features/compliance.md) قبل أن
توجّهه إلى أي شيء حقيقي. التداول محفوف بالمخاطر؛ هذا البرنامج أداة، وليس نصيحة مالية.
