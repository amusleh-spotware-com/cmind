# ยินดีต้อนรับสู่ cMind 👋

คุณอยากสร้างบอทเทรด แบ็คเทสต์มันโดยไม่ทำให้แล็ปท็อปละลาย รันมันข้ามหลายเครื่อง มิเรอร์การเทรดไปยังบัญชี
เป็นสิบ ๆ และให้ AI คอยเฝ้าดูความเสี่ยงในขณะที่คุณนอนหลับ **คุณมาถูกที่แล้ว**

cMind คือ **แพลตฟอร์มปฏิบัติการเทรดโอเพนซอร์สที่โฮสต์เองได้สำหรับ cTrader** ลองนึกภาพว่ามันคือโต๊ะเทรด
ทั้งหมดของคุณ — การเขียน การรัน กองเครื่องประมวลผล การคัดลอกการเทรด และแกน AI — รวมอยู่ในแอปเดียวที่เงียบ
สงบ โทนมืด เป็นมิตรกับมือถือ ซึ่งคุณเป็นเจ้าของตั้งแต่ต้นจนจบ

:::tip ในประโยคเดียว
สร้าง → แบ็คเทสต์ → รัน → คัดลอก กลยุทธ์ cTrader ของคุณในสเกลใหญ่ พร้อม AI ในตัว บนเซิร์ฟเวอร์ของคุณเอง
ภายใต้แบรนด์ของคุณเอง
:::

## จริง ๆ แล้วมันทำอะไรได้บ้าง?

| คุณต้องการ… | cMind ทำให้ได้ | อ่านเพิ่มเติม |
|---|---|---|
| เขียน cBot ในเบราว์เซอร์ | Monaco IDE + เทมเพลต C#/Python, บิลด์แบบแซนด์บ็อกซ์ | [สร้างและแบ็คเทสต์](./features/build-and-backtest.md) |
| แบ็คเทสต์ข้ามเครื่อง | กองโหนดที่ซ่อมแซมตัวเองเลือกเครื่องที่ว่างที่สุด | [การปรับสเกล](./deployment/scaling.md) |
| คัดลอกบัญชีหนึ่งไปยังหลายบัญชี | การมิเรอร์ที่แข็งแกร่งพร้อมการซิงก์ใหม่ ไม่มีการเทรดซ้ำซ้อน | [การคัดลอกการเทรด](./features/copy-trading.md) |
| ให้ AI ทำงานหนักแทน | สร้างกลยุทธ์ ซ่อมแซมตัวเอง ผู้พิทักษ์ความเสี่ยง วิเคราะห์หลังเหตุการณ์ | [แกน AI](./features/ai.md) |
| อยู่ในกฎของ prop firm | ติดตามอิควิตี้แบบสด + จำลองกฎแชลเลนจ์ | [Prop firm](./features/prop-firm.md) |
| ปล่อยมันเป็นผลิตภัณฑ์ *ของคุณ* | ไวต์เลเบลเต็มรูปแบบ: ชื่อ สี โลโก้ favicon | [ไวต์เลเบล](./features/white-label.md) |
| รันบนโทรศัพท์ของคุณ | PWA ที่ติดตั้งได้และเน้นมือถือ | [PWA](./features/pwa.md) |
| ขับเคลื่อนจากไคลเอนต์ AI | เซิร์ฟเวอร์ MCP ในตัว (HTTP + SSE) | [MCP](./features/mcp.md) |

## เส้นทาง 5 นาที ⏱️

ถ้าคุณมี Docker และเวลาห้านาที คุณสามารถลองเล่นกับอินสแตนซ์ cMind จริง ๆ ได้เลยตอนนี้:

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

จากนั้นเปิด **<http://localhost:8080>** ลงชื่อเข้าใช้ แล้วก็เริ่มได้เลย คู่มือฉบับเต็ม (พร้อมการแก้ปัญหา
สำหรับตอนที่ Docker มีความคิดเห็นเป็นของตัวเองอย่างเลี่ยงไม่ได้) อยู่ที่
**[การรันในเครื่อง](./deployment/local.md)**

## ขอพูดเรื่องเงินสักหน่อย 💸

cMind ขับเคลื่อน **เงินทุนจริง** เราจริงจังกับเรื่องนี้ — ทุกการเปลี่ยนแปลงจัดส่งพร้อมการทดสอบระดับยูนิต
การผสานรวม และแบบครบวงจร รวมถึงเส้นทางความล้มเหลว (การเชื่อมต่อหลุด คำสั่งถูกปฏิเสธ โหนดตาย) คุณก็ควร
จริงจังด้วยเช่นกัน: **ทดสอบบนบัญชีเดโมก่อน** และอ่าน [หมายเหตุการปฏิบัติตามข้อกำหนด](./features/compliance.md)
ก่อนจะชี้มันไปที่อะไรก็ตามที่เป็นของจริง การเทรดมีความเสี่ยง ซอฟต์แวร์นี้เป็นเครื่องมือ ไม่ใช่คำแนะนำ
ทางการเงิน
