# Pre koho je cMind?

Krátka verzia: **ľudia s peniazmi na riadení, ktorí chcú mať kontrolu.** cMind je vážna
algoritmus-trading operations konzola — nie spotrebiteľský fintech hračka s konfeti animáciou keď urobíte $3. Či obchodujete, brokerujete alebo hostujete, je tu cesta nižšie postavená pre vás.

## 📈 Obchodníci — self-hostujte svoj celý stôl

Obchodujete na cTrader a ste unavení zlepovaním editora, backtesteru, VPS a troch
prehliadačových záložiek. Self-hostujte cMind a získajte authoring, backtesting, live execution, copy trading a monitorovanie v jednej AI-powered konzole — na vašom boxe, vaše dáta nikdy neopúšťajú.

**Čítajte:** [cMind pre cTrader obchodníkov →](./for-traders.md)

## 🏦 Brokeri — white-label pre vašich klientov

Prevádzkyujete cTrader brokeráž a chcete výhodu nad konkurenciou, ktorá ponúka iba terminál. Dodajte cMind ako svoj vlastný produkt: dajte klientom AI, copy trading a prop-firm challenges pod vašou značkou, obmedzte účty na vašu knihu a otvorte nové príjmové linky.

**Čítajte:** [cMind pre cTrader brokerov →](./for-brokers.md) ·
[White-label pre obchod →](./white-label-for-business.md)

## 🖥️ Cloud & VPS poskytovateľ — predajte ako managed hosting

Prenajímate výpočetnú kapacitu. Ponúknite managed cMind hosting a prikryte si high-value, compute-hungry, sticky workload — a monetizujte subscription, metered compute, white-label a AI.

**Čítajte:** [cMind pre cloud & VPS poskytovateľov →](./for-cloud-providers.md)

## 🧑‍💻 Prispievatelia — pomôžte to vylepšiť

cMind je open source (MIT) a community-shaped: C# 14 / .NET 10, striktný DDD, EF Core + PostgreSQL,
.NET Aspire, MCP server a tri test tiers. Každá osoba vyššie môže — a mala by — to posunúť dopredu:

- **Obchodníci:** pridajte cBot šablóny, žiadajte funkcie, nahláste chyby.
- **Brokeri:** prispejte integráciami a ovládačmi, ktoré váš stôl potrebuje.
- **Poskytovateľ:** upstream vaše operačné opravy a IaC vylepšenia.

Podajte úkoly, pridajte UI preklady alebo AI provider adaptéry a pošlite PRs — [Contributing guide](./contributing.md) a [MCP server docs](./features/mcp.md) vás dostanú spustené.

## Nie je cMind *nie* pre vás?

Buďte k sebe čestní. cMind by mohla byť overkill ak:

- Nikdy ste sa nedotkli terminálu a nechcete začínať — hoci [cloud/VPS provider to môže hostovať pre vás](./for-cloud-providers.md).
- Obchodujete iba manuálne a nezáleží vám na botoch, backtestoch alebo copy tradingu.
- Chcete hosted SaaS s support hotline a nechcete, aby to niekto self-hostoval — cMind je self-hosted podľa návrhu (len sa snažší veľmi, aby to bolo bezbolestné).
