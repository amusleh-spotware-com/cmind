# コピートレーディング

**マスター** cTrader アカウント ミラー 1 個以上 **スレーブ** アカウント上 — クロスブローカー、クロス cID — 宛先あたり制御 + マネーグレード調整。

## コンセプト

- **コピープロファイル** — 1 個マスター (`SourceAccountId`) + 1 個以上 **宛先**。ライフサイクル: `Draft → Running → Paused → Stopped` (`Error` 障害時)。集約ルート: `CopyProfile` (所有 `CopyDestination`)。
- **宛先** — 1 個スレーブ アカウント + マスターそれにコピー方法 完全ルールセット。すべて構成 宛先あたり、1 個マスター フィード 保守的 + アグレッシブ スレーブ 同時。
- **コピーエンジン ホスト** — 実行中ワーカー プロファイル (`CopyEngineHost`)。マスター実行ストリーム サブスクライブ、各イベント 適用 すべての宛先。
- **スーパーバイザ** — `CopyEngineSupervisor`、背景サービス 各ノード。ホスト割り当てプロファイル、クラスタ横 自己回復 ([スケーリング](../deployment/scaling.md) 参照)。

## ミラーされるもの

| マスター イベント | スレーブ アクション |
|--------------|--------------|
| マーケット / マーケット範囲 ポジションオープン | サイズコピー開く (ラベル付き ソースポジション id) |
| リミット / ストップ / ストップリミット 保留中オーダー | マッチング 保留中オーダー 配置 |
| 保留中オーダー 修正 | スレーブ 保留中オーダー インプレース修正 |
| 保留中オーダー キャンセル / 有効期限 | スレーブ 保留中オーダー キャンセル |
| 部分クローズ | スレーブポジション 同比率 クローズ |
| スケールイン (ボリューム増加) | オープン 追加ボリューム (オプトイン) |
| ストップロス / トレーリングストップ 変更 | スレーブポジション 保護修正 |
| 完全クローズ | スレーブコピー クローズ |

すべてのコピー **ラベル付き** ソースポジション/オーダー ID。再接続後 ホスト 調整からの状態リビルド: オープン コピー マスター保有 スレーブ 欠落、クローズ スレーブ「孤立」マスター 保有しない — **複製トレード なし**。

## プロファイル作成

**新しいプロファイル** ダイアログ コピートレーディング ページで すべて フロント収集: プロファイル名、ソース (マスター) アカウント、宛先 (スレーブ) アカウント (マルチセレクト と **すべて選択** ボタン; 選択マスター 除外 スレーブリストから)、+ 下記完全 宛先あたりオプションセット。すべての入力 **保存前に検証** — 欠落 名前/ソース/宛先、非正数 サイズパラメータ、負/不一致 ロットバウンド、範囲外 ドローダウン %、有効化 オーダータイプ なし、空 シンボルフィルター、または 不正形成 シンボルマップペア サーフェス エラーリスト + ブロックセーブ。確認時、プロファイル作成 + 選択 スレーブ すべて 選択設定で追加。

行アクション ライフサイクル尊重: **開始** 有効 実行時のみ、**停止** + **一時停止** 実行時のみ、**削除** 無効 実行中 + 削除前に確認要求 プロファイル + 宛先。

## 宛先あたりオプション

新しいプロファイル ダイアログ設定、コピートレーディング ページ 宛先あたりパネル、または `POST /api/copy/profiles/{id}/destinations` 経由:

- **サイジング** (`MoneyManagementMode` + パラメータ): 固定ロット、ロット/名目乗数、比例 バランス/エクイティ/フリーマージン、固定リスク %、固定レバレッジ、自動比例、**リスク%-ストップから** (M7)。プラス 最小/最大ロット境界 + 強制最小ロット。**リスク-ストップから** サイズ宛先 危機 設定パーセント *そのもの* バランス、導出 **マスター ストップロス距離** から (`マスター危機 2% → スレーブ 自動危機 2%`): `ロット = バランス×% ÷ (ストップ距離 × コントラクトサイズ)`。マスター オープン **なし** ストップロス サイズに距離なし → 使用 設定 **最大リスク フォールバック ロット** (M7) 設定、そうでなければ スキップ (`no_stop_loss`) 推測なし。比例 **エクイティ**/**フリーマージン** サイズ オフ 実 アカウント **エクイティ** (`バランス + Σ フローティング P&L`、導出 cTrader Open API 経由 エクイティ配信しない)、単なる バランスではなく — マスター 座る オープン利益/損失 サイズ コピー 正し。使用マージン 調整 API 露出しない、フリーマージン 扱う エクイティとして (正直 利用可能資金プロキシ); 他のモード 読み取り バランス + スキップ 追加 再評価 ラウンドトリップ。

コピートレーディングの詳細な機能説明については、完全ドキュメント参照。
