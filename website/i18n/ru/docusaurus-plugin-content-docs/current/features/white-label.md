# Брендирование белого ярлыка

Перепроданец перебрендировать приложение — имя продукта, логотип, favicon, цвета, пользовательский CSS — через конфигурацию развертывания, без изменения кода. Каждое значение брендирования **по умолчанию в стандартную личность**: некоконфигурированное развертывание выглядит так же как раньше; перепроданец переопределяет только то, что нужно.

## Модель

- `Core.Options.BrandingOptions` — привязано от `App:Branding`. Строка-основана (край config); каждый цвет проверяется когда тема построена.
- `Core.Branding.HexColor` — объект значения для CSS hex цвета (`#RGB` / `#RRGGBB`), неизменяемый, самовалидирующийся. Неправильный цвет выбрасывает `DomainException` (`domain.branding.color_invalid`) при построении темы — неправильно конфигурированное развертывание отказывает рано при запуске, не отрисовывая сломанную палитру.
- `Web.Components.Theme.Build(BrandingOptions)` — производит тему MudBlazor из брендирования. Только брендированные записи палитры поступают из config; типография, компоновка, нейтральные тона поверхности остаются фиксированными, поэтому продукт держит согласованный внешний вид через перепродавцов.
- `Web.Branding.IBrandingThemeProvider` — одиночка, построить тему один раз, перестроить при изменении параметров. Инъектируется `MainLayout`/`EmptyLayout` для `MudThemeProvider`, по строке приложения для имени/логотипа продукта. `App.razor` читает `IOptionsMonitor<AppOptions>` прямо для страницы `<head>` (название, описание, favicon, theme-colour, пользовательский CSS).

## Конфигурация

```json
{
  "App": {
    "Branding": {
      "ProductName": "AcmeFX",
      "CompanyName": "Acme Markets Ltd",
      "SupportUrl": "https://support.acme.example",
      "Description": "AcmeFX — копирование торговли и автоматизация стратегии.",
      "LogoUrl": "/branding/acme-logo.svg",
      "FaviconUrl": "/branding/acme.ico",
      "PrimaryColor": "#2D7FF9",
      "SecondaryColor": "#1E63C8",
      "AppBarColor": "#0B1220",
      "BackgroundColor": "#0E1525",
      "SurfaceColor": "#161E30",
      "SuccessColor": "#3FB950",
      "ErrorColor": "#F85149",
      "WarningColor": "#D29922",
      "InfoColor": "#2D7FF9",
      "CustomCss": ".mud-appbar { letter-spacing: 1px; }"
    }
  }
}
```

Форма переменной окружения: `App__Branding__ProductName=AcmeFX`, `App__Branding__PrimaryColor=%232D7FF9`.

| Ключ | Эффект | По умолчанию |
|-----|--------|---------|
| `ProductName` | Текст строки приложения + страница `<title>` | `cMind` |
| `LogoUrl` | Изображение логотипа строки приложения; когда пусто, текст имени продукта показывает | *(пусто)* |
| `FaviconUrl` | `<link rel="icon">` | `favicon.svg` |
| `Description` | `<meta name="description">` | стандартное описание |
| `PrimaryColor` / `SecondaryColor` | акцент, значок ящика, кнопки | `#26C281` / `#1FB97A` |
| `AppBarColor` / `BackgroundColor` / `SurfaceColor` | chrome + поверхности; `AppBarColor` ведет `<meta theme-color>` + PWA манифест `theme_color`, `BackgroundColor` манифест `background_color` | тёмная палитра |
| `SuccessColor` / `ErrorColor` / `WarningColor` / `InfoColor` | цвета статуса | стандартные |
| `CustomCss` | инъектированная `<style>` в `<head>` (развертывание-доверитель) | *(пусто)* |
| `ShowSiteLink` | показать кредит ссылку «Создано cMind» на приборной панели | `true` |
| `RequireMfa` | требовать каждому пользователю установить двухфакторную аутентификацию перед использованием приложения | `false` |
| `NodesUi` | сколько поверхности узлов поставляется: `Full` (список + ручное добавление/удаление), `Monitor` (список только для чтения, без добавления/удаления), `Hidden` (нет навигации, нет страницы, нет ручного API) | `Full` |
| `RestrictNodesToOwner` | когда `true`, только владелец может видеть/управлять узлами; иначе вся адм-или-выше персонал поверхность может. Нормальные пользователи никогда не видят узлы так или иначе | `false` |

Активы на которые ссылаются `LogoUrl`/`FaviconUrl` поставляются с Web приложения `wwwroot` (например монтирование папки `wwwroot/branding/`) или любой абсолютный URL.

`App:Branding` проверяется при запуске (`BrandingOptionsValidator`, запустите через `ValidateOnStart`): каждый цвет должен быть действительным hex, `CustomCss` не должен содержать `<`/`>` (не может вырваться из тега `<style>`). Неправильно сконфигурированное развертывание отказывает загрузку с четким сообщением, не отрисовывая сломанную страницу.

## Ссылка создано

Приборная панель отображает небольшую ссылку кредита **«Создано cMind»**, которая указывает на сайт документации проекта.
