# Contrarian Retail Positioning

소매 군대는 FX에서 진정으로 유용한 감정 지표 중 하나입니다 — **역방향** 지표로서. 대부분의 소매 트레이더가ロング이면 가격은 Historically 하락하는 경향이 있었으며 그 반대도 마찬가지입니다. 이 도구는 군대 포지셔닝을 실행 가능한 판독으로 전환합니다.

**cBots → Contrarian Positioning** (`/quant/positioning`)을 엽니다.

## 무엇을 하는가

**소매 트레이더의 %가ロング** (브로커의 감정 페이지 또는 FXSSI와 같은 피드에서)를 입력하면 반환됩니다:

- **역방향 편향** — 60% 이상이ロング이면 **약세** (군대가 너무 많음), 40% 이하이면 **강세** (군대가 너무 적음), 40–60% 의사결정 밴드에서 **중립**;
- **강도** — 군대가 얼마나 기울어져 있는지 (0 = 균형, 1 = 완전히 한쪽).

```http
POST /api/quant/positioning
{ "longPercent": 72 }
```

## 시점에 따른 구축

하단에서 신호 레이어 (`Core.Signals`)는 그것을 알 수 있었瞬시로 stamp되고 그것 없이는 구성될 수 없는 `PointInTimeSignal`을 모델링합니다. 신호를消費하는 모든 백테스트 또는 자율 에이전트는 `IsKnownAt(decisionTime)`을 확인합니다 — 그래서 미래 데이터가 과거 결정에 leak할 수 없습니다. 룩어헤드 편향은 양적 금융에서 최고의 재현성 살해자입니다; 도메인 모델은 구조적으로 불가능하게 만들 것입니다.

## 왜 신뢰할 수 있는가

인프라 의존성이 없는 순수 결정론적 도메인 코드입니다 — 역방향 임계값 및 시점 가드가 단위 테스트되었으며 40/60 경계 및 범위 외 거부도 포함합니다.
