# Triển khai Kubernetes

cMind được xây dựng để Kubernetes. Triển khai stateless (Web + MCP) là một Deployment; PostgreSQL có thể là một StatefulSet hoặc một dịch vụ được quản lý; các node agents tự đăng ký qua HTTP.

Xem tài liệu chi tiết để biết cách cấu hình các resource, scaling, và node discovery.
