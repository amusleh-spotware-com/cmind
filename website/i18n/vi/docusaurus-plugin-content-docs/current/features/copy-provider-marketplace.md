# Provider marketplace

Providers tạo public profile hiển thị strategy, performance, fee. Followers discover và subscribe tới providers.

## Provider profile

Người cung cấp có thể:

- Set public profile name + description.
- Show performance stats (ROI, Sharpe, drawdown).
- Publish performance fees (% of profit).
- Publish rules (min/max balance, trading hours).

Profile linked từ dashboard + marketplace page.

## Marketplace discovery

**Copy Trading → Marketplace** (nếu enabled):

- Browse providers.
- Filter bởi ROI, Sharpe, drawdown.
- Read reviews.

Click để subscribe.

## Subscription

Sau subscribe, setup destinations như bình thường. Marketplace UI prefills provider's published rules.

Fee tracked + reported trên settlement + tax docs.

Xem [copy-performance-fees.md](./copy-performance-fees.md) cho fee calculation.
