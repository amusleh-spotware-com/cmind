# Για ποιον είναι το cMind;

Σύντομη έκδοση: **άνθρωποι με χρήματα επί τοις εκατό που τους αρέσει να ελέγχουν.** Το cMind είναι μια σοβαρή
κονσόλα διαχείρησης αλγοριθμικής διαπραγμάτευσης — όχι ένα καταναλωτικό fintech παιχνίδι με μια κίνηση συσσώρευσης κομφετί όταν
κάνετε 3 δολάρια. Είτε διαπραγματεύεστε, χρεώνετε ή φιλοξενείτε, υπάρχει ένα μονοπάτι παρακάτω κατασκευασμένο για εσάς.

## 📈 Έμποροι — αυτο-κεντρικό όλο το γραφείο σας

Διαπραγματεύεστε σε cTrader και κουράστηκε να κολλάτε μαζί έναν επεξεργαστή, μια δοκιμή, ένα VPS, και τρεις καρτέλες περιηγητή. Αυτο-κεντρικό cMind και λάβετε συγγραφή, δοκιμή, ζωντανή εκτέλεση, αντιγραφή διαπραγμάτευσης, και παρακολούθηση σε μια κονσόλα AI-ενισχυμένη — στο δικό σας κουτί, τα δεδομένα σας ποτέ εγκαταλείποντας αυτό.

**Διαβάστε:** [cMind για εμπόρους cTrader →](./for-traders.md)

## 🏦 Χρεώστες — λευκό-ετικέτα για τους πελάτες σας

Τρέχετε ένα brokerage cTrader και θέλετε ένα πλεονέκτημα έναντι ανταγωνιστών που δεν προσφέρουν τίποτα αλλά ένα τερματικό. Ναυτιλίστε
cMind ως το δικό σας προϊόν: δώστε πελάτες AI, αντιγραφή διαπραγμάτευσης, και προκλήσεις prop-firm σύμφωνα με το δικό σας εμπορικό σήμα, περιορίστε
λογαριασμούς στο βιβλίο σας, και ανοίξτε νέες σειρές εσόδων.

**Διαβάστε:** [cMind για χρεώστες cTrader →](./for-brokers.md) ·
[Λευκό-ετικέτα για επιχειρήσεις →](./white-label-for-business.md)

## 🖥️ Παροχείς Cloud & VPS — πουλήστε ως διαχειρισμένη φιλοξενία

Νοικιάζετε υπολογισμό. Προσφέρετε διαχειρισμένη φιλοξενία cMind και προσγειωθείτε ένα υψηλής αξίας, υπολογισμού-πεινάσμενο, κολλώδες φορτίο εργασίας —
και νομιμοποιήστε τη συνδρομή, το μετρημένο υπολογισμό, το λευκό-ετικέτα, και το AI.

**Διαβάστε:** [cMind για παροχείς cloud & VPS →](./for-cloud-providers.md)

## 🧑‍💻 Συνεισφέροντες — βοηθήστε να το κάνετε καλύτερο

Το cMind είναι ανοικτή πηγή (MIT) και κοινότητα-σχηματισμένο: C# 14 / .NET 10, αυστηρό DDD, EF Core + PostgreSQL,
.NET Aspire, ένας MCP server, και τρεις δοκιμαστικές τομεάς. Κάθε ατομικότητα παραπάνω μπορεί — και θα πρέπει — να το προωθήσει
προς τα εμπρός:

- **Έμποροι:** προσθέστε πρότυπα cBot, ζητήστε δυνατότητες, αναφέρετε σφάλματα.
- **Χρεώστες:** συνεισφέρετε τις ενσωματώσεις και τους ελέγχους που χρειάζεται το γραφείο σας.
- **Παροχείς:** ανάδρομη τα λειτουργικά διορθώματα και τα βελτιώματα IaC σας.

Ανοίξτε ζητήματα, προσθέστε μεταφράσεις UI ή προσαρμογές παροχέα AI, και στείλτε PRs — ο
[Οδηγός συνεισφοράς](./contributing.md) και τα [έγγραφα MCP server](./features/mcp.md) σας ξεκινούν.

## Είναι το cMind *όχι* για εσάς;

Είστε ειλικρινής με τον εαυτό σας. Το cMind ίσως να είναι περισσότερο εάν:

- Ποτέ δεν έχετε αγγίξει ένα τερματικό και δεν θέλετε να ξεκινήσετε — αν και ένας
  [παροχέας cloud/VPS μπορεί να το φιλοξενήσει για εσάς](./for-cloud-providers.md).
- Μόνο διαπραγματεύεστε χειροκίνητα και δεν ενδιαφέρεστε για bots, backtests, ή αντιγραφή διαπραγμάτευσης.
- Θέλετε ένα φιλοξενημένο SaaS με μια γραμμή υποστήριξης hotline και δεν θέλετε κανείς αυτο-φιλοξενία — το cMind είναι
  αυτο-φιλοξενία ως σχεδιασμό (προσπαθεί απλώς πολύ σκληρά να το κάνει απόπειρα).

Ακόμα κακφησία κατά μήκος; Καλώς ήρθατε ομάδα. → [Τρέξτε το τοπικά](./deployment/local.md)
