# Добро дошли у cMind 👋

Дакле, желите да правите трговинске ботове, да их бектестирате без топљења лаптопа, да их покрећете на
више машина, да пресликавате трговине на десетине налога и да пустите вештачку интелигенцију да пази на
ризик док спавате. **На правом сте месту.**

cMind је **платформа отвореног кода за трговинске операције за cTrader, коју можете сами хостовати**.
Замислите је као цео свој трговински сто — писање, извршавање, рачунарску флоту, копирање трговања и
језгро вештачке интелигенције — упаковано у једну мирну, тамну, мобилно оријентисану апликацију коју
поседујете од почетка до краја.

:::tip У једној реченици
Градите → бектестирајте → покрећите → копирајте своје cTrader стратегије у великом обиму, са уграђеном
вештачком интелигенцијом, на сопственим серверима и под сопственим брендом.
:::

## Пут за 5 минута ⏱️

Ако имате Docker и пет минута, можете се одмах поиграти са правом cMind инстанцом:

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

Затим отворите **<http://localhost:8080>**, пријавите се и крените. Комплетан водич (са решавањем
проблема за када Docker неизбежно буде имао своје мишљење) налази се у
**[Локално покретање](./deployment/local.md)**.

## Кратка реч о новцу 💸

cMind покреће **прави капитал**. Схватамо то озбиљно — свака промена се испоручује са јединичним,
интеграционим и end-to-end тестовима, укључујући путеве отказа (прекинуте везе, одбијени налози, мртви
чворови). И ви бисте то требало да схватите озбиљно: **прво тестирајте на демо налогу** и прочитајте
[напомене о усклађености](./features/compliance.md) пре него што га усмерите на било шта право. Трговање
је ризично; овај софтвер је алат, а не финансијски савет.
