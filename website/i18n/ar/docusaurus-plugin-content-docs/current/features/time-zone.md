# المنطقة الزمنية

يُعرض كل وقت في التطبيق حسب منطقتك الزمنية، لا منطقة الخادم. يُحفَظ اختيارك في ملفك الشخصي ويتبعك عبر الأجهزة.

عند أول زيارة يتبنى التطبيق منطقة متصفحك تلقائيًا. يمكنك تغييرها في أي وقت من الإعدادات ← المنطقة الزمنية؛ والافتراضي للنشر هو خيار العلامة البيضاء App:Branding:DefaultTimeZone (الافتراضي UTC). تُخزَّن الأوقات وتُعاد من واجهة API دائمًا بتوقيت UTC — يُحوَّل العرض فقط.

- ترتيب التحديد: منطقة ملفك الشخصي، ثم ملف تعريف الارتباط، ثم افتراضي النشر، ثم UTC.
- يعمل الاكتشاف مرة واحدة ولا يتجاوز منطقة اخترتها.
- التنسيق يتبع لغتك؛ التسميات النسبية مثل «قبل دقيقتين» لا تتأثر.
