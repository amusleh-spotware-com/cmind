# دفتر_التداول_و_المدرب

الفئة_الأحدث_والفعلياً_المفيدة_لـ_AI-for-trading_ليست_التنبؤ_بالسوق_—_إنها_
تحليل *سلوكك*._دفتر_التداول_يحول_تاريخك_من_التشغيلات_واختبارات_الخلفية_
إلى_تعليقات_صادقة_حتى_تستطيع_تحسين_الاستراتيجية_التي_لديها_بالفعل.

افتح **AI → Trading Journal** (`/journal`).

## ماذا_يُظهر

من_حالاتك_(التشغيلات_واختبارات_الخلفية)_يحسب_،_بشكل_حتمي:

- **عد_الربح/الخسارة/الإخفاق_ومعدل_الربح** _عبر_اختباراتك_الخلفية؛
- **الرؤى_السلوكية** —_التسريبات_التي_تكلف_متداولي_التجزئة_بصمت:
  - **التركيز_المفرط** —_معظم_نشاطك_في_رمز_واحد؛
  - **الإخفاقات_المتكررة** —_حصة_عالية_من_التشغيلات_فشلت_في_البناء_
    أو_التكوين؛
  - **تحيز_الخسارة** —_اختبارات_خلفية_خاسرة_أكثر_من_الرابحة_(مع_دفع_
    لتشغيل_مختبر_النزاهة_والتأكد_من_أن_الحافة_حقيقية)؛
  - شهادة_صحة_نظيفة_عندما_لا_ينطبق_أي_من_الأعلى.

```http
GET /api/journal
```

## لماذا_هو_موثوق

التحليل_السلوكي_هو_كود_مجال_نقي_وحتمي_(`Core.Journal`)_بدون_اعتماد_
على_البنية_التحتية_—_مختبر_بالوحدة_للتركيز_المفرط_و_الإخفاقات_المتكررة_
و_تحيز_الخسارة_و_الحالة_المتوازنة_و_الحساب_الفارغ._الحقائق_تأتي_أولاً؛_
مدرب_AI_(Portfolio_Digest)_هو_طبقة_سرد_اختيارية_في_الأعلى_،_محصورة_
بمفتاح_Anthropic_API_،_لذا_الدفتر_يعمل_بشكل_كامل_بدون_تكوين_AI.
