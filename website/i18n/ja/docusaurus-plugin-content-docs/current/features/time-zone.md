# タイムゾーン

アプリが表示するすべての時刻は、サーバーではなくご自身のタイムゾーンで表示されます。選択内容はプロフィールに保存され、デバイス間で引き継がれます。

初回訪問時、アプリはブラウザーのタイムゾーンを自動的に採用します。設定 → タイムゾーンからいつでも変更できます。デプロイの既定はホワイトラベルオプション App:Branding:DefaultTimeZone（既定は UTC）です。時刻は常に UTC で保存され API から返され、表示のみが変換されます。

- 解決順序: プロフィールのゾーン、次に Cookie、次にデプロイの既定、最後に UTC。
- 検出は一度だけ実行され、ご自身が選んだゾーンを上書きしません。
- 書式は言語に従います。「2 分前」のような相対表記は影響を受けません。
