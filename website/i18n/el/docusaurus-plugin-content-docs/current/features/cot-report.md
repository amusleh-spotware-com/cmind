# Commitment of Traders (COT)

Το cMind παρέχει ένα ενσωματωμένο αναφορά **Commitment of Traders** — την εβδομαδιαία ανάλυση CFTC σχετικά με ποιος έχει θέση long και short στην αγορά ΗΠΑ futures (εμπορικοί hedgers, μεγάλοι speculators, κεφάλαια), με διαδραστικά ιστορικά γραφήματα, ένα κανονικοποιημένο **δείκτη COT**, ένα πιστοποιημένο REST API για cBots και εργαλεία MCP για clients τεχνητής νοημοσύνης. Τα δεδομένα προέρχονται απευθείας από τα **δημόσια σύνολα δεδομένων CFTC Socrata** — χωρίς κλειδί API, χωρίς aggregator. Όπως το οικονομικό ημερολόγιο, είναι ένα αποσυνδεδεμένο module που μπορεί να απενεργοποιηθεί χωρίς καμία επίδραση στον πυρήνα trading.

## Τι σας δίνει

- **Όλες τις τρεις οικογένειες αναφορών, μόνο futures και futures + options συνδυασμένα:**
  - **Κληρονομιά (Legacy)** — Μη-εμπορική (μεγάλοι speculators), Εμπορική (hedgers), Μη-αναφέρεται.
  - **Λεπτομερής κατανομή (Disaggregated)** — Παραγωγός/Έμπορος, Swap Dealers, Managed Money, Άλλα αναφερόμενα.
  - **Traders in Financial Futures (TFF)** — Dealer, Asset Manager, Leveraged Funds, Άλλα αναφερόμενα.
- **Ένας επιμελημένος κατάλογος αγορών** — Κύρια ζεύγη FX, χρυσό/ασήμι/χαλκό, αργό και φυσικό αέριο, Treasuries, δείκτες μετοχών, κρυπτονομίσματα και κύριες σοροί/soft commodities — καθένα αντιστοιχισμένο στον σταθερό κωδικό σύμβασης CFTC και, όπου αμφίβολο, σε ένα σύμβολο που μπορεί να διαπραγματευθεί (π.χ. Euro FX → `EURUSD`, Χρυσό → `XAUUSD`).
- **Ο δείκτης COT (0–100)** — όπου βρίσκεται η τρέχουσα καθαρή θέση του speculator μέσα στο ιστορικό του εύρος (προεπιλογή ~3 χρόνια). Οι αναγνώσεις κοντά στα ακραία σημεία σηματοδοτούν συνωστισμένη θέση που συχνά προηγείται μιας αναστροφής· η αναφορά επισημαίνει ένα **μακρύ άκρο** (≥80) ή **σύντομο άκρο** (≤20).
- **Ορθότητα σημείου στο χρόνο.** Μία εβδομαδιαία αναφορά μετράται την Τρίτη αλλά δημοσιοποιείται μόνο την επόμενη Παρασκευή; κάθε ανάγνωση τιμά τη στιγμή κυκλοφορίας, έτσι ένα σήμα θέσης backtested δεν βλέπει ποτέ μια αναφορά πριν να δημοσιευθεί (χωρίς look-ahead).

## Χρήση της σελίδας

Ανοίξτε **Commitment of Traders** από την αριστερή πλοήγηση. Επιλέξτε μια **αγορά**, έναν **τύπο αναφοράς** (Legacy / Disaggregated / Financial) και αλλάξτε **Futures + options** για εναλλαγή μεταξύ futures μόνο και του συνδυασμένου παραλλαγής. Η σελίδα δείχνει:

- **Καθαρή θέση με την πάροδο του χρόνου** — ένα διαδραστικό γράφημα γραμμής της καθαρής θέσης (long − short) κάθε κατηγορίας trader σε ολόκληρο το ιστορικό παράθυρο.
- **Δείκτης COT** — ένα γράφημα γραμμής του δείκτη 0–100, με την πιο πρόσφατη ανάγνωση και την ακραία ετικέτα του.
- **Πρόσφατο στιγμιότυπο** — ένας πίνακας long / short / net / % ανοιχτού ενδιαφέροντος ανά κατηγορία trader, συν συνολικό ανοιχτό ενδιαφέρον και ημερομηνία αναφοράς.

Κάθε διάγραμμα διαθέτει κουμπιά γραμμής εργαλείων **μεγέθυνση / σμίκρυνση** (και επαναφορά), και μπορείτε να σύρετε στον άξονα χρόνου για να μεγεθύνετε. **Λήψη CSV** κατεβάζει το πλήρες εβδομαδιαίο ιστορικό της επιλεγμένης αγοράς και τύπου αναφοράς ως αρχείο έτοιμο για υπολογιστικό φύλλο. Χρησιμοποιήστε **Σύγκριση αγορών** για να επικαλύψετε πολλές αγορές σε ένα διάγραμμα — τα διαγράμματα σύγκρισης σχεδιάζουν την καθαρή θέση του speculators και τον δείκτη COT κάθε επιλεγμένης αγοράς παράπλευρα, ώστε να μπορείτε να διαβάσετε την τοποθέτηση μεταξύ αγορών με μια ματιά.

## Πώς ρέουν τα δεδομένα

Η βάση δεδομένων είναι η κρυφή μνήμη. Ένας εβδομαδιαίος εργάτης πρόσληψης τραβά τα έξι σύνολα δεδομένων CFTC για τις παρακολουθούμενες αγορές, ενημερώνει τον κατάλογο αγορών και προσθέτει κάθε νέα αναφορά **ιδεμποτεντικά** (η επανεκτέλεση δεν δημιουργεί ποτέ διπλοτυπίες). Επιπλέον, τα δεδομένα **φορτώνονται κατ' απαίτηση**: την πρώτη φορά που ζητείται μια αγορά, λαμβάνεται από την πηγή CFTC και αποθηκεύεται, και κάθε επόμενο αίτημα εξυπηρετείται κατευθείαν από τη βάση δεδομένων. Η κρυφή μνήμη **ανανεώνεται όταν κυκλοφορούν νέες εβδομαδιαίες αναφορές** — μόλις η νεότερη αποθηκευμένη αναφορά είναι περισσότερο από μια εβδομάδα παλιά, το επόμενο αίτημα διαφανώς τραβά και προσθέτει τα τελευταία δεδομένα (περιορίζεται ώστε η πηγή να μη φορτώνεται ποτέ). Το πρώτο φορτίο γεμίζει πολλά χρόνια ιστορικού· μια διακοπή πηγής μειώνεται στην εξυπηρέτηση των καλύτερων δεδομένων κρυφής μνήμης. Όλα τρέχουν εκτός του κουτιού χωρίς κλειδί· ένα προαιρετικό token εφαρμογής Socrata αυξάνει μόνο το όριο ταχύτητας.

## Διαμόρφωση

Όλα τα κλειδιά βρίσκονται κάτω από `App:Cot` (δείτε [διακόπτες χαρακτηριστικών](./feature-toggles.md) και
[ρυθμίσεις ιδιοκτήτη λευκής ετικέτας](./white-label-owner-settings.md)):

| Κλειδί | Προεπιλογή | Σκοπός |
|-----|---------|---------|
| `IngestionEnabled` | `true` | Αν ο εβδομαδιαίος εργάτης πρόσληψης εκτελείται. |
| `PollInterval` | `6h` | Πόσο συχνά ο εργάτης ψηφοφορεί τα σύνολα δεδομένων CFTC. |
| `BackfillYears` | `5` | Χρόνια ιστορικού που αποσύρονται κατά την πρώτη εκτέλεση. |
| `ReconcileLookbackWeeks` | `4` | Πρόσφατες εβδομάδες που ανασυγχρονίζονται κάθε κύκλο για να συλλάβουν αναθεωρήσεις. |
| `SocrataAppToken` | — | Προαιρετικό token που αυξάνει το ανώνυμο όριο ταχύτητας. |
| `CotIndexLookbackWeeks` | `156` | Εβδομαδιαίες αναφορές που χρησιμοποιούνται ως εύρος δείκτη COT (~3 χρόνια). |

## Gating

Η ορατότητα είναι ένα δύο-επίπεδο gating, πανομοιότυπο με το οικονομικό ημερολόγιο: το hard gating λευκής ετικέτας `App:Branding:EnableCot` (επίπεδο build) **και** ο διακόπτης χαρακτηριστικού runtime `App:Features:Cot`. Με καθένα ανενεργό, η σύνδεσμος πλοήγησης, η σελίδα, το REST API και τα εργαλεία MCP όλα εξαφανίζονται (το API επιστρέφει `404`). Επειδή η πηγή δεδομένων είναι χωρίς κλειδί, δεν υπάρχει gating κλειδιού πηγής δεδομένων — enabled σημαίνει ορατό.

## Για προγραμματιστές

- Τομέας: `Core.Cot` — aggregates `CotMarket` και `CotReport`, value object `CotPositions`, domain service `CotIndexCalculator` και ports `ICotReports` / `ICotSource`.
- Υποδομή: `Infrastructure.Cot` — anti-corruption parser `CftcSocrataSource`, rate gate, append-only write service, read side και εβδομαδιαίος εργάτης πρόσληψης (EF `cot` schema).
- Πρόσβαση cBot και AI: [COT cBot API](./cot-cbot-api.md) (REST, JWT `market:read`) και MCP tools
  `CotMarkets`, `CotLatest`, `CotHistory`, `CotHealth`.
