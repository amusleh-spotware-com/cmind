# cMind へようこそ 👋

トレーディングボットを作り、ノートPCを溶かさずにバックテストし、複数のマシンで実行し、十数個の口座へ
取引をミラーリングし、眠っている間も AI にリスクを見張らせたい。**まさにうってつけの場所です。**

cMind は **cTrader 向けのオープンソースかつセルフホスト可能なトレーディング・オペレーション・プラット
フォーム**です。作成、実行、コンピュートのフリート、コピートレード、そして AI コア — あなたのトレーディ
ングデスク一式を、静かでダークな、モバイル対応のアプリにまとめ、端から端まであなたが所有します。

:::tip 一文で言うと
自分のサーバー上で、自分のブランドのもとに、AI を組み込んで、cTrader 戦略を大規模に 構築 → バックテスト
→ 実行 → コピー します。
:::

## 実際に何ができる？

| やりたいこと | cMind の対応 | 詳しく |
|---|---|---|
| ブラウザで cBot を書く | Monaco IDE + C#/Python テンプレート、サンドボックスビルド | [構築とバックテスト](./features/build-and-backtest.md) |
| 複数マシンでバックテスト | 自己修復するノードフリートが最も空いているマシンを選ぶ | [スケーリング](./deployment/scaling.md) |
| 1 口座を多数へコピー | 二重発注のない、再同期対応の堅牢なミラーリング | [コピートレード](./features/copy-trading.md) |
| 面倒な作業を AI に任せる | 戦略生成、自己修復、リスクガード、事後分析 | [AI コア](./features/ai.md) |
| プロップファームの規則内に収める | ライブ・エクイティ追跡 + チャレンジ規則のシミュレーション | [プロップファーム](./features/prop-firm.md) |
| *あなたの*製品として提供 | 完全ホワイトラベル：名称、配色、ロゴ、ファビコン | [ホワイトラベル](./features/white-label.md) |
| スマホで実行 | インストール可能なモバイルファースト PWA | [PWA](./features/pwa.md) |
| AI クライアントから操作 | 組み込み MCP サーバー（HTTP + SSE） | [MCP](./features/mcp.md) |

## 5 分コース ⏱️

Docker と 5 分があれば、今すぐ本物の cMind インスタンスを触り始められます：

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

次に **<http://localhost:8080>** を開いてサインインすれば、もう始められます。（Docker が必ず何か言って
くるときのトラブルシューティングを含む）完全なウォークスルーは
**[ローカルで実行](./deployment/local.md)** にあります。
