# الاختبار_بالتوتر

مجموعة_الإجهاد._تدق_أجزاء_التطبيق_التي_يمكن_أن_تكلّف_المستخدمين_
أموالاً_—_خاصة **نسخ_التداول** _بحمل_عدائي_ومُعشَّم_ومُحقن_بالأخطاء._
تؤكد_بقاء_النظام_صحيحاً._تعيش_في_`tests/StressTests`،_تشغل_في_
بوابة_`dotnet test` _الخضراء_الافتراضية.

## النهج_—_الاختبار_المحاكي_الحتمي_(DST)

أفضل_طريقة_لاختبار_أنظمة_مالية_موزعة_=_ **الاختبار_المحاكي_الحتمي**،
لكل_TigerBeetle،_FoundationDB،_Antithesis:_تشغل_منطق_حقيقي_مقابل_
*عالم_مُحاكى*،_تدفع_بحمل_ **بذروي** _عشوائي_+_الأخطاء_المُحقنة،
تؤكد_الثوابت_عند_الاستقرار._كل_البذوري_و_الحتمية_→_أي_
فشل_يُعيد_إنتاج_التشابك_بالضبط_من_البذرة._مُجمَّعة_مع:

- **حقن_خطأ_هندسة_الفوضى** (أسلوب_Netflix_Chaos_Monkey)_—_
  انقطاعات_الاتصال،_رفض_الأوامر،_تدوير_الرمز،_موت_العقدة.
- **ثوابت_مبنية_على_الخصائص** —_لا_تؤكد_تسلسلات_الاستدعاء_ال确切؛_
  تؤكد_خصائص_يجب_أن_تصمد_بغض_النظر_عن_كيف_تتخلّف_الأحداث_
  (التقارب،_لا_أيتام،_على_الأكثر_واحد_يحمل_الاستئجار).

التطبيق_يشحن_عالماً_مثالياً_لـ_DST:_`FakeTradingSession`،
جلسة_Open_API_في_الذاكرة_المؤمنة_بـ_cTrader._مجموعة_الإجهاد_
تعيد_استخدامه_(مرتبط،_مصدر_حقيقة_واحد)_لا_محاكاة_،_لذا_
الوسيط_المُحاكى_يتbehavior_مثل_الوسيط_الحقيقي.

## ماذا_تغطي

### نسخ_التداول_(التركيز_الأساسي)

تُدفع_عبر_`CopyDstWorld` (`tests/StressTests/CopyTrading/`)_،
تشغل_`CopyEngineHost` _الحقيقية_مقابل_جلسة_الأصلي،_تصدر_
حمل_مصدر_متسق_العضوية:

| السيناريو | يُدق |
|---|---|
| `Mass_fan_out…` | 1_مصدر_→_80_وجهة،_150_فتح_ثم_إغلاق؛_Fan-out_كامل_+_تفريغ |
| `High_frequency_open_close…` | 300_فتح/إغلاق_متداخلة_سريعة؛_لا_مواقع_مسرّبة |
| `Partial_close_and_scale_in_storm…` | عاصفة_إغلاق_جزئي_+_scale-in؛_استقرار_مجموعة_التسميات |
| `Connection_flap_storm…` | قطع_/إعادة_اتصال_مقبس_مُكرر_+_عدم_التزامن_في_منتصف_الرحلة؛_تقارب_المصادقة |
| `Order_rejection_cascade…` | مجموعة_فرعية_ترفض_كل_أمر؛_الوجهات_الصحية_لا_تتأثر،_ثم_تشفى_ذاتها_عبر_المصادقة |
| `Token_rotation_storm…` | تبديلات_رمز_سريعة_في_المكان_خلال_عاصفة_أمر |
| `Randomized_chaos_workload…` (10_بذور) | **قلب_DST** —_كل_نوع_حدث__+_كل_خطأ_متداخلة_غير_متوقعة |
| `CopyLeaseReclaimStressTests` | موت_العقدة_+_استرداد_الاستئجار_عبر_عنقود_مُوسع_(مجال_نقي،_`FakeTimeProvider`) |

**ثابت_التقارب.**_في_الراحة،_كل_وجهة_صحية_تُنسخ_بالضبط_مجموعة_
مواقع_المصدر_المفتوحة_—_لا_أيتام،_لا_مفقودات._يُؤكد_على_ *مجموعة*
التسمية_(scale-in_يفتح_legitimately_موقع_وجهة_ثانٍ_تحت_نفس_
تسمية_المصدر_،_لذا_التسميات_المكررة_مُتوقعة)._الوجهة_الحالية_
ترفض_الأوامر_يُسمح_لها_بالتأخر،_تُصادَق_بمجرد_الشفاء.

**ثابت_الاستئجار.**_في_عنقود_حيث_العقد_تموت_وتُحيى_حسب_جدول_
مُبذور،_على_الأكثر_عقدة_واحدة_تحمل_الاستئجار_الساري_على_ملف؛_
الاستئجار_الميت_ينتهي_بالضبط_عند_انتهاء_الصلاحية،_يُسترد؛_
العنقود_الصحي_يستقر_مع_كل_ملف_مُحتجز_بواسطة_عقدة_واحدة_بالضبط._
يعكس_`CopyEngineSupervisor`'_ _من_predicate_ادعاء_ضد_أساليب_
الاستئجار_الخاص_بـ_`CopyProfile`_.

### سلامة_الخيط_من_الـ_harness

`FakeTradingSession` _أحادية_الخيط؛_حمل_الإجهاد_يُغيرات_من_خيط_
الاختبار_بينما_المضيف_يقرأ/يكتب_من_حلقة_الخلفية_الخاصة_به._
`SyncTradingSession` _يلتف_عليها،_يجعل_كل_عملية_جلسة_ذرية_على_
بوابة_واحدة_(دون_الاحتفاظ_بالبوابة_عبر_استدعاء_العودة_→
سيف_العكس_ترتيب_القفل_مقابل_`_stateGate` _المضيف_و_ deadlock)._
المحاكي_نفسه_تُرك_دون_لمس.

## الأخطاء_المكتشفة

- **سباق_إعادة_المصادقة_عند_البدء_في_`CopyEngineHost`.**_`OnReconnected` _
  موصول_قبل_تحميل_المرجع_الأولي_+_أول_مصداقة_،_التي_شغلت_بدون_
  `_stateGate`._قطع_المقبس_خلال_البدء_شغل_ثاني_مصداقة_متزامن،
  أفسدت_قاموس_الحالة_غير_المتزامنة_للمضيف_(`_symbolDetails`،
  `_sourceVolumes`)._تم_الإصلاح:_تشغيل_التحميل_الأولي_+_أول_مصداقة_
  تحت_البوابة._سباق_إنتاجي_،_ليس_artefact_اختبار_—_حمل_الإجهاد_
  الفوضوي_كشفه.

## التشغيل

```bash
dotnet test tests/StressTests/StressTests.csproj
```

المجموعة **مُسلسلة** (`[assembly:_CollectionBehavior(DisableTestParallelization_=_true)]`):
كل_اختبار_يدور_حلقة_خلفية_حية_،_تدفع_إلى_الاستقرار_تحت_ساعة_
حائطية_،_لذا_التشغيل_المتوازي_يجوّع_مهام_المضيف_ويجعل_
فترات_التقارب_متقطعة._أحجام_الأحمال_مُصممة_للتوقف_في_ثوانٍ_
لذا_المجموعة_تبقى_في_بوابة_الإخضرار_الافتراضية._الفشل_يطبع_
بذرته؛_أعد_تشغيل_تلك_البذرة_لإعادة_إنتاج_التشابك_بالضبط.
