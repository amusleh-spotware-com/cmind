# cMind για σύννεφο & VPS προμηθευτές 🖥️

Είστε ήδη ενοικίαση υπολογισμού. cMind είναι ένα έτοιμο, ανοιχτή-πηγή προϊόν είστε μπορώ να τυλίξετε αυτό υπολογισμού
γύρω: **προσφορά διαχειριζόμενο cMind φιλοξενείες** και γη ένα υψηλή-αξία, κολλώδη, υπολογισμό-πεινώντας φορτίο —
αλγοριθμικό έμπορα, κομίστρες, prop φίρμες, και διαπραγμάτευσης κοινότητες ποιο θέλουν πλατφόρμα τρέχοντας
χωρίς γίνοντας το ops ομάδα τους.

:::tip TL;DR
Τρέχουν ο astateless σειρών + Postgres + ένα κόμβο στόλο. χέρι κλιέντες ένα branded URL. Μετατροπή το
συνδρομή, υπολογισμό, λευκό-ετικέτα, και AI. → [Ανάπτυξη σε το σύννεφο](./deployment/cloud.md)
:::

## Γιατί προσφορά διαχειριζόμενο cMind

- **Χωρίς κτίριο κόστος.** Είναι ανοιχτή πηγή, MIT-αδέσμευτος, και ήδη τεκμηριώνεται, δοκιμάστηκε, και containerized.
  Είστε πακέτο και λειτουργούν αυτό — είστε δεν κτίριο αυτό.
- **Α διαφοροποιημένο προϊόν για α κερδοφόρο κόλπο.** Algo διαπραγμάτευσης είναι υπολογισμό-πεινώντας: backtests και
  ζωντανό κόμβι καίω CPU, το οποίο είναι *χρησιμοποίηση του λογαριασμού* είστε ήδη πώλησης.
- **Κολλώδη κλιέντες.** Έμπορα ποιο κτίριο και τρέχουν στρατηγικές εσωτερικό πλατφόρμα δεν κουρέλη παραλήψη.
- **Στρέφει α προειδοποίηση σε ένα upsell.** cMind είναι αυτο-φιλοξενούμενη κατά σχεδιασμό — για κλιέντες ποιο "δεν θέλει σε
  να είναι το ops ομάδα," *σας* είναι η απάντηση.

## Ποιο αγορά διαχειριζόμενο cMind από σας

- **Ατομικό quants & έμπορα** ποιο θέλουν αυτό φιλοξενείες. → [Για έμπορα](./for-traders.md)
- **cTrader κομίστρες** τρέχοντας α λευκό-ετικέτα για δικό τους κλιέντες. → [Για κομίστρες](./for-brokers.md)
- **Prop φίρμες & αντιγραφή-διαπραγμάτευσης επιχειρήσεις** ποιο απαιτούν branded, ελεγκτική υποδομή.

## Τι "διαχειριζόμενο cMind" σημαίνει σε τρέχουν

Είστε λειτουργούν τρεις σειρών. ο κλιέντης παίρνει ένα branded ιστό URL:

| Σειρών | Τι αυτό είναι | Όπου αυτό τρέχει |
|---|---|---|
| Astateless (Web + MCP) | Εφαρμογή + API + MCP διακομιστή | Κάθε κοντέινερ πλατφόρμα, autoscaled |
| Βάση δεδομένων | PostgreSQL | Διαχειριζόμενο Postgres (RDS / Ευέλικτος Διακομιστής / δικό σας δικό) |
| Κόμβο στόλο | Κτίρια & τρέχουν cTrader κοντέινερ | **VMs ή Kubernetes — χρειάζεται προνομιούχο Docker** |

:::warning Ένα πράγμα σε εμβέλεια πάνω μπροστά
Κόμβο πράκτορες κτίριο και τρέχουν cTrader κοντέινερ, έτσι τα απαιτούν **προνομιούχο Docker**. Αυτό απαγορεύει
serverless κοντέινερ runtimes (Azure κοντέινερ Εφαρμογές, AWS Fargate) *για πράκτορες* — τρέχουν αυτά σε
[Kubernetes](./deployment/kubernetes.md), α VM, ή EC2. Ο astateless σειρών τρέχει οπουδήποτε.
:::

Πραγματικό, αντιγραφή-ολόκληρη ανάπτυξη οδηγούς κάνουν αυτό συγκεκριμένο: [σύννεφο αναφορά](./deployment/cloud.md) ·
[Azure](./deployment/cloud-azure.md) · [AWS](./deployment/cloud-aws.md) ·
[Kubernetes](./deployment/kubernetes.md) · [Κλίμακα](./deployment/scaling.md).

## Πώς είστε μετατροπή αυτό

- **Διαχειριζόμενο φιλοξενείες συνδρομή.** Μηνιαία Κινητής / Ομάδα / Επιχείρηση σχέδια μέγεθος κατά κόμβο στόλο και
  backtest ταυτόχρονα.
- **Χρήση & υπολογισμό metering.** Λογαριασμό backtest-ώρες, ζωντανό-κόμβο-ώρες, και αποθήκευση — φυσικά metered
  από κοντέινερ στόλο είστε ήδη τρέχουν.
- **Λευκό-ετικέτα reseller σειρών.** Χρέωση περισσότερα για ένα πλήρη rebrand (λογότυπο, χρώματα, PWA,
  `ShowSiteLink=false`) και για ενεργοποίηση premium δυνατότητες μέσα
  [δυνατότητα toggles](./features/feature-toggles.md). → [Λευκό-ετικέτα](./features/white-label.md)
- **Διαχειριζόμενο AI.** Δεσμευμένο ένα προεπιλογή AI παροχέα κλειδί έτσι κάθε κλιέντη χρήστες παίρνουν AI με χωρίς ρύθμιση, και
  σημάδι πάνω χρήση — ή προσφορά φέρνουν-δικό-κλειδί. → [AI δυνατότητα](./features/ai.md)
- **Prop-firm & αντιγραφή-διαπραγμάτευσης έσοδο κοινή.** Ξενιάγηση φίρμες τρέχοντας προκλήσεις και επιδόσεων τέλη και
  πάρει ένα πλατφόρμα κοπή. → [Prop-firm](./features/prop-firm.md) ·
  [Επιδόσεων τέλη](./features/copy-performance-fees.md) ·
  [Παροχέα αγορά](./features/copy-provider-marketplace.md)
- **Ρύθμιση, onboarding & SLA.** Επισυνάπτω επαγγελματική υπηρεσίες και premium υποστήριξη.

## Multi-ενοικιαζόμενο μοτίβα

- **Ανάπτυξη-ανά-ενοικιαζόμενο (προτείνεται).** Ένα branded παράδειγμα ανά κλιέντη — ισχυρό απομόνωση,
  ανά-ενοικιαζόμενο κοσμήματος και βάση δεδομένων, ένα διακριτό κόμβο ένωση token ανά ενοικιαζόμενο. Κοσμήματος είναι ανάγνωση από
  `IOptionsMonitor`, έτσι κάθε παράδειγμα φέρει δικό του ταυτότητα.
  → [Multi-ενοικιαζόμενο κοσμήματος](./white-label-for-business.md#multi-ενοικιαζόμενο-ανά-κλιέντη-κοσμήματος) ·
  [Κόμβο ανακάλυψη](./operations/node-discovery.md)
- **Κοινή έλεγχος επίπεδο (προηγμένα).** Μονάδα πολλά παρουσίες από δικό σας δικό provisioning σταθμό, σπάρη
  κοσμήματος και δυνατότητες ανά ενοικιαζόμενο προγραμματικά.

## Metering χρήση για λογαριασμοί

Ένας ιδιοκτήτης/διαχειριστής-μόνο **`GET /api/usage`** τελικό σημείο επιστροφή ένα διαβάστε-μόνο περίληψη ένας παροχέα μπορώ δημοσκοπία και
λογαριασμό σε — χωρίς κάθε νέα τομέα ή αποθήκευση, αυτό προβολή υπάρχον κατάσταση:

```json
{
  "χρήστες": { "συνολό": 42 },
  "κόμβι": { "συνολό": 6, "online": 5 },
  "παρουσίες": { "συνολό": 1280, "backtestsRunning": 3, "runsRunning": 11 },
  "cbots": { "συνολό": 210 },
  "tradingAccounts": { "συνολό": 88 }
}
```

Δημοσκοπία αυτό ανά ενοικιαζόμενο ανάπτυξη σε μονάδα κάθισμα-βασισμένο, στόλο-βασισμένο, ή φορτίο-βασισμένο τιμολόγηση. Ζευγάρι με
[καταγραφή & παρατηρησιμότητα](./operations/logging.md) για λεπτότερη υπολογισμό metering.
