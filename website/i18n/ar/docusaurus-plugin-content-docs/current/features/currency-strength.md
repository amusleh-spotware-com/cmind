# قوة_العملة_الكلية_بالذكاء_الاصطناعي_و_مخطط_الآفاق

توفر_cMind **محرك_قوة_العملة_الكلية_المدعوم_بالذكاء_الاصطناعي_والحتمي_الرياضي**._
يرتب_كونكون_من_العملات_القابلة_للتكوين_—_العملات_الثمانية_الرئيسية_بالإضافة_إلى_
العملات_الناشئة_والمتداولة_—_حسب **القوة_الأساسية** _الحالية_،
ويطرح **مخطط_اتجاهي_مستقبلي** _لكل_زوج_على_أفق_مختار_(1M_/_3M_/_6M_/_12M)._
كل_ترتيب_وكل_تحيز_زوج_وكل_رقم_يُحسب_بالرياضيات_الحتمية_الخالصة_في_جوهر_
المجال؛_LLM_فقط *يجمع*_المدخلات_المستقبلية_التي_لا_تنشرها_البيانات_و *يشرح*
النتيجة_بالإنجليزية_البسيطة._لا_يخترع_أبداً_ترتيباً_أو_اتجاهاً_أو_رقماً.

> **محدودية_صادقة.**_الأساسيات_تتنبأ_بقيمة_متوسطة_إلى_طويلة_الأجل_جيداً_وبقيمة_
> قصيرة_الأجل_بشكل_سيء._عامل_هذا_كـ_مرشح_موضع_/_التقارب،_ **وليس**_كـ_إشارة_
> تحديد_التوقيت_قصير_الأجل._القراءات_بالقرب_من_الإصدارات_ذات_التأثير_العالي_
> (NFP/CPI/البنوك_المركزية)_ضجيج._ليست_مشورة_مالية.

## كيف_يعمل

1. **الأساسيات_الحالية_تأتي_من_التقويم_الاقتصادي،_وليس_من_LLM.**_الأرقام_الصلبة_
   —_أسعار_السياسة،_CPI_مقابل_الهدف،_GDP،_التوظيف،_ميزان_التجارة_—_وأحجامها
   **z-scores**_المفاجأة_مصدر_من **نقطة-الوقت** _من_[التقويم_الاقتصادي](./economic-calendar.md)
   وحدة_(FRED/BLS/BEA/ECB_ وجداول_البنوك_المركزية)._لقطة_تاريخية_لا_تصل_
   إلى_النظرية_المسبقة.
2. **LLM_يجمع_فقط_ما_لا_يستطيع_التقويم_نشره**_—_لكل_عملة:_المسار **الأمامي**
   (مسار_سعر_السياسة_المتوقع_بالنقاط_الأساسية،_اتجاه_التضخم-مقابل-الهدف،
   زخم_النمو)_و_ **جيوسياسي**
   outlook_(risk-on/off،_التعريفات،_المالية/الدين،_الانتخابات)_،
   بالإضافة_إلى_أي_أرقام_حالية_EM/exotic_التي_يفتقر_إليها_التقويم._JSON_صارم،
   التحقق_مدرك_للمستوى،_البحث_في_الويب_مفعل.
3. **المجال_يحسب_الترتيب_والمصفوفة_الأمامية_بشكل_حتمي.**_كل_محرك_يحصل_عليه_كـ
   **z-score_ضمن-المستوى**_(لذا_التضخم_50%_لا_يشوه_العملات_الرئيسية)_،
   winsorized_،
   موزون_بمجموع_إلى_مركب_،_ورتب_الأقوى_→_الأضعف_مع_كسر_التعادل_المستقر_ISO._
   الطبقة_الأمامية_تحمل_كل_مركب_على_طريقته_—
   `projected_=_current_+_horizonScale_·_Σ_trajectoryDriver·weight`_
   وتخطط_الفرق_المُقدّر_لكل_زوج_إلى_ **تحيز_اتجاهي**_(▲_ارتفاع_/_▬_محايد_/_▼_
   انخفاض)_مع_إقناع.
4. **LLM_يشرح** _الترتيب_وأعلى_نداءات_الزوج_بلغة_واضحة.

## الكونكون_المستوي_(الرئيسية_+_EM_+_المتداولة)

الكونكون **قابل_للتكوين_عند_النشر** (`App:CurrencyStrength:Universe`)_—_إضافة_
عملة_هي_إعداد_وليس_كود._كل_عملة_تحمل **مستوى** (`Major`/_`EmergingMarket`/_`Exotic`)_الذي_
يضبط_الوزن_وعرض_النطاق_الميت_وحد_الإقناع:

- **الرئيسيات** —_USD،_EUR،_GBP،_JPY،_AUD،_NZD،_CAD،_CHF_(قيادة_مستوى_السعر).
- **الأسواق_الناشئة** —_CNH،_INR،_BRL،_MXN،_ZAR،_KRW،_SGD،_PLN_(+_Skandi_NOK/SEK)_؛_
  carry_+_risk_+_الضعف_الخارجي_مرجحة_للأعلى،_ثقة_متوسطة.
- **المتداولة** —_TRY،_HUF،_CZK،_بالإضافة_إلى_HKD/SAR_المربوطة_بـ_USD_؛_
  ثقة_منخفضة،_نطاق_ميت_أعرض،_حد_إقناع_محدد. **العملات_المربوطة/_المُدارة_بثقل**
  (HKD،_SAR،_CNH)_تُحدد،_مسارها_يُخفّض_وزنها،
  وتخطيط_الزوج_لها_يُclampe_نحو_`Neutral`_لذا_لا_تُقرأ_الربط_كإشارة_
   حرة_التعويم.

لأن_الإحصاءات_الرسميّة_EM/exotic_أقل_تكراراً_ومعدّلة_ومنتنة_أحياناً،
الأرقام_التي_يجمعها_AI_تحمل **ثقة_لكل_مستوى** _تُعرض_كـ_شارة_الموثوقية.

## الوصول_البرمجي

نموذج_قراءة_مشترك_واحد_(`ICurrencyStrengthQuery`)_قابل_للوصول_بثلاث_طرق:

- **ذكاء_اصطناعي_في_التطبيق** —_يُحقن_مباشرة_(في_العملية)_في_ميزات_AI.
- **MCP** —_أداة_`currency_strength`(معاملات_`horizon`،_`tier`)_لعملاء/وكلاء_AI.
- **cBot_REST** —_`GET /api/market/v1/currency-strength/{latest,history,pair/{base}/{quote}}`،
  مؤمّن_بنفس_آلية_`CalendarJwt` _كـ_[calendar_cBot_API](./calendar-cbot-api.md)_مع_
  إضافة_نطاق_**`market:read`**._يسجل_cBot_عميل_واجهة_برمجة_بـ_`market:read`،
  يتبادل_معرفه_وسره_لرمز_مؤقت_قصير_عند_`POST /api/calendar/v1/token`،
  ويستدعي_النقاط_الطرفية_بـ_`Bearer`_token._لا_مخطط_رمز_ثانٍ،
  لا_سر_ثانٍ_—_رمز_مسروب_هو_قراءة_فقط_محدودة_السوق_قصيرة_الأجل_ومُلغاة.

انظر_[calendar_cBot_API](./calendar-cbot-api.md)_لتدفق_الرمز_ونموذج_النسخ_واللصق.
