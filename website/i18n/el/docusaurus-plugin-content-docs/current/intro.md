# Καλώς ήρθατε στο cMind 👋

:::warning Λογισμικό Alpha — δεν είναι έτοιμο για παραγωγή
Το cMind βρίσκεται σε ενεργή ανάπτυξη. Αναμένετε τραχιές γωνίες, σπαστικές αλλαγές μεταξύ εκδόσεων και λειτουργίες που ακόμα είναι σε εξέλιξη. **Χρειαζόμαστε δοκιμαστές κοινότητας, αναφορείς σφαλμάτων και πρώιμους συνεισφέροντες** που θα βοηθήσουν να το διαμορφώσουν. Αν αντιμετωπίσετε κάποιο πρόβλημα, [αναφέρετέ το](https://github.com/amusleh-spotware-com/cmind/issues/new?template=bug_report.yml) — τα σχόλιά σας από τον πραγματικό κόσμο είναι η πιο πολύτιμη συνεισφορά που μπορείτε να κάνετε τώρα.
:::

Θέλετε λοιπόν να φτιάχνετε bots συναλλαγών, να τα κάνετε backtest χωρίς να λιώσετε το laptop σας, να τα
τρέχετε σε πολλά μηχανήματα, να αντιγράφετε συναλλαγές σε δεκάδες λογαριασμούς και να αφήνετε μια ΤΝ να
προσέχει τον κίνδυνο ενώ κοιμάστε. **Είστε ακριβώς στο σωστό μέρος.**

Το cMind είναι μια **ανοικτού κώδικα, αυτο-φιλοξενούμενη πλατφόρμα λειτουργιών συναλλαγών για το
cTrader**. Σκεφτείτε το ως ολόκληρο το γραφείο συναλλαγών σας — δημιουργία, εκτέλεση, έναν στόλο
υπολογισμού, copy trading και έναν πυρήνα ΤΝ — συσκευασμένα σε μια ήρεμη, σκούρα, φιλική προς κινητά
εφαρμογή που σας ανήκει από άκρη σε άκρη.

:::tip Με μία πρόταση
Δημιουργήστε → backtest → εκτελέστε → αντιγράψτε τις στρατηγικές σας στο cTrader σε κλίμακα, με
ενσωματωμένη ΤΝ, στους δικούς σας διακομιστές και με τη δική σας επωνυμία.
:::

## Η διαδρομή των 5 λεπτών ⏱️

Αν έχετε Docker και πέντε λεπτά, μπορείτε να πειραματίζεστε με ένα πραγματικό instance του cMind αυτή τη στιγμή:

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

Στη συνέχεια ανοίξτε το **<http://localhost:8080>**, συνδεθείτε και ξεκινήσατε. Ο πλήρης οδηγός (με
αντιμετώπιση προβλημάτων για όταν το Docker αναπόφευκτα έχει απόψεις) βρίσκεται στο
**[Εκτέλεση τοπικά](./deployment/local.md)**.

## Μια γρήγορη κουβέντα για τα χρήματα 💸

Το cMind διακινεί **πραγματικό κεφάλαιο**. Το παίρνουμε στα σοβαρά — κάθε αλλαγή παραδίδεται με unit,
integration και end-to-end tests, συμπεριλαμβανομένων των μονοπατιών αποτυχίας (διακοπές σύνδεσης,
απορριφθείσες εντολές, νεκροί κόμβοι). Θα πρέπει να το πάρετε στα σοβαρά κι εσείς: **δοκιμάστε πρώτα σε
demo λογαριασμό** και διαβάστε τις [σημειώσεις συμμόρφωσης](./features/compliance.md) πριν το στρέψετε σε
οτιδήποτε πραγματικό. Οι συναλλαγές έχουν ρίσκο· αυτό το λογισμικό είναι εργαλείο, όχι οικονομική συμβουλή.
