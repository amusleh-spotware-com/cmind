# Беларабежена cMind за ваш посао 🏢

Водите проп фирму, радан берзе или услугу копирања трговања? cMind је грађен од првог дана да буде
**препродан као ваш сопствени производ**. Свака површина — име, логотип, фавикон, боје, чак и
инсталирива телефонска апликација — савијају се на вашу марку. Ваши клијенти виде *вашу* компанију. Без промена кода,
без вилке, само конф.

:::tip[РЕЗИМЕ]
Упутите `App:Branding` на ваше име, боје и логотип. Поново покрените. Готово. Пуна техничка референца живи
у [документу белабежене функције](./features/white-label.md).
:::

## Шта можете препребранд

| Површина | Шта се мења |
|---|---|
| **Име производа** | Текст апликације бара + наслов језичке картице |
| **Логотип & фавикон** | Ваше марке свуда, укључујући язичек браузера |
| **Боје** | Пуна палета — примарна, површине, боје статуса — протекче кроз цео UI *и* собствени CSS апликације преко дизајна токена |
| **Инсталирива апликација (PWA)** | Назив дода-на-дом-екран, икона и залена користи вашу марку |
| **Мета / SEO** | Опис и URL подршке су ваши |
| **Прилагођени CSS** | Убацити вашу сопствену полиш за последњих 5% |

Све задане вредности на задату cMind идентичност, па преписујете само шта вам је стало.

## 60-секундна ребранд

Поставите ово на вашој развоји (JSON конф или променљивих окружења):

```json
{
  "App": {
    "Branding": {
      "ProductName": "AcmeFX",
      "CompanyName": "Acme Markets Ltd",
      "SupportUrl": "https://support.acme.example",
      "LogoUrl": "/branding/acme-logo.svg",
      "FaviconUrl": "/branding/acme.ico",
      "PrimaryColor": "#2D7FF9",
      "SecondaryColor": "#1E63C8",
      "ShowSiteLink": false
    }
  }
}
```

Облик променљиве окружења: `App__Branding__ProductName=AcmeFX`. Боје се валидирају при покретању —
лоша хекс вредност неће покренути јасну поруку уместо приказивања сломљене странице. Лепо и
јасно, управо када то желите.

## Веза „Напајано од cMind"

По **подразумевању**, панелна прилика показује мала, укусна **"Напајано од cMind"** веза која
упутити посетиоце назад на ову веб страну. По подразумевању је јер смо поносни на пројекат и
то помаже другим трговцима да га пронађу — али то је **ваш позив**.

- **Задржи то** (подразумевано): сукобна кредитна веза на панелној прилици. Ништа вас не кошта, помаже пројекту.
- **Скриј то**: поставити `App__Branding__ShowSiteLink=false` и потпуно нестаје — савршено за
  комплетно беларабежену развој где је производ неумитно *ваш*.

Видети [документ белабежене функције](./features/white-label.md#powered-by-link) за управо где то
рендерира.

## Мултинајмерска, по-клијент брендирање

Пошто је брендирање само конф развоја, свака станарска развој може да нози своју идентичност. Покрените одвојену
инстанцу по клијенту, или вожење брендирања из ваше сопствене контролне равни — апликација је чита из
`IOptionsMonitor`, па чак и може позавити тему живо када опције промену.

Парови ово са:

- **[Прекидачи функција](./features/feature-toggles.md)** — одлучите које способности сваки станарски види.
- **[Правила проп-фирме](./features/prop-firm.md)** — наведи своја правила изазова са живом одрживањем капитала.
- **[Комисије за перформансе](./features/copy-performance-fees.md)** + **[тржиште добављача](./features/copy-provider-marketplace.md)** — монетизирајте копирање трговања.
- **[Усклађеност](./features/compliance.md)** — задржи записник за ревизију коју ће ваш регулатор питати.

## Средства & хостинг

Пустите ваш логотип/фавикон у Web апликација `wwwroot/branding/` (или упутите `LogoUrl`/`FaviconUrl`
на неки апсолутни URL). Развоје како вам одговара — [Docker](./deployment/local.md),
[Kubernetes](./deployment/kubernetes.md), [Azure](./deployment/cloud-azure.md), или
[AWS](./deployment/cloud-aws.md).
