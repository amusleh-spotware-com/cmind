# White-label cMind لعملك

تشغيل شركة prop أو مكتب الوسيط أو خدمة نسخ التداول؟ تم بناء cMind منذ اليوم الأول ليتم **إعادة بيعها كمنتج خاص بك**. كل سطح - الاسم والشعار والفافيكون والألوان وحتى التطبيق الهاتفي القابل للتثبيت - ينحني إلى علامتك التجارية. عملاؤك يرون *شركتك*. لا تغييرات الكود و لا fork و فقط الإعدادات.

:::tip TL;DR
اشير `App:Branding` في اسمك والألوان والشعار. إعادة تشغيل. تم. المرجع التقني الكامل يعيش في [وثيقة ميزة White-label](./features/white-label.md).
:::

## ما يمكنك إعادة تسمية

| السطح | ما يتغير |
|---|---|
| **اسم المنتج** | شريط التطبيق النص + عنوان التبويب المتصفح |
| **الشعار والفافيكون** | علاماتك في كل مكان و بما في ذلك علامة التبويب المتصفح |
| **الألوان** | لوحة كاملة - أساسية و الأسطح وألوان الحالة - تتدفق عبر واجهة المستخدم بأكملها *و* CSS الخاص بالتطبيق عبر رموز التصميم |
| **تطبيق قابل للتثبيت (PWA)** | إضافة إلى شاشة المنزل و اسم و رمز و splash استخدم علامتك التجارية |
| **Meta / SEO** | الوصف و دعم URL هي لك |
| **مخصص CSS** | حقن الملمع الخاص بك للآخر 5% |

كل شيء الافتراضي إلى هوية cMind الأساسي و لذا تستبدل فقط ما تهتم به.

## إعادة تسمية 60 ثانية

تعيين هذه على النشر الخاص بك (JSON config أو متغيرات environment):

```json
{
  "App": {
    "Branding": {
      "ProductName": "AcmeFX",
      "CompanyName": "Acme Markets Ltd",
      "SupportUrl": "https://support.acme.example",
      "LogoUrl": "/branding/acme-logo.svg",
      "FaviconUrl": "/branding/acme.ico",
      "PrimaryColor": "#2D7FF9",
      "SecondaryColor": "#1E63C8",
      "ShowSiteLink": false
    }
  }
}
```

شكل متغير البيئة: `App__Branding__ProductName=AcmeFX`. التحقق من الألوان في بدء التشغيل - قيمة hex سيئة تفشل الحذاء مع رسالة واضحة بدلاً من عرض صفحة مكسورة. جميل وعالي و بالضبط عندما تريد.

## رابط "مدعوم بـ cMind"

بشكل **افتراضي** لوحة التحكم تظهر صغيرة و ذوقية **"مدعوم بـ cMind"** رابط الذي يشير الزوار مرة أخرى إلى هذا الموقع. إنه بشكل افتراضي لأننا فخورون بالمشروع و يساعد المتاجرين الآخرين على العثور عليه - لكنها **اختيارك**.

- **احفظه** (افتراضي): رابط ائتمان دقيق على لوحة التحكم. لا يكلفك شيء و يساعد المشروع.
- **إخفاء:** تعيين `App__Branding__ShowSiteLink=false` و يختفي تماماً - مثالي للنشر white-label بالكامل حيث المنتج بلا شك *لك*.

انظر [وثيقة ميزة White-label](./features/white-label.md#powered-by-link) بالضبط حيث يعرض.

## Multi-tenant و per-customer branding

لأن العلامة التجارية هي فقط نشر إعدادات كل نشر عميل يمكن أن يحمل هويته الخاصة. تشغيل مثيل منفصل لكل عميل أو قيادة العلامة التجارية من لوحة التحكم الخاصة بك - التطبيق يقرأ من `IOptionsMonitor` لذا يمكن حتى إعادة بناء المظهر مباشرة عند تغيير الخيارات.

زوج هذا مع:

- **[مفاتيح الميزات](./features/feature-toggles.md)** — قرر أي إمكانيات كل عميل يرى.
- **[قواعس Prop-firm](./features/prop-firm.md)** — فرض قواعس التحدي الخاص بك مع تتبع الإنصاف الحي.
