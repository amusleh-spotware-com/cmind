# Хоризонтално скалирање

cMind скалира са минималним оператером напором. Два stateful оптерећења — покрени/тестирање
извршавање, копирање-трговања — обоје користе базу као координативна тачка, тако додавање реплика потребе
без спољног координатора (без ZooKeeper, без лидер избора).

## Копирање-трговања (самолечење лизинга)

Сваки чвор трчи `CopyEngineSupervisor` (закупљена на `App:Copy:Enabled`). Сваки помирење циклус,
надзорник:

1. **Захвата** сваки редовни профил неприписана *или* лизинга-истекла, у јединственој `UPDATE` —
   два надметања надзорника никад обоје захватају исти профил, тако профил копиран од управо један
   чвор (без двоструких налога).
2. **Обнови** лизингу на профилима то хостира.
3. Хостира приписане профиле, пушања приступ-жетон ротације до живог домаћина у месту (без
   тока нарушаја).

Пропаст чвора → престане обнавити; једном `App:Copy:LeaseTtl` пролази, било какви преживелац чвор претрази
његови профили следећи циклус, гради стање од помирења без дуплирања трговина. **Скалирање
из** = додај реплике; неприписани/слободни профили подигнуто аутоматски.

**Благо скалирање-у / котрљавајуће ажурирање (S1)** = на `SIGTERM`, `CopyEngineSupervisor.StopAsync`
**отпушта лизингу овог чвора** (`AssignedNode`/`LeaseExpiresAt` → нула) тако преживелац их претрази
его *веома наредну* циклус помирења — **не** после пуни `LeaseTtl`. Само брз крах чека TTL.
Копирање-агент је `terminationGracePeriodSeconds` (подразумевано 30) дає отпущања време за завршетак пре
подова убиј.

### Дугмићи (`App:Copy`)

| Постављање | Подразумевано | Напомене |
|---------|---------|-------|
| `Enabled` | `false` | Укљу копирање смештаја за чвор. |
| `ReconcileInterval` | `30s` | Колико често чвор захвата/обнови/помирење. |
| `LeaseTtl` | `120s` | Благо пре молчания чвора профили подигнуто. Держи неколико помирење интервала тако спора циклус не узрокује лажно передачу. |
| `NodeName` | машина име | Постави различито када два надзорника деле домаћин. |
