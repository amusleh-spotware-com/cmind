# Часовой пояс

Каждое время в приложении отображается в вашем часовом поясе, а не сервера. Ваш выбор сохраняется в профиле и следует за вами на разных устройствах.

При первом посещении приложение автоматически принимает пояс вашего браузера. Вы можете изменить его в любой момент в Настройки → Часовой пояс; значение по умолчанию для развёртывания — white-label опция App:Branding:DefaultTimeZone (по умолчанию UTC). Время всегда хранится и возвращается API в UTC — преобразуется только отображение.

- Порядок определения: пояс профиля, затем cookie, затем значение по умолчанию развёртывания, затем UTC.
- Определение выполняется один раз и никогда не переопределяет выбранный вами пояс.
- Форматирование следует вашему языку; относительные метки вроде «2 минуты назад» не затрагиваются.
