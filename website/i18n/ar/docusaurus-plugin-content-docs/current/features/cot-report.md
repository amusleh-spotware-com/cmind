# Commitment of Traders (COT)

تُرسل cMind تقرير **Commitment of Traders** المدمج — وهو تحليل الهيئة الفيدرالية التجارية الأمريكية CFTC الأسبوعي لمن لديهم صفقات طويلة وقصيرة في سوق العقود الآجلة الأمريكي (المتحوطون التجاريون، المضاربون الكبار، الصناديق)، مع رسوم بيانية تاريخية تفاعلية، **مؤشر COT** معياري، وAPI REST مصرح به لـ cBots وأدوات MCP لعملاء الذكاء الاصطناعي. تأتي البيانات مباشرة من **مجموعات بيانات CFTC العامة على Socrata** — بدون مفتاح API، بدون وسيط. مثل التقويم الاقتصادي، يعتبر وحدة نمطية منفصلة يمكن تعطيلها دون أي تأثير على نواة التداول.

## ما الذي يوفره لك

- **جميع عائلات التقارير الثلاث، العقود الآجلة وحدها والعقود الآجلة + الخيارات معاً:**
  - **الإرث (Legacy)** — غير تجاري (المضاربون الكبار)، تجاري (المتحوطون)، غير قابل للإبلاغ.
  - **المصنفة (Disaggregated)** — المنتج/التاجر، وسطاء المقايضة، الأموال المدارة، الآخرون المبلغ عنهم.
  - **المتاجرون في العقود الآجلة المالية (TFF)** — الوسيط، مدير الأصول، الصناديق ذات الرافعة المالية، الآخرون المبلغ عنهم.
- **كتالوج السوق المختار** — أزواج الفوركس الرئيسية، الذهب/الفضة/النحاس، الخام والغاز الطبيعي، سندات الخزانة، مؤشرات الأسهم، العملات المشفرة والحبوب الرئيسية/المحاصيل الناعمة — كل منها مرتبط برمز عقد CFTC المستقر، وحيث لا التباس فيه، بشكل رمز قابل للتداول (مثل Euro FX → `EURUSD`, الذهب → `XAUUSD`).
- **مؤشر COT (0–100)** — حيث تقع صافي مركز المضارب الحالي داخل نطاقه التاريخي (المظهر الافتراضي ~3 سنوات). تشير القراءات بالقرب من النقاط القصوى إلى تموضع مكتظ غالباً ما يسبق انعكاساً؛ يُسمي التقرير **قراءة عالية (≥80)** أو **قراءة منخفضة (≤20)**.
- **دقة النقطة الزمنية.** يتم قياس التقرير الأسبوعي يوم الثلاثاء لكن لا يصبح عاماً إلا يوم الجمعة التالي؛ كل قراءة تكرم لحظة الإصدار تلك، لذا فإن إشارة التموضع المختبرة لا ترى تقريراً قبل نشره (بدون نظر للأمام).

## استخدام الصفحة

افتح **Commitment of Traders** من التنقل الأيسر. اختر **السوق** و**نوع التقرير** (الإرث / المصنفة / المالية) والتبديل إلى **العقود الآجلة + الخيارات** للتبديل بين العقود الآجلة وحدها والمتغير المدمج. تعرض الصفحة:

- **التموضع الصافي بمرور الوقت** — رسم بياني خطي تفاعلي لموضع صافي كل فئة متداول (طويل − قصير) عبر نافذة المحفوظات.
- **مؤشر COT** — رسم بياني خطي للمؤشر 0–100، مع أحدث قراءة وتسميتها القصوى.
- **لقطة أحدث** — جدول طويل / قصير / صافي / % من الفائدة المفتوحة لكل فئة متداول، بالإضافة إلى إجمالي الفائدة المفتوحة وتاريخ التقرير.

## كيفية تدفق البيانات

يسحب عامل الامتصاص الأسبوعي مجموعات بيانات CFTC الست للأسواق المتتبعة، ويقوم بدمج كتالوج السوق ويضيف كل تقرير جديد **بشكل مثيري (لا يؤدي إعادة التشغيل أبداً إلى تكرار لقطة). يقوم التشغيل الأول بملء عدة سنوات من التاريخ؛ تقوم التشغيلات اللاحقة بإعادة مزامنة الأسابيع الأخيرة لالتقاط التعديلات المتأخرة. يعمل كل شيء من الصندوق بدون مفتاح؛ رمز تطبيق Socrata الاختياري فقط يرفع حد المعدل.

## التكوين

جميع المفاتيح تحت `App:Cot` (انظر [تبديلات الميزات](./feature-toggles.md) و
[إعدادات مالك العلامة البيضاء](./white-label-owner-settings.md)):

| المفتاح | الافتراضي | الغرض |
|-----|---------|---------|
| `IngestionEnabled` | `true` | ما إذا كان عامل الامتصاص الأسبوعي يعمل. |
| `PollInterval` | `6h` | عدد مرات استطلاع العامل لمجموعات بيانات CFTC. |
| `BackfillYears` | `5` | سنوات التاريخ المسحوبة في التشغيل الأول. |
| `ReconcileLookbackWeeks` | `4` | الأسابيع الأخيرة المعاد مزامنتها كل دورة لالتقاط التعديلات. |
| `SocrataAppToken` | — | رمز اختياري يرفع حد المعدل المجهول. |
| `CotIndexLookbackWeeks` | `156` | التقارير الأسبوعية المستخدمة كنطاق فهرس COT (~3 سنوات). |

## الحد

الرؤية هي بوابة ذات طبقتين، متطابقة مع التقويم الاقتصادي: بوابة العلامة البيضاء الثابتة `App:Branding:EnableCot` (مستوى البناء) **و** تبديل الميزة في وقت التشغيل `App:Features:Cot`. مع أيهما معطل، رابط التنقل والصفحة وAPI REST وأدوات MCP كلها تختفي (API يعيد `404`). لأن مصدر البيانات بدون مفتاح، لا توجد بوابة مفتاح مصدر البيانات — المفعل يعني مرئي.

## للمطورين

- المجال: `Core.Cot` — `CotMarket` و `CotReport` aggregates، كائن القيمة `CotPositions`، خدمة المجال `CotIndexCalculator`، والمنافذ `ICotReports` / `ICotSource`.
- البنية التحتية: `Infrastructure.Cot` — محلل مكافحة الفساد `CftcSocrataSource`، بوابة المعدل، خدمة الكتابة المضافة فقط، جانب القراءة وعامل الامتصاص الأسبوعي (مخطط EF `cot`).
- وصول cBot والذكاء الاصطناعي: [COT cBot API](./cot-cbot-api.md) (REST، JWT `market:read`) وأدوات MCP
  `CotMarkets`, `CotLatest`, `CotHistory`, `CotHealth`.
