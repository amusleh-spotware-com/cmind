# AI 매크로 통화 강도 및 전방향 전망

cMind는 **AI 보조, 수학적 결정론적** 매크로 통화 강도 엔진을 제공합니다. 구성 가능한 통화 범위 — 8개 주요 통화 + 신흥 시장 및 이국적인 통화 — 를 **현재** 기본 강도로 순위를 매기고, 선택한 기간(1M / 3M / 6M / 12M)에 걸쳐 모든 쌍에 대한 **전방향 전망**을 투영합니다. 모든 순위, 모든 쌍 편견 및 모든 수는 도메인 코어의 순수 결정론적 수학으로 계산됩니다; LLM은 오직 데이터가 게시할 수 없는 전방향 입력을 *수집*하고 결과를 일반 영어로 *설명*할 뿐입니다. 순위, 방향 또는 숫자를 결코 발명하지 않습니다.

> **정직한 한계.** 기본 요소는 중장기 가치를 잘 예측하고 단기 가치를 종종 예측하지 못합니다. 이를 포지셔닝 / 합류 필터로 취급합니다, **단기 타이밍 신호가 아닙니다**. 고영향 릴리스(NFP/CPI/중앙은행) 근처의 판독값은 잡음입니다. 재정 조언이 아닙니다.

## 작동 원리

1. **현재 기본 요소는 LLM이 아닌 경제 달력에서 제공됩니다.** 하드 수치 — 정책 금리, CPI vs 목표, GDP, 고용, 무역 수지 — 및 **놀라움 z-점수**는 **시점**에서 [경제 달력](./economic-calendar.md) 모듈(FRED/BLS/BEA/ECB 및 중앙은행 일정)에서 출처됩니다. 역사적 스냅샷은 전방향 보기를 유출하지 않습니다.
2. **LLM은 달력이 게시할 수 없는 것만 수집합니다** — 통화당: **전방향** 궤적 (예상 정책금리 경로 bp, 인플레이션 추세 vs 목표, 성장 모멘텀) 및 **지정학적** 전망 (위험 온/오프, 관세, 재정/부채, 선거), 달력이 부족한 EM/이국적인 현재 수치. 엄격한 JSON, 계층 인식 검증, 웹 검색 켜짐.
3. **도메인은 순위 및 전방향 행렬을 결정론적으로 계산합니다.** 각 드라이버는 **계층 내 z-점수**(50% 인플레이션 이국적인 절대 주요 수치를 왜곡하지 않음)로 점수가 매겨지고, 윈저화되고, 가중 합산되어 강함→약함으로 순위가 매겨지며 안정적인 ISO 동점 분할이 있습니다. 전방향 계층은 각 합성을 궤적을 따라 수행합니다 — `projected = current + horizonScale · Σ trajectoryDriver·weight` — 각 쌍의 투영 차이를 **방향 편견**(▲ 상승 / ▬ 중립 / ▼ 하강)과 확신으로 매핑합니다.
4. **LLM은** 순위 및 상위 쌍 호출을 일반 언어로 설명합니다.

## 드라이버

| 드라이버 | 강도에 미치는 영향 | 비고 |
|---|---|---|
| 정책 금리 및 궤적 | 더 높은 / 강경 ⇒ 강함 | 최고 가중치; 중앙은행 발산은 가장 큰 격차를 주도합니다. |
| 인플레이션 (CPI vs 목표) | 목표 이상 ⇒ 약함 | 역으로 점수 매김 (구매력 저항). |
| GDP 성장 | 상대적 성장이 높음 ⇒ 강함 | 패널 vs 차분. |
| 고용 | 더 강한 노동 ⇒ 강함 | 정책 경로를 공급합니다. |
| 무역 수지 / 경상 수지 | 흑자 ⇒ 강함 | 구조적 수요. |
| 정책 입장 | 강경 ⇒ 강함 | 주요 장기 드라이버. |
| 놀라움 모멘텀 | 최근 상승 ⇒ 강함 | 달력의 놀라움 z-점수에서. |
| 지정학적 / 위험 | 위험 오프 ⇒ 안전 피난처 (USD/JPY/CHF) 강함 | 제한된 전방향 위험 델타. |
| 실질 수익률 / 캐리 *(EM/이국적)* | 긍정적 실질 금리 ⇒ 강함 | 조용한 체제에서 EM 드라이버 지배. |
| 외부 취약성 *(EM/이국적)* | 적자 / 낮은 준비금 / USD 부채 ⇒ 약함 | 구조적 감가 압력. |
| 무역 조건 *(상품 수출국)* | 수출 가격 상승 ⇒ 강함 | BRL, ZAR, CLP, NOK, AUD, CAD. |
| 정치적 / 제도적 위험 *(EM/이국적)* | 불안정 ⇒ 약함 | 더 넓은 데드 밴드, 상한선 확신. |

## 계층화된 범위 (주요 + EM + 이국적)

범위는 **배포 구성 가능**(`App:CurrencyStrength:Universe`)입니다 — 통화 추가는 코드가 아닌 구성입니다. 각 통화는 가중치, 데드 밴드 너비 및 확신 상한선을 조정하는 **계층**(`주요` / `신흥시장` / `이국적`)을 전달합니다:

- **주요** — USD, EUR, GBP, JPY, AUD, NZD, CAD, CHF (금리 수준 주도).
- **신흥 시장** — CNH, INR, BRL, MXN, ZAR, KRW, SGD, PLN (+ 스칸디나비아 NOK/SEK); 캐리 + 위험 + 외부 취약성 가중화, 중간 신뢰.
- **이국적** — TRY, HUF, CZK, 더하기 USD 페깅 HKD/SAR; 낮은 신뢰, 더 넓은 데드 밴드, 상한선 확신. **고정 / 무겁게 관리** 통화 (HKD, SAR, CNH)는 표시되고, 궤적은 가중치가 낮아지며, 쌍 전망은 중립향 조정되어 페그가 자유 변동 신호로 읽혀지지 않습니다.

공식 EM/이국적 통계가 낮은 빈도, 개정되고 때때로 불명확하므로, AI 수집 수치는 신뢰성 배지로 표시되는 **계층별 신뢰를** 전달합니다.

## 우아한 저하

| 달력 | AI | 결과 |
|---|---|---|
| ✅ | ✅ | 전체 순위 + 전방향 투영 + 서술 (`CalendarAndAi`). |
| ✅ | ❌ | 달력만 현재 순위, 전방향 투영 없음 (`CalendarOnly`). |
| ❌ | ✅ | AI 수집 현재 수치 + 전방향, 낮은 신뢰 (`AiOnly`). |
| ❌ | ❌ | 스냅샷 없음 — 위젯은 숨어있고 페이지는 빈 상태를 표시합니다. |

앱은 어느 쪽이든 변경 없이 실행됩니다. AI는 AI 키에 게이트됩니다; 달력 레그는 자체 화이트 라벨 게이트 + 런타임 토글을 존경합니다.

## 사용하기

- **AI 활성화**(설정 → AI) 및 자신의 대시보드 **사용자 정의** 대화상자에서 **위젯을 켭니다** ("통화 강도" — 옵트인, 기본값으로 숨김). 위젯은 상위 강함/약함 통화 및 상위 3M 쌍 호출을 표시합니다; 전체 페이지로 링크됩니다.
- **전체 페이지** — `/ai/currency-strength`: 기간 선택기 (1M/3M/6M/12M), 계층 필터(모두/주요/EM/이국적), 현재 순위, 전방향 예측, 쌍 전망 행렬 (편견 + 확신, 페깅/낮은 신뢰 표시), 및 AI 서술. **지금 새로고침** (소유자)를 눌러 재생성합니다. 백그라운드 워커 (`App:CurrencyStrength:RefreshEnabled`, **기본값 `true`**)는 일정에 따라 새로고침하므로 페이지는 상자 밖에서 채워집니다; 배포 또는 소유자는 끕니다 (또는 AI / 경제 달력 기능을 비활성화하면 새로고침자는 스냅샷 없음으로 저하됩니다).

## 프로그래밍 방식 액세스

하나의 공유 읽기 모델 (`ICurrencyStrengthQuery`)은 세 가지 방식으로 도달 가능합니다:

- **앱 내 AI** — 직접 주입 (프로세스 내) AI 기능으로.
- **MCP** — `currency_strength` 도구 (매개변수 `horizon`, `tier`) AI 클라이언트/에이전트용.
- **cBot REST** — `GET /api/market/v1/currency-strength/{latest,history,pair/{base}/{quote}}`, [달력 cBot API](./calendar-cbot-api.md)와 추가 **`market:read`** 범위로 **동일한** `CalendarJwt` 기계장치로 보호됩니다. cBot은 `market:read`로 API 클라이언트를 등록하고, `POST /api/calendar/v1/token`에서 단기 JWT로 id + secret을 교환하고, `Bearer` 토큰으로 엔드포인트를 호출합니다. 두 번째 JWT 체계 없음, 두 번째 비밀 없음 — 누출된 토큰은 읽기 전용, 시장 범위, 단기 및 취소 가능합니다.

[달력 cBot API](./calendar-cbot-api.md)에서 토큰 흐름 및 복사 붙여넣기 샘플을 참조하십시오.
