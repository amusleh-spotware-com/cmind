# 시간대

앱이 표시하는 모든 시간은 서버가 아닌 사용자 본인의 시간대로 렌더링됩니다. 선택 내용은 프로필에 저장되어 여러 기기에서 따라갑니다.

첫 방문 시 앱이 브라우저의 시간대를 자동으로 채택합니다. 설정 → 시간대에서 언제든지 변경할 수 있으며, 배포 기본값은 화이트라벨 옵션 App:Branding:DefaultTimeZone(기본 UTC)입니다. 시간은 항상 UTC로 저장되고 API에서 반환되며 표시만 변환됩니다.

- 확인 순서: 프로필 시간대, 그다음 쿠키, 그다음 배포 기본값, 그다음 UTC.
- 감지는 한 번만 실행되며 사용자가 선택한 시간대를 덮어쓰지 않습니다.
- 서식은 사용자 언어를 따릅니다. 「2분 전」 같은 상대 표기는 영향을 받지 않습니다.
