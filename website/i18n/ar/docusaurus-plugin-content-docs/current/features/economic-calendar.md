# التقويم_الاقتصادي

توفر_cMind **تقويماً_اقتصادياً_الخاصة_بها**_—_جدول_الإصدارات،_الفعلي،
التوقعات،_المراجعات_ونموذج_التأثير_المبني_على_البيانات_—_مصدر_من **السلطات_الأصلية**
(البنوك_المركزية_والوكالات_الإحصائية_الوطنية)_،_مع **عدم_الاعتماد** على_ForexFactory،
FXStreet،_Investing.com_أو_أي_مجمّع._هو_صحيح_في_نقطة_زمنية_،
يحتفظ_بـ_≥_10_سنوات_من_التاريخ،_ومتصل_بالتداول،_API_العام،
MCP_،_cBots_،_AI_،_التنبيهات_واختبارات_backtest_._إنه_وحدة_مُفككة:_يمكن_تعطيله_بدون_
أي_تأثير_على_أساس_التداول.

> **الحالة.** تم تنفيذ وشحن P0–P4. النواة المجالية، والاستمرارية (مخطط EF `calendar`، القراءة/الكتابة الإلزامية، مصادر FRED و BLS وجداول البنوك المركزية، عامل الإدخال المحدود بالإعداد مع تتبع حداثة كل مصدر)، وواجهة JWT REST API الإصدارية، وواجهة `/economic-calendar` المحمولة أولاً، وأدوات MCP، وواجهة cBot JWT API، وتنبيهات الأحداث عالية التأثير، وإيقاف نسخ التداول عند وجود أخبار، وطبقة أحداث backtest، وتدفق SSE، ووصلات الويب الموقّعة بـ HMAC، وعميل `CmindCalendarClient` المكتوب — جميعها منفّذة ومُختبرة بالتكامل. إضافات P5 (تحليلات المفاجآت، تصدير iCal/CSV، بحث بالكلمات، إجماع قابل للإدراج) هي العناصر المتبقية — انظر مراحل الإصدار أدناه.

## ما_يجعله_مختلفاً

الشكاوى_المتكررة_ضد_التقويمات_الرائدة_أصبحت_قيود_التصميم_الخاصة_بنا:

- **لا_تغييرات_صامتة_في_تقييم_التأثير.**_تقييم_التأثير_لديها **حتمي_ومؤرشف_ومدقق**._
  كل_تغيير_هو_مراجعة_مسجلة_مع_طابع_زمني_—_وليس_كتابة_صامتة._يمكن_للمستخدم_أن_يرى_
  بالضبط *لماذا*_الحدث_مرتفع.
- **مرساة_UTC_واحدة_لكل_حدث.**_كل_حدث_مرساة_إلى_لحظة_UTC_واحدة_من_جدول_المصدر_
  الرسم_الرسمي؛_المنطقة_الزمنية_الخاصة_بالمصدر_مخزنة،_ويستخدم_التقديم_لكل_مستخدم_
  منطقة_زمنية_IANA_صريحة_مع_التوقيت_الصيفي_الذي_تتعامل_ معه_قاعدة_المنطقة_—_وليس_
  تبديل_±1h_يدوي.
- **سلاسل_مراجعة_كاملة،_في_كل_مكان.**_القيمة_الأصلية_وكل_مراجعة_هي_من_الدرجة_
  الأولى_ومكشوفة_بشكل_متطابق_عبر_API_و_MCP_وسطح_cBot.
- **≥_10_سنوات_من_التاريخ،_لا_جدار.**_نطاق_تصفح_غير_مقيد؛_لا_حد_60_يوما،_لا_بوابة_
  تسجيل.
- **نقطة_في_الوقت_بالبناء.**_كل_حقيقة_تحمل_`KnownAt`(متى *عرفنا*_هي)_و_`EffectiveAt`
  (لحظة_الحدث)._"كما_بدا_التقويم_في_الوقت_T"_هو_استعلام_من_الدرجة_الأولى،
  لذا_قاعدة_الأخبار_المُختبر_بالbacktest_تتسbehavior_بالضبط_مثل_الحية_—_لا_نظرية_مسبقة_
  من_استخدام_القيم_المعدلة_في_التاريخ.

## نموذج_التأثير

درجة_التأثير_هي_دالة_حتمية_نقية_في_`[0,_100]`_،_محكمة_إلى_Low_/_Medium_/_High_/
Critical._مدخلاتها_فقط_بيانات_معروفة_في_وقت_التقييم_(لا_تسريب_مستقبلي):

- **السابق_السلسلة** —_وزن_أساسي_لكل_فئة_مؤشر_(قرار_سعر_ال-interest_يتجاوز_CPI_،
  الذي_يتجاوز_استطلاع_ثانوي).
- **بصمة_التقلب_المحقق** —_متوسط_العائد_المطلق_لرموز_الرمز_الأساسية_في_النافذة_
  بعد_إصدارات_هذه_السلسلة_السابقة:_"هذا_الإصدار_تاريخياً_يحرك_السعر_بهذه_الكمية"_
- **حساسية_المفاجأة** —_مدى_قوة_ارتباط_المفاجأة_المطلقة_(z-score)_تاريخياً_مع_
  حركة_ما_بعد_الإصدار.

النتيجة_تمزج_هذه_بأوزان_ثابتة_وتختم_`ImpactModelVersion`._إعادة_الحساب_هي_
عملية_صريحة_ومسجلة_تنتج **مراجعة_جديدة**_—_وليس_طفرة_—_لذا_النتيجة_دائماً_قابلة_
للإعادة_الإنتاج_من_مدخلاتها.

## تخطيط_البلد_→_العملة_→_الرمز

أكثر_مشكلات_التكامل_الخوارزمية_المذكورة_هي_تُحل_مرة_واحدة_،
كـ_دالة_نقية:_بلد_ mapsإلى_عملته_(كل_دولة_منطقة_اليورو_تنتشر_إلى_EUR)_،
وعملة_ mapsإلى_رموز_المراقبة_التي_تسعرها_على_أي_ساق._إذاً **EURUSD_متأثر_
بأحداث_الاتحاد_الأوروبي_والولايات_المتحدة**_؛_XAUUSD_عرض_لـ_USD_؛_US500_ mapsإلى_USD_.
هذا_يدفع_مرشح_الأخبار،_حل_الرموز_المتأثرة_وحسابات_المنع.

## سياسة_نافذة_الأخبار

`NewsWindowRule` هو_`{_minImpact,_beforeMinutes,_afterMinutes,_currencies?,_series?_}`._
تنفيذ_واحد_مشترك_ونقي_يجيب_"هل_اللحظة_T_داخل_منع_للرمز_S؟"_—_يستخدم_بواسطة_
مرشح_أخبار_cBot_،_إيقاف_نسخ_التداول_وصريحة_خطر_AI_،_لذا_لا_يمكنهم_أبداً_التباعد._
عند_عدم_اليقين_الجواب_المنع_يغيب_إلى_القيمة_المحافظ_المُكوَّنة_(فشل_مغلق_
افتراضياً)_حتى_لا_يُمكن_لفجوة_البيانات_إطلاق_التداول_صامتاً_عبر_إصدار_مرتفع_
التأثير.

## نقطة_في_الوقت_و_المراجعات

الفعلي_والتوقعات_ودرجات_التأثير_هي **إلزامية**._كل_حدث_يمتلك_سلسلة_مرتبة_
من_المراجعات_،_رتيبة_في_`KnownAt`:

- `Scheduled` —_الحدث_تمت_جدولته_أولاً_(تأثير_سابق،_لا_فعلي).
- `Released` —_الفعلي_الأول_وصل.
- `Revised` —_قيمة_معدلة_لاحقة_وصلت.
- `Rescheduled` —_المصدر_حرك_لحظة_الإصدار_(قابلة_للتدقيق،_قابلة_للتنبيه).
- `Rescored` —_درجة_التأثير_أُعيد_حسابها_ضمن_إصدار_نموذج_جديد.

الاستعلام_`as of`_لحظة_ماضية_يعيد_بالضبط_المراجعة_المعروفة_ثم_—_الضمان_الذي_
يقتل_النظرية_المسبقة_في_قواعد_الأخبار_المُختبر_بالbacktest_.

## التوقعات_/_الإجماع

المصدر_الرسمي.

## تحديد_المعدّل_و_الخطة_الاحتياطية

تنشر_المصادر_الخارجية_حدود_المعدّل_(FRED_يسمح_بحوالي_120_طلباً/دقيقة)._
التقويم_مُبنى_لأنه **لا_يتجاوز_أبداً_حد_المزود**_،_ولأن_التعرض_للتقييد_أو_القطع_
لا_يتدهور_القراءات:

- **تحديد_استباقي.**_كل_عميل_HTTP_للمصدر_يمر_عبر_بوابة_معدّل_مشتركة_آمنة_لـ_الخيوط_
  توزع_الطلبات_الخارجية_إلى_ميزانية_مُكوَّنة_(`App:Calendar:FredRequestsPerMinute`،
  الافتراضي_100_—_بشكل_مقصود_أقل_من_سقف_المزود)._الطلبات_تُوضع_في_قائمة_انتظار_
  وتُوتَّر،_ولن_تُطلق_دفعات.
- **احترام_`429_Retry-After`.**_إذا_أعاد_مزود_أبداً_`429_Too_Many_Requests`،
  تسحب_البوابة_كل_المصدر_للخلف_بالوقت_التبريد_الذي_يطلبه_الخادم_(أو_
  `App:Calendar:RateLimitBackoff`،_الافتراضي_60_ثانية)_قبل_المكالمة_التالية_
  —_لا_حلقة_إعادة_محاولة_ضيقة.
- **مرونة_قياسية.**_كل_عميل_مصدر_أيضاً_يرث_عامل_المرونة_الخاص_بالتطبيق_
  (إعادة_محاولة_مع_تراجع_+_ jitter،_قاطع_الدائرة،_المهلات)_،_لذا_الومضات_العابرة_
 تُمتص_ومصدر_فاشل_بشكل_مستمر_يُوقف_(تغطيته_تصبح_قديمة)_دون_التأثير_على_الآخرين.
- **الخطة_الاحتياطية_—_ذاكرة_القراءة_المستمرية.**_القراءات_ **لا_تُخدم**_باستدعاء_
  مزود._بمجرد_جلب_نطاق_يُ persisted_إلزامياً_إلى_Postgres_ويُخدم_من_هناك_إلى_الأبد_
  (انظر_§_"تحميل_عند_الطلب")_._لذا_حتى_عند_تقييد_مزود_أو_توقفه،
  يستمر_التقويم_في_الإجابة_من_بيانات_مُخبأة_صحيحة_نقطة_الوقت_؛_النطاق_المفقود_
  ببساطة_يبقى_غير_مغطى_ويُعاد_محاولته_في_دورة_الإدخال_التالية._إجابات_المنع_
  additionally_fail_to_the_conservative_default_under_uncertainty_،
  so_a_data_gap_never_green-lights_trading_through_a_release.
- **استطلاع_رخيص.**_الجلب_المشروط_(ETag_/_If-Modified-Since_/_مؤشرات_التجديد_
  للمصدر)_و_"جلب_نطاق_مرة_واحدة،_أبداً_مرة_أخرى"_memoria_تحافظ_على_حجم_
  الطلب_الفعلي_بعيداً_عن_أي_حد_في_التشغيل_العادي_—_بوابة_المعدّل_هي_شبكة_أمان،
  وليست_المسار_الشائع.

## تفعيل_/_تعطيل

مستويان_مستقلان_،_تماماً_مثل_ميزات_cMind_الأخرى:

  من_واجهة_إدارة_الميزات؛_لا_إعادة_نشر،_يأخذ_تأثيره_مباشر.

  الافتراضي_`true`)._معادو_التسويق_يحددونه_`false`_لإزالة_الميزة_كلياً؛_
  المشغل_لا_يستطيع_إعادة_تفعيلها_بعدها.

الحالة_الفعالة_هي_`Branding.EnableEconomicCalendar_&&_FeatureToggle.EconomicCalendar`._
عند_التعطيل،_يدخل_التنقل_مخفي_و_`/economic-calendar`،
  `/api/calendar/**`_
  وأدوات_التقويم_MCP_تُعيد_`404` _feature-disabled_نظيف_—_وليس_`500`._التاريخ_
 المستمر_محفوظ_عند_تعطيل_وقت_التشغيل_لإعادة_التفعيل_فوري.
