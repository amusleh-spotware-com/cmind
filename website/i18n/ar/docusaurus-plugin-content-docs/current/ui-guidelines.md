# إرشادات_تصميم_الواجهة_—_MANDATORY

ملزم_لكل_قطعة_واجهة_جديدة_أو_مغيرة_في_هذا_التطبيق_(صفحات_Blazor،
الحوارات،_المكونات)._هذا_مصدر_الحقيقة_المشار_إليه_بواسطة_`CLAUDE.md`._
إذا_حظرتك_قاعدة،_توقف_واسأل_—_لا_تشحن_واجهة_تخالفه._جذر_في_
`plans/ui-overhaul.md`.

## 1. المحمول_أولاً_دائماً

- **صمم_للهاتف_360–430px_أولاً**،_ثم_حسّن_للأعلى_بـ_`min-width`_media_queries_/
  MudBlazor_ breakpoint_props._لا_تبدأ_بالكمبيوتر_المكتبي_مع_`max-width`_
  _overrides.
- **لا_انتقال_أفقي_في_أي_عرض_320–1920px.**_إذا_كان_المحتوى_أعرض_من_
  المنفذ_،_إنه_خطأ.
- أهداف_اللمس_≥ **44px** (`var(--app-touch-target)`)._مدخلات_النص_≥_16px_
  خط_(يوقف_تكبير_iOS-on-focus).
- احترام_النتوءات:_استخدم_`env(safe-area-inset-*)`؛_المنفذ_يحتوي_ bereits_`viewport-fit=cover`.
- احترام_`prefers-reduced-motion` —_لا_معلومات_أساسية_تُنقل_فقط_بالرسوم_المتحركة.

## 2. رموز_التصميم_—_لا_قيم_محددة_بثابت

- كل_اللون/نصف_القطر/التباعد_ياتي_من **رموز_التصميم**:_سمة_MudBlazor_
  (`Web/Components/Theme.cs`)_+
  الخصائص_المخصصة_CSS_المنبعثة_من_`Web/Branding/BrandingCss.cs`_
  (`var(--app-primary)`،
  `--app-surface`،_`--app-border`،_`--app-text*`،_`--app-radius`،_…).
- **لا_تحدد_لوناً_سداسياً_أو_نصف_قطر_أو_سلسلة_علامة_في_مكون_أو_قاعدة_CSS_أبداً.**
  اقرأ_رمزاً._الرموز_تتدفق_من_`BrandingOptions` _للعلامة_البيضاء_،
  لير_パレット_العلامة_يصل_إلى_واجهتك_مجاناً.
- قيمة_جديدة_تأثر_بالعلامة_→_أضف_رمزاً_+_حقل_علامات؛_لا_تضمّنه_مباشرة.

## 3.تخطيط_متجاوب_و_بيانات

- **الجداول_تنهار_إلى_بطاقات_على_الهواتف.**_كل_`MudTable` _تضبط_`Breakpoint="Breakpoint.Sm"`_
  وكل_`MudTd` _لها_`DataLabel`._لا_جدول_واسع_خام_على_المحمول._(قالب:_
  `Components/Pages/Nodes.razor`.)
- الشبكات:_`MudItem xs="12" sm="6" md="4"` —_عرض_كامل_على_الهاتف،
  أعمدة_متعددة_للأعلى.
- النماذج_عمود_واحد_على_المحمول؛_أهداف_اللمس_الكبيرة؛_`inputmode`/`autocomplete`
  على_المدخلات؛_`inputmode`رقمي/عشري_للمال/النسبة_المئوية.
-وفر_ **تحميل_وحالة_فارغة_وحالة_خطأ** _على_كل_قائمة/تفاصيل_—_مُقاسة_
  للمحمول.
- **التنقل_السفلي_للهاتف** (`Components/Layout/BottomNav.razor`)_هو_التنقل_
  الأساسي_للهاتف؛_الدرج_المُجمَّع_هو_القائمة_الكاملة._أضف_الوجهات_عالية_
  المرور_هناك؛_أبقها_≤_5_عناصر.

## 4. الحوارات_(إنشاء/تحرير)

- كل_إجراء_إضافة/إنشاء/تحرير/جديد_يستخدم_ **حوار_MudBlazor**
  (`IDialogService.ShowAsync<TDialog>`)_،_وليس_نموذج_صفحة_مضمن._الحوارات_
  تعيش_في_`Web/Components/Dialogs/`_،_تعرض_`[Parameter]`s_،
  تُرجع_`public_sealed_record_…Result(...)` _مُتداخلة._إجراءات_الصف_(
  start/stop/delete)_تبقى_مضمنة_كأزرار_أيقونات.
- على_الهواتف_،_الحوارات_يجب_أن_تكون **شاشة_كاملة/عرض_كامل** _و_مدركة_لوحة_
  المفاتيح.

## 5. المساعدة_المضمنة_—_كل_ضابط

- كل_خيار_غير_بديهي_أو_تحديد_أو_تبديل_أو_إجراء_يحصل_على_ **`<HelpTip Text="…" />`**

  حدّث_كلاهما_في_نفس_الالتزام.
