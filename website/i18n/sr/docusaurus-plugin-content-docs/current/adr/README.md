# Записи архитектурских одлука

Ови записи бележе дизајнерске одлуке које **не можеш закључити са кода** — компромисе, путање не
предузиме, и зашто. Свака је кратка: *Контекст → Одлука → Последице*. Нова структурна одлука → додај ADR
овде (следећи број) тако следећи инжињер (човек или AI) наслеђује размишљање, не само
резултат.
