# 機能 — 完全ツアー 🧭

グランドツアー へようこそ。cMind は *大量* 詰め込む 1 個アプリ、ここで マップです。各機能 自分 深いダイブ ドキュメント — あなたが 掻きたい どんな かゆみ でも クリックスルー。

## 🔁 コピートレーディング

王冠宝石。マスター アカウント ミラー 多くの上に、保つ シンク even インターネット もしら時。

- **[コピートレーディング](./copy-trading.md)** — コア: ミラーリング、オーダータイプ、SL/TP、スリッページ、デスync/再同期。
- **[実行透明性](./copy-execution-transparency.md)** — 正確 見る コピーされた、時に、なぜ。
- **[パフォーマンスフィー](./copy-performance-fees.md)** — チャージ あなたのシグナル、ハイウォーターマークスタイル。
- **[プロバイダマーケットプレイス](./copy-provider-marketplace.md)** — トレーダー 発見 してフォロー プロバイダ。
- **[通知](./copy-notifications.md)** — 取得 告げた あなたが必要なとき。
- **[AI コピー推奨](./ai-copy-recommender.md)** — AI 提案 誰 フォロー。
- **[Open API トークン ライフサイクル](./token-lifecycle.md)** — cMind 保つ どのように 正確に 1 個有効 トークン cID ごと。

## 📊 あなたのホームベース

- **[ダッシュボード](./dashboard.md)** — ライブ、モバイルファースト コマンドセンター: KPI スパークライン、アクティビティチャート、ステータスリング、ライブフィード、(管理者向け) クラスタヘルス。リフレッシュ 自分。

## 🧠 AI コア

チャットボックス ボルト オン サイド ではなく — AI その実際 *作業 やる*。

- **[AI アシスタント、エージェント、リスクガード & アラート](./ai.md)** — 戦略生成、自己修復ビルド、バックグラウンド リスクガード 自動停止 ボット、スマートアラート。

## 🛠️ ビルド & 実行

- **[ビルド & バックテスト cBots](./build-and-backtest.md)** — ブラウザ内 Monaco IDE、C#/Python テンプレート、サンドボックス ビルド、ライブ エクイティ曲線。
- **[MCP サーバ](./mcp.md)** — 公開 cMind のツール HTTP + SSE 経由 AI クライアント ドライブ可能。

## 🏢 ビジネスとして実行

- **[ホワイトラベル / ブランディング](./white-label.md)** — リブランド すべてのサーフェス 設定経由。
- **[プロップファーム チャレンジ シミュレーション](./prop-firm.md)** — 実行 毎日損失、ドローダウン、ターゲット ルール ライブエクイティで。
- **[機能トグル](./feature-toggles.md)** — 決定 どの各デプロイメント/テナント 見る。
- **[コンプライアンス / 法務](./compliance.md)** — 保持 監査証跡 あなたの規制当局 要求。

## 📱 エクスペリエンス

- **[インストール可能アプリ (PWA)](./pwa.md)** — モバイルファースト、オフラインシェル、ホーム画面追加。
- **[UI デザインシステム & モバイルファースト](../ui-guidelines.md)** — デザイントークン + ルール 背後 外観。
