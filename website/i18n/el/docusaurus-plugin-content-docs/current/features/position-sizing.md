# Μέγεθος θέσης & χαρτοφυλάκιο

«Πόσο μεγάλη πρέπει να είναι αυτή η συναλλαγή;» είναι η ερώτηση που καθορίζει αν ένα πλεονέκτημα σύνθεση ή ανατινάσσεται.
Τα θεσμικά απαντούν με **στόχευση μεταβλητότητας** και το **κριτήριο Kelly**, και χτίζουν ένα βιβλίο με
**risk parity** αντί για ίσα δολάρια. Το cMind φέρνει και τα δύο στο retail — ντετερμινιστικά μαθηματικά σε μια σειρά αποδόσεων μιας στρατηγικής, με μια σύσταση σε απλά αγγλικά.

Ανοίξτε **cBots → Position Sizing** (`/quant/sizing`).

## Μέγεθος μεμονωμένης στρατηγικής

Δεδομένων των αποδόσεων μιας στρατηγικής (ή της καμπύλης ιδίων κεφαλαίων), μιας στοχευόμενης ετήσιας μεταβλητότητας, ενός κλάσματος Kelly και ενός ορίου μόχλευσης, το μέγεθος αναφέρει:

- **Πραγματική ετήσια μεταβλητότητα** — η δική της μεταβλητότητα της στρατηγικής, ετήσια με τον κανόνα της τετραγωνικής ρίζας του χρόνου.
- **Μέγεθος με στόχευση μεταβλητότητας** — η έκθεση που κάνει την πραγματική μεταβλητότητα να ανταποκρίνεται στον στόχο σας (`στόχος ÷ πραγματική μεταβλητότητα`), με ανώτατο όριο το όριο μόχλευσής σας. Οι χαμηλότερης μεταβλητότητας στρατηγικές λαμβάνουν μεγαλύτερο μέγεθος.
- **Full Kelly** — το κλάσμα ανάπτυξης-βέλτιστο `f* = μ / σ²` (μέσο προς διακύμανση των αποδόσεων).
- **Fractional Kelly** — `f*` κλιμακωμένο με το κλάσμα Kelly. Half-Kelly (0.5) είναι η κοινή ασφαλής επιλογή;
  το full Kelly είναι διαβόητα πολύ επιθετικό για πραγματικές, αβέβαιες ανταγωνιστικές θέσεις.
- **Προτεινόμενη έκθεση** — το **μικρότερο** (ασφαλέστερο) μεταξύ του μεγέθους με στόχευση μεταβλητότητας και του fractional Kelly, με ανώτατο όριο. Μια στρατηγική χωρίς θετικό πλεονέκτημα (full Kelly ≤ 0) έχει μέγεθος **μηδέν**.

```http
POST /api/quant/sizing
{ "returns": [...], "targetVolatility": 0.10, "kellyFraction": 0.5, "leverageCap": 3 }
```

## Κατανομή χαρτοφυλακίου

Δώστε δύο ή περισσότερες στρατηγικές (ευθυγραμμισμένες χρονικές σειρές αποδόσεων) και χτίζει ένα βιβλίο με **αντίστροφη-μεταβλητότητα risk parity** — κάθε στρατηγική ζυγίζεται με `1 / μεταβλητότητα`, κανονικοποιημένα — ώστε ο κίνδυνος, όχι τα δολάρια, να μοιράζεται ομοιόμορφα. Επιστρέφει επίσης:

- τον **πίνακα συσχέτισης** μεταξύ των στρατηγικών σας (εντόπισε αυτές που είναι κρυφά το ίδιο στοίχημα);
- την **αναμενόμενη μεταβλητότητα χαρτοφυλακίου** σε αυτά τα βάρη, από τη δειγματική συνδιακύμανση;
- έναν παράγοντα **μόχλευσης** που κλιμακώνει όλο το βιβλίο προς τη στοχευόμενη μεταβλητότητά σας (με ανώτατο όριο).

```http
POST /api/quant/portfolio
{ "strategies": [[...], [...]], "targetVolatility": 0.10, "leverageCap": 3 }
```

## Γιατί είναι αξιόπιστο

Ολα αυτά είναι καθαρός, ντετερμινιστικός domain code (`Core.Portfolio`) χωρίς εξάρτηση υποδομής και χωρίς εξωτερικές κλήσεις — unit-tested για την κλιμάκωση στόχευσης μεταβλητότητας, τον τύπο Kelly, την ιδιότητα ίσου κινδύνου των αντίστροφης-μεταβλητότητας βαρών και τον πίνακα συσχέτισης. Συμβουλευτικός εξ ορισμού: οι αριθμοί είναι μια σύσταση, ποτέ αυτόματη εντολή.
