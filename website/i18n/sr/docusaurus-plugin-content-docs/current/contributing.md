# Допринос cMind-у 🛠️

Хвала што си тутра. cMind постаје боља сваки пут када неко отвори проблем, пријави прецизно
cTrader понашање, исправи брзи у овим док-ама, или пошаље PR. **Не требаш бити .NET
чаробњак** — тестери, трговци, и док-исправљачи су толико вредни као особе пишу агрегате.

:::tip[Канонски водич живи у складишту]
Ова страница је пријатељска на-рампа. Потпуни, увек-текућа процес — основна правила, кодирање
конвенције, преглед ток — је у **[CONTRIBUTING.md](https://github.com/amusleh-spotware-com/cmind/blob/main/CONTRIBUTING.md)**.
:::

## Твој прави допринос у ~10 минута

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
dotnet restore
dotnet build          # 0 упозорења, или CI ће вас љубазно одбити
dotnet test           # јединица + интеграција + E2E
```

Нашао нешто да исправиш? Гранајте га, промените га, додајте тест, и отворите PR. То је цела петља.

## Кућна правила (кратка верзија)

cMind креће **прави новац**, тако да неки ствари нису преговарачи — и искрено, чине кодну базу
радост за рад:

- **Строга Domain-Driven Design.** Пословна логика живи на агрегатима и value објектима, никад не
  крајњим тачкама или UI. (Постоји пријатељски водич за то у складишту.)
- **Три теста слоја, сваких промена.** Јединица + интеграција + E2E, *укључујући* пропутане путање (пропали
  конекције, одбијени налози, мртви чворови). Зелени тестови су цена допуста.
- **Нула упозорења.** `TreatWarningsAsErrors=true`. Модерне C# 14 идиоме.
- **Без тајни, без магичних стрингова, никад `DateTime.UtcNow`** (инјектирај `TimeProvider` уместо).
- **Документи у истој комисији.** Промени понашање → ажурира своју документ. Да, то укључује ове веб страницу.

Пунина детаља, са *зашто* иза сваког правила, у
[CONTRIBUTING.md](https://github.com/amusleh-spotware-com/cmind/blob/main/CONTRIBUTING.md) и
[AGENTS.md](https://github.com/amusleh-spotware-com/cmind/blob/main/AGENTS.md).

## Допринос са AI 🤖

Искрено добродошле **AI-помогнута PR-е** — овај пројекат је саграђен да буде радио на агентима као и
људи. Ако водиш Claude, Copilot, или слично: окрени га на
[AGENTS.md](https://github.com/amusleh-spotware-com/cmind/blob/main/AGENTS.md), дозволи му прочитај угнездене
`CLAUDE.md` датотеке, и одржи га исте чине (тестови, нула упозорења, DDD). Добра AI PR је
неразликована од добра људска PR — исти преглед, исти добродошла.

## Буди одличан једно другом

Имамо [Кодекс понашања](https://github.com/amusleh-spotware-com/cmind/blob/main/CODE_OF_CONDUCT.md).
Суштина: буди добра, претпостави добру веру, и упамти да је особа (или особина агента) на
другом крају. Питај рано питања — то је снага, не досада.

Добро дошли на борду. Не можемо чекати да видимо шта грађиш. 🎉
