# Ζώνη ώρας

Κάθε ώρα στην εφαρμογή εμφανίζεται στη δική σας ζώνη ώρας, όχι του διακομιστή. Η επιλογή σας αποθηκεύεται στο προφίλ σας και σας ακολουθεί σε όλες τις συσκευές.

Στην πρώτη επίσκεψη η εφαρμογή υιοθετεί αυτόματα τη ζώνη του προγράμματος περιήγησής σας. Μπορείτε να την αλλάξετε ανά πάσα στιγμή από Ρυθμίσεις → Ζώνη ώρας· η προεπιλογή ανάπτυξης είναι η επιλογή white-label App:Branding:DefaultTimeZone (προεπιλογή UTC). Οι ώρες αποθηκεύονται και επιστρέφονται πάντα σε UTC — μόνο η προβολή μετατρέπεται.

- Σειρά επίλυσης: ζώνη προφίλ, μετά cookie, μετά προεπιλογή ανάπτυξης, μετά UTC.
- Ο εντοπισμός εκτελείται μία φορά και δεν παρακάμπτει ποτέ ζώνη που επιλέξατε.
- Η μορφοποίηση ακολουθεί τη γλώσσα σας· σχετικές ετικέτες όπως «πριν από 2 λεπτά» δεν επηρεάζονται.
