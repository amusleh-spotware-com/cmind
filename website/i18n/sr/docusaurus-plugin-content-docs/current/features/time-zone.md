# Временска зона

Свако време које апликација приказује исписује се у вашој временској зони, не серверовој. Ваш избор се чува у профилу и прати вас на свим уређајима.

При првој посети апликација аутоматски преузима зону вашег прегледача. Можете је променити било кад у Подешавања → Временска зона; подразумевана за примену је white-label опција App:Branding:DefaultTimeZone (подразумевано UTC). Времена се увек чувају и враћају преко API у UTC — конвертује се само приказ.

- Редослед разрешавања: зона профила, затим колачић, затим подразумевана примене, затим UTC.
- Откривање се извршава једном и никада не мења зону коју сте изабрали.
- Форматирање прати ваш језик; релативне ознаке попут „пре 2 минута“ нису погођене.
