# Копирање трговања

Огледало **мастер** cTrader налог на један+ **роб** налога — крос-берза, крос-cID — са по-одредишта контролом + паре-разред помирба.

## Концепти

- **Профил копирања** — један мастер (`SourceAccountId`) + један+ **одредишта**. Животни циклус: `Draft → Running → Paused → Stopped` (`Error` при неуспеху). Агрегат корен: `CopyProfile` (власнички `CopyDestination`).
- **Одредишта** — један роб налог + пун скуп правила за како мастер копиран на то. Сва конф по-одредишта, тако један мастер храни конзервативна + агресивна роба одједном.
- **Копирање мотор домаћин** — радни радник за профил (`CopyEngineHost`). Нати мастер извршавања ток, применити сваког события на свако одредишта.
- **Надзорник** — `CopyEngineSupervisor`, позадина служба на сваки чвор. Домаћин додељени профили, само-оздрављивање преко кластер (видети [скалирање](../deployment/scaling.md)).

## Шта се огледа

| Мастер ferdinand | Роб акција |
|--------------|--------------|
| Тржиште / тржиште-опсег позицију отворена | Отворити прилагођена копија (означена са извор позицију ID) |
| Граница / стоп / стоп-граница позиција налога | Поместити подударни позиција налога |
| Позиција налога амендирати | Амендирати огледало позиција налога на месту |
| Позиција налога отказ / истека | Отказ огледало позиција налога |
| Делимична затвари | Затвори исту пропорцију роб позицију |
| Скала-у (томе повећање) | Отворити додано томе (опционално) |
| Стоп-лос / завршног-стоп промена | Амендирати роб позицију заштиту |
| Пуна затвари | Затвори роб копија |

Свака копија **означена са извор позицију/налога ID**. После препознај домаћин обнови стање од помирба: отворава копие мастер држи али роб недостаје, затвара роб „сирочета" мастер дакле више не садржи — **без дупцирања трговине**.

## Создавање профила

**Нови профил** дијалог на Копирање трговања страна пролази сви унапред: профил име, извор (мастер) налог, одредишта (роб) налога (мулти-одабир са **Одабери све** дугме; одабран мастер искључена од роб листа), + потпуна по-одредишта опција скуп доле. Сви учинци **валидирано пред чувањем** — недостаје име/извор/одредишта, не-позитивна величина параметар, негативна/неконзистентна много граница, ван-опсега преклон %, нема налога врста омогућена, празна симбол филтер, или деформисана симбол-карта парови површина као листа грешка + блок чување. На потврди, профил направљен + сваки одабран роб додан са одабран подешавање.

Редови акције поштују животни циклус: **Почни** омогућена само када не трчи, **Стоп** + **Паузирај** само када трчи, **Избаци** деактивирана док трчи + пита потврду пред уклањањем профила + одредишта.

## По-одредишта опције

Поставити у Нови профил дијалог, на Копирање трговања страна по-одредишта панела, или преко `POST /api/copy/profiles/{id}/destinations`:

- **Величина** (`MoneyManagementMode` + параметар): фиксна много, много/имена множитељ, пропорцијална биланс/капитал/слободно-маржа, фиксна риск %, фиксна полугања, аутоматско-пропорцијална, **риск-%-од-стоп** (M7). Плус мин/макс много граница + принуди-мин-много. **Риск-од-стоп** величина одредишта тако то ризици конфигурирано проценат од *своја сопствена* биланс, дериван од **мастер лос-лос растојање** (`мастер ризици 2% → роб аутоматско-ризици 2%`): `много = биланс×% ÷ (stopDistance × contractSize)`. Мастер отворена **без** лос-лос има нема растојање да величина против → користи конфигурирано **макс-риск повратни много** (M7) ако поставити, инач пропусти (`no_stop_loss`) не погађа. Пропорцијална-**капитал**/**слободно-маржа** величина из правог налога **капитал** (`биланс + Σ плутајући P&L`, дериван по cTrader Open API која не доставити капитал), не обичан биланс — тако мастер седећи на отворена профит/лос величине копие право. Користени маржа не излож од помирба API, тако слободно-маржа сматран као капитал (поштена доступно-средства прокси); други режими читају биланс + пропусти додатна преречуну круг-тешко.
- **Смер филтер**: обе / дугачак-само / кратак-само. **Обрни**: флип страна (+ замени SL↔TP) за противан копија.
- **Управљање-само** (Игнори-Нови-Трговање/Затвори-Само): огледало затвара, делимична затвара + заштита промене на већ-копираних позиција, али отворена **не** нову позиција/позиција налога (пропусти `manage_only`). Користити да намотај одредишта доле без сечење постојећи копие.
- **Синц-Отворен-на-почетак** / **Синц-Затворен-на-почетак** (подразумевано на): на профил **прво** исправљање, да ли открити копие мастер је-постојеће позиција, + да ли затвори копие мастер затворена док профил стао. Обе примени само на почетак — мид-трчи препознај је увек помирба потпуно тако исправљање опоравак независно.
- **Симбол карта** + **симбол филтер** (забележена / блеклиста). Свака симбол-карта унос носи опционално **по-симбол том множитељ** (cMAM по-симбол преписивање) скалирање копија величина за то симбол на врху одредишта величина (1 = не промена). Цела карта увоз/извоз као **CSV** (`GET …/symbol-map.csv`, `PUT …/symbol-map/csv`; колоне `Source,Destination,VolumeMultiplier`) — сваки редак валидиран кроз домена вредност објекти, тако деформисана датотека не може произвести неважећа карта.
- **Трговачки-часови прозор** (C18) — по-одредишта дневни UTC прозор (`start`/`end` минута-дана, крај јесте; `start == end` = читав дан). Нова отворена ван прозора пропусти (`trading_hours`); прозор са `start > end` омота минуло поноћ (нпр. 22:00–06:00). Постојеће позиција остају управљаних.
- **Извор-етикета филтер** (C18, cTrader еквивалент од MT магична-број филтер) — када поставити, копирај само мастер трговања чија етикета одговара **управо** (нпр. један бот трговања, или ручно-само етикета); инач пропусти (`source_label`). Празно = копирај све. Носио на `ExecutionEvent.SourceLabel` од мастер позицију/налога `TradeData.Label`, хоноран на исправљање а.
- **Налог заштита** (ZuluGuard / Глобални налог заштита) — гледање одредишта **живо капитал** (`биланс + Σ плутајући P&L`, анкета сваки `CopyDefaults.EquityGuardInterval`) против `StopEquity` под и/или опционално `TakeEquity` плафон. Нарушај, применити режим: **CloseOnly** (стоп нову копие, чувати управљање постојеће), **Замрзнуто** (стоп отварање), **Продај-Изван** (затвори **сваки** копија на одредишта одмах). Једном пао, одредишта закупљен — не нову отворена док домаћин рестарт — + `CopyAccountProtectionTriggered` упозорење подигнуто. `SellOut` захтевају `StopEquity`; `TakeEquity` мора седети горе `StopEquity`. **Не-гарантија узорак:** продај-изван користи тржиште извршавања — као сваки конкурент еквивалент, не могу гарантира испуни цена у брз/развучена тржишта.
- **Равна-Све паника дугме** (C8) — `POST /api/copy/profiles/{id}/flatten` одмах затвори **сваки** копирана позицију на свако одредишта + браве против нова отворена. Маршрута крос-процес: API скупови заставе, надзорник доставити покренута домаћин (поново користи токен-ротације канала), који равна на месту; заставе јасна тако гори управо једном (`CopyFlattenAll` упозорење). Корисник тада паузирати/стоп профил.
- **Проп-фирме правило стража** (C7) — спровођење проп-фирме копирач корисници питају за. По одредишта, **дневни-лос капа** (лос од дна отварање капитал) и/или **завршног-преклон** граница (лос од покренута врхо капитал), обе у налог валута. Нарушај одредишта **аутоматско-равна** (свака копија затворена) + **закључана** остатак UTC дана (нова отворена пропусти `prop_lockout`); `CopyPropRuleBreached` упозорење гори. Закупљен јасна када UTC дана рол преко (свежа базна линија/врхо узети). Акције исти живо-капитал анкета као налог заштита.
- **Извршавања јектор** (C11, изван подразумевано) — насумично `0..N` ms задршка пред стављањем сваки копија, да де-корелирати близо-идентична налога времена путем корисника **сопствено** налога. **Усклађеност узорак:** помоћ за проп фирме то *дозволи* копирање — **не** алат да избегне фирме то забрани то; остајући унутар вашег фирме правила је ваша одговорност.
- **Конф лок** (C9) — замрзнути одредишта подешавање за период (`POST …/destinations/{id}/lock` са минута). Док закупљен, одредишта не могу бити уклоњена (агрегат одбија са `CopyDestinationConfigLocked`) — намеран стража против импулсивна промена за време преклон. Лок истиче аутоматски на њена времена.
- **Конзистенција пре-упозорење** (C10) — упозори (једном по UTC дана) када одредишта **дневни профит** достигне конфигурирано проценат дана отварање капитал (`CopyConsistencyThresholdApproaching`), тако проп-фирме конзистенција правило поштовано *пред* то тресе. Профит-страна, независна од лос-страна закупљен; трче изван исти дана базна линија као проп-правило стража.
- **Налога-врста филтер** — одабери управо који мастер налога врстa да копирај: тржиште, тржиште-опсег, граница, стоп, стоп-граница (`CopyOrderTypes` заставе; подразумевано све). cMAM-стил избирљивост.
- **Копирање SL / Копирање TP** — огледало мастер лос-лос / узимање-профит, или управљање заштита независно.
- **Копирање завршног стоп**, **огледало делимична затвари**, **огледало скала-у** — сваки независно укључити/искључити.
- **Копирање позиција истека** (подразумевано на) — огледало мастер позиција налога Добро-До-датума истека времена печата.
- **Копирање мастер проскок** (подразумевано на) — за тржиште-опсег + стоп-граница налога, поместити роб налог са мастер управо проскок-у-тачка (базна цена узета од роб живо место).
- **Охране**: макс преклон %, дневни лос капа, макс копија задршка, проскок филтер (пропусти копија ако роб цена премештена изван N пипс од мастер унос). **Макс копија задршка** измерена против мастер ferdinand реално сервер времена печата (`ExecutionEvent.ServerTimestamp`) преко убачена `TimeProvider`: сигнал старији од конфигурирано макс-заостатак пропусти, тако затајена копија никад поместити касно (раније задршка је увек нула + стража мртав).
- **SL/TP прецизност нормализирање** (M6) — копирана лос-лос/узимање-профит цене заокружена на **одредишта** симбол цифра прецизност пред амендирати, тако мастер цена на финија прецизност (или крос-берза цифра неслагање) никад триповање сервер `INVALID_STOPLOSS_TAKEPROFIT`.
- **Одбијање кругови прекидач / Fol Стража** (G8) — одредишта одбијање `CopyDefaults.RejectionBudget` отворена у редо је **тресе**: не нова отворена за хладина прозор (`CopyDestinationTripped` упозорење гори), стопирање одбијање олуја од чекан (проп-фирме) налог. Постојеће позиција дакле управљаних + затворена док тресе; прекидач аутоматско-рестарт после хладина + успешна копија јасна бројач.
- **Много санитета плафон** (C14) — апсолутна макс копија величина и/или вишеструко-мастер капа. Рачунато копија превишне апсолутна капа, или превишне `N×` мастер сопствено много величина, **тврдо-блокирано** (површина као `lot_sanity` пропусти, antal на `cmind.copy.skipped`) не поместити — браниште против катастрофално-преколички класа (0.23-много мастер окрећућа у 3 много на сваки пријемник преко трча множитељ или заокруживање буг). Обе димензије подразумевано `0` (изван).

## Поузданост & граничне случајеви

Мотор саграђен за реалност то ма-шта могу неуспешна било кад:

- **Роб-позиција пунити-корелација тајм-изван** (C13) — огледало роб позиција чији мастер позиција нестао (ни почиваја ни свежо пуњена) отказана после корелација тајм-изван, тако роб копија не може пуни неповезана у неуправљаног позицију (`CopyPendingTimedOut`). Исправљање такође чистити налога-ID-означена пуњена-позиција сирочета.
- **Робусна затвари/равна** (M8) — затварање сирочета на исправљање, или равна на стража брешњ, толеријно позицију берзи већ затворена (`POSITION_NOT_FOUND`): свака затвара трче независно, тако једна затајена ID никад абортирање исправљање или оставља остатак налога од-равна.

- **Почни са мастер већ у трговина** — на почетак домаћин помирба + отворава копие мастер је-постојеће позиција.
- **Везни пада / исправљање** — на препознај домаћин помирба: отворава недостајући копие, затвара сирочета, обилазак-етикета позиција. Не двоструке налога.
- **Налога стављање неуспех** — неуспех на једна одредишта записана, никад браве други одредишта.
- **Једна важећи токен по cID** — cTrader неважећи cID стари приступ токен моменат нови се издао. cMind замене покренута домаћин токен **на месту** (обнови-аут на живо утичница) тако копирање наставља без пада ток. Видети [токен животни циклус](token-lifecycle.md).

## Проверљивост

Свака акција емитирања структурирано, извор-генерисано дневна ferdinand (`LogMessages`) са профил ID, одредишта cID, налога/позиција ID, + вредности — налог поместити/пропусти (са разлог), делимична затвари, заштита примени, завршног применити, позиција положена/амендирана/отказана, истека огледало, тржиште-опсег проскок огледало, токен замени, исправљање резиме. Ово је записник аудита за усклађеност + спор разрешава.

Поред дневници, мотор емитирања **OpenTelemetry метрике** на `cMind.Copy` јачина (регистрирано у дељена OTel цевовод, извозира преко OTLP / на Azure Monitor као остатак): `cmind.copy.latency` (мастер-ferdinand → опход, ms), `cmind.copy.dispatch.duration` (вентилатор-изван на свако одредишта, ms), `cmind.copy.slippage.points`, `cmind.copy.placed` (означена од одредишта), `cmind.copy.skipped` (означена од разлог), + `cmind.copy.failed`. Ово направи задршка/проскок регресија мерљива, не управо видљива у дневна линија — живо свита твр их против буџет.

## API

- `GET /api/copy/profiles` — листа.
- `POST /api/copy/profiles` — направити (са опционално одредишта налога ID).
- `GET /api/copy/profiles/{id}` — потпуна детаљ incl. сваки одредишта опција.
- `POST /api/copy/profiles/{id}/destinations` — додај одредишта са потпуна опција скуп.
- `DELETE /api/copy/profiles/{id}/destinations/{destinationId}` — уклоњи.
- `POST /api/copy/profiles/{id}/{start|pause|stop}` — животни циклус.

## Тестови

- **Јединица** (`tests/UnitTests/CopyTrading`) — величина режима, одлука филтери, налога-врста филтер, истека копија, тржиште-опсег/стоп-граница проскок, SL/TP укључити/искључити, делимична затвари, позиција амендирани/отказан, почни-са-отворена, одсуствује→исправљање→исправљање, на-месту токен замени, крос-cID неважећи. Трче против `FakeTradingSession`, cTrader-верна у-меморијски симулатор.
- **Интеграција** (`tests/IntegrationTests/CopyLive`) — чвор-афинитета/лизе захтева, токен-верзија пропагација на правој Postgres.
- **E2E** (`tests/E2ETests`) — одредишта-опција круг-тешко кроз API + UI, потпуна животни циклус.
- **Стрес / DST** (`tests/StressTests`) — детерминистичка-симулација тестирање: семена насумично оптерећење + неисправност убризгавање (утичница заклопа, налог одбијање, тржиште-опсег одбијање, токен ротације, чвор смрт) ездраво `CopyEngineHost` на квиетност + твр скупљање инвазивни. Видити [тестирање/стрес-тестирање.md](../testing/stress-testing.md). Ова свита површина + исправљен прави покуска раса: `OnReconnected` жица пред иницијално референца-учитај + исправљање, тако утичница заклопа за време покуска могао трче друга исправљање истовремено + покаљена домаћин је-истовремено стање речници — покуска учитај + прво исправљање сада трче под `_stateGate`.
- **Живо** — правост cTrader демо налога; видити [тестирање/живо-копирање-трговања.md](../testing/live-copy-trading.md).

Видити [разв-верификације.md](../testing/dev-credentials.md) за једна верификације датотека живо + E2E слојеви читају.
