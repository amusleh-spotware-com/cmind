# لوحة_التحكم

أول_شيء_تراه_عند_تسجيل_الدخول،_وبصراحة_الصفحة_التي_ستتركها_مفتوحة_طوال_اليوم.
صفحة_الوصول_(`/`،_`Components/Pages/Index.razor`)_هي **مركز_قيادة_مباشر_ومتقدم_للهاتف**
لنشاط_المستخدم_المسجل_الدخول_عبر_عمليات_cBot_واختبارات_backtest_والموارد_و(للمديرين)_عنقود_
العقدة._تتحدث_ذاتها_،_تبدو_رائعة_على_الهاتف،_ولا_تجبرك_أبداً_على_الضغط_على_F5.

## ماذا_تعرض

من_الأعلى_إلى_الأأسفل،_مرتبة_حسب_الأولوية_للهاتف_(كل_كتلة_هي_عنصر_تخزين_بعرض_كامل_
على_الهاتف،_شبكة_متجاوبة_على_الكمبيوتر_اللوحي/سطح_المكتب):

1. **الرأس** —_العنوان،_مؤشر_مباشر_(نقطة_نابضة_حقيقية؛_ثابت_تحت_
   `prefers-reduced-motion`)_،_وقت_آخر_تحديث،_و **تبديل_الفترة** (`1H · 24H · 7D · 30D`)
   الذي_يشغل_KPIs_والرسم_البياني.
2. **KPIs_البطلة** —_أربع_بطاقات_سريعة_القراءة،_كل_واحدة_رقم_كبير_+
   خط_بياني_SVG_مدمج_،_و(حيث_مجدي)_ **الفرق_مقابل_الفترة_السابقة**:
   - **نشط_الآن** —_التشغيلات_واختبارات_backtest_الحالية_التي_تبدأ/تعمل.
   - **معدل_النجاح** —_المكتمل_÷_(المكتمل_+_الفاشل)_على_الفترة؛_الفرق_بالنقاط_المئوية.
   - **مكتمل** —_التشغيلات/اختبارات_backtest_المكتملة_هذه_الفترة؛_الفرق_مقابل_الفترة_السابقة.
   - **فشل** —_الإخفاقات_هذه_الفترة؛_الفرق_(أقل_أفضل،_لذا_الانخفاض_يظهر_أخضر).
3. **رسم_النشاط_البياني** —_خط_زمني_مساحي_ApexCharts_للبدايات_/_الاكتمالات_/_الإخفاقات_
   لكل_دلو_زمني.
4. **حلقة_حالة_Instance** —_قائمة_دائرية_للتشغيل_/ _backtests_/ _المعلق_/ _المكتمل_/ _الفاشل،_
   الإجمالي_في_المركز.
5. **اختبارات_Backtest** —_لوحة_ثلاثية_(قيد_التشغيل_/ _مكتمل_/ _فاشل)_،_انقر_إلى_`/backtest`.
6. **نسخ_التداول** —_ملفات_نسخ_التداول_مع_نقطة_حالة_مباشرة،_عدد_الوجهات،_و **علامة_Live**
   على_الملفات_القيد_التشغيل؛_انقر_إلى_`/copy-trading`.
7. **وكلاء_AI** —_وكلاء_التداول_المدفوعون_بالشخصية_مع_حالة_التشغيل_(المحرف_·_الحالة)_
   ووقت_آخر_إجراء؛_انقر_إلى_`/agent-studio`.
8. **تدفق_النشاط_المباشر** —_أحدث_20_حدثاً_(الأحدث_أولاً)_مع_نقطة_ملونة_حسب_الحالة_
   ووقت_نسبي.
9. **صحة_العنقود** (المديرون_فقط) —_العقد_النشطة_مقابل_الإجمالية_ومقياس_الاستخدام.
10. **مربعات_الموارد** —_cBots_،_حسابات_التداول،_معرفات_cTrader_،_مفاتيح_MCP_
    (انقر_إلى_صفحاتها).

## تخصيص_لوحة_التحكم_الخاصتك

كل_كتلة_أعلاه_هي ** أداة_تحكمها**._اضغط **تخصيص** (أعلى_يمين_الرأس)_لفتح_
حوار_حيث_ **تظهر/تخفي** أي_أداة_و **إعادة_ترتيبها** بالأسهم_لأعلى/لأسفل. **إعادة_الضبط_
إلى_الافتراضي**_يستعيد_ترتيب_الكتالوج._اختيارك_**يُخزن_في_الخادم_لكل_مستخدم**_،
لذا_يتبعك_عبر_المتصفحات_والأجهزة_—_ليس_هذا_التبويب_فقط.

- الأدوات_المحصورة_بالميزات_والمديرين_فقط_(نسخ_التداول،_وكلاء_AI،_صحة_العنقود)_
  تظهر_فقط_في_الحوار_عندما_ النشر/الدور_يسمح_بها.
- كتالوج_الأدوات_مصدر_حقيقة_واحد_في_`Core/Dashboard/DashboardWidgets.cs`؛_التقديم_
  (التسمية_الأيقونة_التوفر)_يعيش_في_`Components/Dashboard/DashboardWidgetMeta.cs`.

## كيف_تبقى_مباشرة

الصفحة_تستعلم_`GET /api/dashboard/overview?period=<1h|24h|7d|30d>`_كل_10_ثوانٍ_وتعيد_
عرض_الأدوات_في_مكانها_—_لا_إعادة_تحميل_يدوية._خطأ_الاسترجاع_العابر_يُبلع_ويُعاد_
في_العلامة_التالية؛_الحلقة_تتوقف_بشكل_أنيق_عند_التخلص._التحميل_الأول_يظهر_هيكلاً؛_
فشل_مستمر_يعرض_بطاقة_خطأ_مع **إعادة_المحاولة**؛_مستخدم_بدون_بيانات_يرى_KPIs_صفرية_
ونص_حالة_فارغة.

## الواجهة_الخلفية

- `Endpoints/DashboardEndpoints.cs`_تعين_`/overview`_(وتحتفظ_بالنظير_القديم_
  `/stats`)._هي_لكل_مستخدم_ومحصورة_بالميزات_عبر_`ICurrentUser`؛_الساعة_تأتي_من_`TimeProvider`._
  كما_تعين_`GET/PUT /api/dashboard/layout`_—_تخطيط_أدوات_المستخدم،_محمل_عند_بدء_
  الصفحة_ومحفوظ_من_حوار_التخصيص.
- **استمرار_التخطيط**_هو_المجموع_`UserDashboard` (`Core/Dashboard/UserDashboard.cs`):_
  لوحة_واحدة_لكل_مستخدم_(فريدة_على_`UserId`)_،_تمتلك_قائمة_مرتبة_من_إعدادات_الأدوات_
  (مرئية_ترتيب)_مخزنة_كعمود_`jsonb`._القائمة_المرتبة_تتغير_فقط_عبر_`Apply`/_`Reset`_,
  اللذين_يتحققان_من_كل_مفتاح_مقابل_كتالوج_`DashboardWidgets`_ويحافظان_على_اكتمال_
  القائمة_وإزالة_التكرار._المفاتيح_غير_المعروفة_ترفض_بـ_`DomainException` →_`400`.
- `Endpoints/DashboardQuery.cs`_يبني_نموذج_القراءة_`DashboardOverview`:_لقطة_حالة_
 كل_الوقت_(العد_المجموعات)_،_مجموعة_نافذة_من_الحالات_المادية_مرة_واحدة،_وعد_الموارد/العقد._
  حالة_الحالة_وطوابع_الوقت_الطرفية_تعيش_على_أنواع_TPH_الفرعية_(وليس_الأعمدة)_،
  لذا_الصفوف_تُقرأ_في_الذاكرة_عبر_مساعدي_`InstanceEndpoints.GetStartedAt/GetStoppedAt`_المشاركين._
  وقت_الحدث_=_`stopped ?? started ?? created`.
- `Endpoints/DashboardModels.cs`_holds the DTOs،_خطة_الفترة_→_(نافذة،_عد_الدلو)_
  و_`DashboardMath`_
  —_الرياضيات_النقية_الحتمية_لـ_التجميع_و_KPI/الفرق_(لا_I/O،_`now`_يُمرر).
  فروق_KPI_تقارن_النافذة_الحالية_بالنافذة_السابقة_فوراً_(الاستعلام_يجلب_نافذة_مزدوجة_لهذا)._
  لا_توجد_تغذية_حيّة_لحساب_P&L_—_المنصة_لديها_حقوق_فقط_لـ_اختبارات_backtest_
  وتتبع_prop-firm_—_لذا_لوحة_التحكم_مُخصصة_بشكل_متعمد *تشغيلية*_(النشاط،
  الإنتاجية،_معدل_النجاح)_،_وليس_مؤشر_أرصدة_الوسيط.
