# Внесение вклада в cMind 🛠️

Спасибо за то, что вы здесь. cMind становится лучше каждый раз, когда кто-то открывает проблему, сообщает точное поведение cTrader, исправляет опечатку в этих самых документах или отправляет PR. **Вам не нужно быть волшебником .NET** — тестеры, трейдеры и исправители документов так же ценятся, как люди, пишущие агрегаты.

:::tip Каноничный гайд живет в репозитории
Эта страница — дружественный въезд. Полный, всегда актуальный процесс — основные правила, соглашения кодирования, поток проверки — находится в **[CONTRIBUTING.md](https://github.com/amusleh-spotware-com/cmind/blob/main/CONTRIBUTING.md)**.
:::

## Ваш первый вклад за ~10 минут

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
dotnet restore
dotnet build          # 0 предупреждений, или CI вежливо откажет вам
dotnet test           # единица + интеграция + E2E
```

Нашли что-то, что нужно исправить? Ветвитесь, изменяйте, добавляйте тест и откройте PR. Это весь цикл.

## Правила дома (короткая версия)

cMind движет **реальными деньгами**, поэтому некоторые вещи не подлежат обсуждению — и честно, они делают кодовую базу радостью для работы:

- **Строгий Domain-Driven Design.** Бизнес-логика живет на агрегатах и объектах значений, никогда в конечных точках или пользовательском интерфейсе. (Есть дружественный плейбук для этого в репозитории.)
- **Три уровня тестирования, каждое изменение.** Единица + интеграция + E2E, *включая* пути отказа (потерянные соединения, отклоненные заказы, мёртвые узлы). Зелёные тесты — цена входа.
- **Ноль предупреждений.** `TreatWarningsAsErrors=true`. Современные идиомы C# 14.
- **Нет секретов, нет магических строк, никогда `DateTime.UtcNow`** (инъектируйте `TimeProvider`).
- **Документы в один commit.** Измените поведение → обновите его документ. Да, это включает этот сайт.

Полная деталь, с *почему* позади каждого правила, в [CONTRIBUTING.md](https://github.com/amusleh-spotware-com/cmind/blob/main/CONTRIBUTING.md) и [AGENTS.md](https://github.com/amusleh-spotware-com/cmind/blob/main/AGENTS.md).

## Внесение вклада с ИИ 🤖

Мы искренне приветствуем **вспомогательные ИИ PR** — этот проект построен так, чтобы его работали агенты и люди. Если вы управляете Claude, Copilot или подобным: укажите на [AGENTS.md](https://github.com/amusleh-spotware-com/cmind/blob/main/AGENTS.md), позволяйте ему прочитать вложенные файлы `CLAUDE.md`, и придерживайтесь одного уровня (тесты, ноль предупреждений, DDD). Хороший ИИ PR неотличим от хорошего человеческого PR — такая же проверка, такой же приём.

## Будьте превосходны друг к другу

У нас есть [Кодекс поведения](https://github.com/amusleh-spotware-com/cmind/blob/main/CODE_OF_CONDUCT.md). Суть: будьте добры, предполагайте добросовестность и помните, что на другом конце человек (или агент человека). Задавайте вопросы рано — это сила, не беспокойство.

Добро пожаловать на борт. Мы с нетерпением ждём, чтобы увидеть, что вы построите. 🎉
