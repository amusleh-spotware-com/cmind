# Trading Journal & Coach

Новейшая действительно полезная категория ИИ для трейдинга — это не предсказание рынка, а анализ *вашего собственного* поведения. Trading Journal превращает вашу историю запусков и бэктестов в честную обратную связь, чтобы вы могли улучшить стратегию, которая у вас уже есть.

Открыть **AI → Trading Journal** (`/journal`).

## Что показывает

Из ваших инстансов (запусков и бэктестов) вычисляет детерминированно:

- **Количество побед / поражений / неудач и винрейт** по вашим бэктестам;
- **Поведенческие инсайты** — утечки, которые тихо стоят розничным трейдерам:
  - **Over-concentration** — большая часть активности в одном символе;
  - **Repeated failures** — высокая доля запусков, которые не удалось собрать или сконфигурировать;
  - **Losing bias** — больше проигрышных, чем выигрышных бэктестов (с напоминанием запустить Integrity Lab и проверить, реально ли преимущество);
  - чистое здоровье, когда ничего из вышеперечисленного не применяется.

```http
GET /api/journal
```

## Почему это надёжно

Поведенческий анализ — чистый, детерминированный доменный код (`Core.Journal`) без инфраструктурных зависимостей — модульные тесты покрывают over-concentration, повторяющиеся неудачи, смещение к убыткам, сбалансированный случай и пустой аккаунт. Факты идут первыми; ИИ-коуч (Portfolio Digest) — опциональный нарративный слой поверх, подключённый к Anthropic API, поэтому журнал полностью работает без настроенного ИИ.
