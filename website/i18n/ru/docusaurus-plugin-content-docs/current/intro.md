# Добро пожаловать в cMind 👋

:::warning[Альфа-версия — не готова к продакшену]
cMind находится в активной разработке. Ожидайте шероховатостей, критических изменений между версиями и незавершённых функций. **Нам нужны тестировщики из сообщества, репортёры багов и ранние контрибьюторы**, чтобы помочь сформировать его. Если вы столкнулись с проблемой, [сообщите о ней](https://github.com/amusleh-spotware-com/cmind/issues/new?template=bug_report.yml) — ваши отзывы из реального мира сейчас самое ценное, что вы можете внести.
:::

Итак, вы хотите создавать торговых ботов, тестировать их на истории, не расплавив ноутбук, запускать их
на нескольких машинах, зеркалировать сделки на десяток счетов и позволить ИИ следить за риском, пока вы
спите. **Вы попали точно по адресу.**

cMind — это **открытая, самостоятельно размещаемая платформа торговых операций для cTrader**.
Представьте её как весь ваш торговый деск — разработку, исполнение, вычислительный флот, копи-трейдинг и
ядро ИИ — упакованный в спокойное, тёмное, удобное для мобильных приложение, которым вы владеете
полностью.

:::tip[В одном предложении]
Создавайте → тестируйте на истории → запускайте → копируйте свои стратегии cTrader в масштабе, со
встроенным ИИ, на собственных серверах и под собственным брендом.
:::

## Путь за 5 минут ⏱️

Если у вас есть Docker и пять минут, вы можете прямо сейчас поковыряться в реальном экземпляре cMind:

```bash
git clone https://github.com/amusleh-spotware-com/cmind.git
cd cmind
cp .env.example .env        # set OWNER_EMAIL + OWNER_PASSWORD
docker compose up --build
```

Затем откройте **<http://localhost:8080>**, войдите — и вперёд. Полное руководство (с устранением
неполадок на случай, когда у Docker неизбежно появятся своё мнение) находится в разделе
**[Запуск локально](./deployment/local.md)**.

## Пара слов о деньгах 💸

cMind оперирует **реальным капиталом**. Мы относимся к этому серьёзно — каждое изменение поставляется с
модульными, интеграционными и сквозными тестами, включая пути сбоев (обрывы соединений, отклонённые
ордера, мёртвые узлы). Вам тоже стоит отнестись к этому серьёзно: **сначала тестируйте на демо-счёте** и
прочитайте [заметки о комплаенсе](./features/compliance.md), прежде чем направлять его на что-то
реальное. Торговля рискованна; это ПО — инструмент, а не финансовый совет.
