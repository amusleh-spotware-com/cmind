# Функции — полный тур 🧭

Добро пожаловать на грандиозный тур. cMind упаковывает *много* в одно приложение, поэтому вот карта. Каждая возможность имеет свой углублённый документ — нажмите, чтобы перейти к тому, что вас интересует.

## 🔁 Копирование торговли

Жемчужина корны. Отражайте мастер-счет на много, и держите их в синхронизации даже когда интернет плохо себя ведет.

- **[Копирование торговли](./copy-trading.md)** — ядро: отражение, типы заказов, SL/TP, проскальзывание, разсинхронизация/повторная синхронизация.
- **[Прозрачность выполнения](./copy-execution-transparency.md)** — смотрите ровно что было скопировано, когда и почему.
- **[Комиссии за производительность](./copy-performance-fees.md)** — взимайте за ваш сигнал, стиль отметки максимума.
- **[Рынок поставщиков](./copy-provider-marketplace.md)** — позволяйте трейдерам обнаруживать и следить за поставщиками.
- **[Уведомления](./copy-notifications.md)** — получайте сообщения, когда что-то нуждается в вас.
- **[Рекомендатель копирования ИИ](./ai-copy-recommender.md)** — позволяйте ИИ предложить кому следить.
- **[Жизненный цикл маркера Open API](./token-lifecycle.md)** — как cMind держит ровно один действительный маркер на cID.

## 📊 Ваша базовая станция

- **[Приборная панель](./dashboard.md)** — живой центр управления приоритета мобильности: KPI с спарклинами, график активности, кольцо статуса, живой канал и (для администраторов) здоровье кластера. Это обновляет себя.

## 🧠 Ядро ИИ

Не чат-бокс, привинченный сбоку — ИИ, который действительно *делает работу*.

- **[Помощник ИИ, агент, защита рисков и уведомления](./ai.md)** — генерация стратегии, самовосстановление сборок, защита рисков фона, которая может автоматически остановить ботов, и умные уведомления.

## 🛠️ Сборка и запуск

- **[Сборка и бэктестирование cBots](./build-and-backtest.md)** — браузер-интегрированная IDE Monaco, шаблоны C#/Python, изолированные сборки и живые кривые капитала.
- **[MCP-сервер](./mcp.md)** — выставляйте инструменты cMind через HTTP + SSE, чтобы клиенты ИИ могли управлять этим.

- **[Переключатели функций](./feature-toggles.md)** — решайте, что видит каждое развертывание/клиент.
