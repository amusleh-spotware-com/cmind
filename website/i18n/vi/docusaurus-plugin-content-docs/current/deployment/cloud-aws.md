# Triển khai AWS

cMind chạy tốt trên AWS với:

- **ECS Fargate** cho stateless tier (Web + MCP)
- **Aurora PostgreSQL** hoặc **RDS** cho database
- **EC2** cho node fleet (cần Docker)

Xem hướng dẫn chi tiết để biết IAM, logging, monitoring và cost optimization.
