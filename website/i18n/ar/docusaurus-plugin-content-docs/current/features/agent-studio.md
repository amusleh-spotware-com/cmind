# استوديو_الوكلاء

يتيح_استوديو_الوكلاء_لك_إنشاء **وكيل_تداول_بطابع_شخصي** —_بدون_كود_—_وتمنحه_إدارة_
حساباتك_نحو_أهداف_قابلة_للقياس._وكيل_يشبه_cBot_مدفوعاً_بالشخصية:_تختار_محرفاً_
ووضعية_،_تحدد_الحراسات،_ويعمل_تحت **نواة_الاستقلالية_والأمان**.

افتح **AI → Agent Studio** (`/agent-studio`).

## إنشاء_وكيل

يجمع_حوار_**وكيل_جديد**،_بدون_كود:

- **الاسم** و**المحرف** —_Scalper،_Day Trader،_Swing Trader،_Position Trader،_News Trader،
  Contrarian،_Mean Reversion_أو_Breakout/Momentum._كل_إعداد_محدد_يحدد_إيقاعاً_وحالة_معقولة.
- **الوضعية** —_شراسة،_صبور_واتجاه_التتبع_انزلاقات.
- **الحسابات المُدارة** — **يُطلب على الأقل واحد لإنشاء الوكيل** (وكيل بدون حساب لا يمكن أن يبدأ أبدًا، لذلك *إنشاء* يبقى معطلاً حتى تختار واحدًا). إذا لم تقم بربط حساب تداول حتى الآن، يقول الحوار ذلك ويشير إليك لربط واحد أولاً.
- **مستوى_الاستقلالية** — **Advisory** (يقت_بس)_أو **Approval-gated** (يتصرف_فقط_بعد_
  موافقتك_لكل_إجراء). **Full Auto** (بدون_موافقة_لكل_صفقة)_يتطلب_أيضاً **مظروف_المخاطر**
  وقبول_إخلاء_المسؤولية_قبل_أن_يستطيع_التسليح.

الشخصية_تُجمع **بشكل_حتمي** في_موجه_نظام_الوكيل_(لا_يكتبها_LLM)_،_لذا_التكوين_نفسه_
ينتج_دائماً_نفس_التعليمات_—_قابلة_للإعادة_الإنتاج_ومدققة.

## القائمة

يظهر_كل_وكيل_في_جدول_غرفة_التحكم:_الوكيل،_النوع،_عدد_الحسابات_التي_يديرها،
أهدافه،_حالة_التشغيل،_وآخر_إجراء_،_مع_ضوابط **Start / Stop / Kill**._مفتاح_الإيقاف_الطارئ_
يوقف_الوكيل_الفوري_على_الفور.

## الأمان_ثابت_م_domain_وليس_إعداداً

كل_ما_لمس_المال_يمر_عبر **نواة_الاستقلالية_والأمان**:

- **مظروف_المخاطر** —_حدود_صلبة_لكل_أمر_(أقصى_خسارة_يومية،_التعرض_المفتوح،
  حجم_المركز،_الرافعة_،
  الخسائر_المتتالية،_الأوامر/الساعة،_الرموز_المسموحة)._كل_أمر_يُتحقق_منه_ضده_قبل_الإرسال؛_
  المخالفة_تُرفض،_لا_تُعدَّل._مطلوب_قبل_أن_يصل_الوكيل_إلى_Full Auto.
- **قاطع_الدائرة** —_يوقف_بشكل_حتمي_مخاطر_جديدة_في_سلسلة_خسائر،_اختراق_خسارة_
  يومية،_اختراق_هدف_أداء_صعب،
  أو **عدم_توفر_مزود_AI**_(نموذج_متوقف_أو_يهلوس_لا_يفتح_مواقف_جديدة).
- **موافقة_إخلاء_المسؤولية_المؤرشفة** —_موافقة_لمرة_واحدة_ومؤرشفة_مطلوبة_لتسليح_Full Auto_
  (موافقة_قانونية_مطلوبة،_وليس_موافقة_لكل_صفقة)_؛_تحديث_الإخلاء_يفرض_إعادة_الموافقة.
- **مفتاح_الإيقاف_الطارئ** —_إيقاف_طارئ_مُ idempotent_لكل_وكيل_قيد_التشغيل.

## الأهداف

امنح_الوكيل **أهدافاً_قابلة_للقياس** —_مثلاً *الحفاظ_على_أقصى_سحب_أقل_من_4%*،
  *عامل_الربح_على_الأقل_1.5*،
  *معدل_الربح_≥_55%*._كل_هدف_هو **Hard** (حارس_—_مخالفة_تشغل_قاطع_الدائرة)_أو
  **Soft** (يوجه_الاستدلال_فقط)_،_يُقيَّم_على_أنه_On-track_/ _At-risk_/ _Breached_.

## خط_الأنابيب_القراري

بمجرد_البدء،_يعمل_الوكيل_حلقة_**24/7_تحت_الإشراف** (`AgentRuntimeService`)._كل_علامة_،
لكل_حساب_مدار،_يقرأ:_حالة_الحساب_**الحتمية**_(الحقيقة_المؤكدة،_وليس_ذاكرة_النموذج)_؛_
يسأل_محرك_القرار_عن_خطوة؛_يمررها_عبر_بوابة_الأمان_(`AgentDecisionProcessor`)_—
مستوى_الاستقلالية_→_قاطع_الدائرة_→_مظروف_المخاطر_؛_يكتب_سجلاً_قرارياً_إلزامياً_
**`AgentDecisionRecord`**_؛_ويتوقف_أو_ينفذ_كما_توجّه_البوابة._الحلقة_**معزولة_العطل**_
(فشل_وكيل_لا_يمس_آخر_ولا_المضيف)_و**آمنة_افتراضياً**:_خاملة_ما_لم_يتم_تكوين_AI_
*و*_`App:Ai:AgentRuntimeEnabled`_ضُبط،_ولا_تفتح_مخاطر_جديدة_أبداً_بينما_مزود_AI_غير_متوفر.

- **بوابة_الموافقة** —_أمر_الوكيل_المقترح_بوابة_الموافقة_يُسجل_كـ**Pending**_ولا_
  يفعل_شيئاً_حتى_يوافق_المالك_(`POST /api/agent-studio/{id}/decisions/{seq}/approve`_
  أو_`/reject`)_؛ **Full Auto**_يمر_عبر_المظروف_بدون_موافقة_لكل_صفقة؛ **Advisory**_فقط_يقت_بس.
- **دفتر_التدقيق** —_كل_قرار_قابل_لإعادة_العرض:_الاستدلال_(XAI)_،_الدليل_الذي_استشهد_به،
  verdict_البوابة،_نية_الأمر_وما_إذا_نُفذ،_في_`GET /api/agent-studio/{id}/decisions`.
- **مكتب_البحث** —_مناظرة_وكلاء_متعددون_عند_الطلب:_محللو_Alpha/Sentiment/Technical/Risk_
  каждый_даёт_взгляд_و_المراجع_يلخص_اقتراحاً_(`POST /api/agent-studio/{id}/debate`).
- **الذاكرة** —_الوكيل_يتذكر_كل_قرار_و_يستدعي_الذاكرة_الأخيرة_في_الموجه_التالي_
  للاستمرارية_(`GET /api/agent-studio/{id}/memory`).

كل_صف_في_القائمة **تفاصيل**_تفتح_تدفق_قرارات_الوكيل_(مع_موافقة/رفض_على_
الأوامر_المعلقة)_،
ذاكرة_الوكيل،_وعلامة_تشغيل_المناظرة.

## الحسابات المُدارة والتحرير

عند إنشاء وكيل، تختار الحسابات المصرفية التي يديرها — **يُطلب على الأقل واحد عند الإنشاء** (يكون زر *إنشاء* معطلاً حتى يتم تحديد واحد، وتطلب إنشاء نقطة النهاية رفض تحديد فارغ). يمكن **تحرير** كل وكيل بعد ذلك (الاسم والمزاج والاستقلالية والحسابات المُدارة) من أيقونة قلم على صف القائمة. ضوابط دورة الحياة (التفاصيل والتحرير والبدء والإيقاف والقتل) هي أزرار أيقونة، كل واحد معطل في الحالات التي لا ينطبق فيها الإجراء.
