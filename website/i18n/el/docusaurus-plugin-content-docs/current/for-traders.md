# cMind για cTrader έμπορας 📈

Ήδη έμπορο σε cTrader. Ήδη juggle ένα κώδικα επεξεργαστής, ένα backtester, ένα VPS, και τρεις
πρόγραμμα περιήγησης καρτέλες. **cMind κατέρρευσε όλα αυτά σε ένα σκοτεινό, πληκτρολόγιο-φιλικό κονσόλα είστε τρέχει
ο εαυτός σας** — και είναι ανοιχτή πηγή, έτσι τίποτα για το άκρο σας, στρατηγικές σας, ή διαπιστευτήρια
ποτέ φύλλα η κιβώτιο σας.

:::tip TL;DR
Αυτο-ξενιάγηση cMind σε ένα φορητό, μια φτηνή VPS, ή ένα σπίτι διακομιστή. Συγγραφέας, backtest, τρέχουν, και παρακολουθούν cBots
σε ένα μέρος, με ένα AI πυρήνα κάνοντας χρέη. → [Τρέχουν αυτό σε 5 λεπτά](./deployment/local.md)
:::

## Γιατί αυτο-ξενιάγηση αντί ένα φιλοξενείες υπηρεσία;

- **Κατέχει τη στοίβα και τα δεδομένα σας.** Τα cBots σας, διαπιστευτήρια, tokens, και ίδια κεφάλαια ιστορία ζουν σε
  **δικό σας** υποδομή — χωρίς τρίτη κόμμα, χωρίς lock-in, χωρίς "είμαστε sunsetting αυτό προϊόν" email.
- **Είναι γνησίως δικό σας για αλλαγή.** C# 14 / .NET 10, αυστηρό DDD, EF Core + PostgreSQL, ένα MCP
  διακομιστή — όλα ανοιχτή πηγή και hackable. Διαχωρισμό αυτό, επέκταση αυτό, στείλτε ένα PR.
- **Χωρίς ανά-δυνατότητα paywall.** Φέρνουν τη δικός σας AI κλειδί για κάθε παροχέα. κάθε AI δυνατότητα είναι σε.

Προτιμώ δεν να τρέξουν διακομιστή ο εαυτός σας; Μια φιλοξενείες εταιρεία μπορώ να τρέξουν ένα διαχειριζόμενο cMind για σας —
δείτε [Για σύννεφο & VPS προμηθευτές](./for-cloud-providers.md).

## Ένα κονσόλα, χωρίς καρτέλα-juggling

- **Συγγραφέας** σε ένα πραγματικό Monaco IDE (ο VS Code επεξεργαστής), με C# **και** Python πρότυπα και
  sandboxed `dotnet κτίριο` σε φορητό κοντέινερς. → [Κτίριο & backtest](./features/build-and-backtest.md)
- **Backtest** διαμέσου ένα στόλο κόμβι και παρακολουθήστε ίδια κεφάλαια καμπύλες ρεύμα πίσω ζωντανό.
- **Τρέχουν** στρατηγικές ζωντανό και **παρακολουθούν** τους από ένα πίνακας ελέγχου. → [Πίνακας ελέγχου](./features/dashboard.md)
- **Αντίγραφο** ένα κύριο λογαριασμό σε πολλά λογαριασμούς διαμέσου κομίστρες και cTrader αναγνωριστικά, με συμφωνία
  που επιζεί πηγαίνει συνδέσεις και περιστροφή tokens. → [Αντιγραφή διαπραγμάτευσης](./features/copy-trading.md)

## AI που κάνει χρέη, δεν μικρό ομιλία

Φέρνουν δικό σας API κλειδί (κάθε υποστηριζόμενη παροχέα — σύννεφο ή ένα τοπικό μοντέλο) και παίρνουν απλό-ανάλυση → ένα
πραγματικό compiling cBot με αυτο-επιδιόρθωση βρόχο, παράμετρο tuning, backtest post-θνήσιμα, και ένα κίνδυνο
φρουρό που μπορώ να αυτόματη-σταματά ένα δυσπειθής bot. → [Γνωρίστε το AI πυρήνα](./features/ai.md)

## Τρέχει όπου κάνετε

Ξεκίνηση σε ένα φορητό με `κομψό docker δημιουργία`, αποφοίτηση σε ένα φτηνή VPS ή ένα σπίτι διακομιστή όταν είστε
έτοιμος, και ελέγχος σε δοκιμές έμπορα από το τηλέφωνο — cMind είναι ένα εγκαταστάσιμο, κινητή-πρώτα
[PWA](./features/pwa.md). → [Τρέχουν αυτό τοπικά](./deployment/local.md)

Θέλετε τη δική σας AI πελάτη να τη διευθύνει; Υπάρχει ένα ενσωματωμένο [MCP διακομιστή](./features/mcp.md).

## Βοήθεια κάνουν αυτό καλύτερα

cMind είναι ανοιχτή πηγή και MIT-αδέσμευτος — ο χάρτης είναι κοινότητα-σχήμα:

- Αρχείο ζητήματα και δυνατότητα αιτήσεις, και ψήφο σε τι σημαίνει.
- Προσθήκη cBot πρότυπα, AI παροχέα προσαρμογές, ή UI μεταφράσεις.
- Αποστολή PRs — τρεις δοκιμή σειρών (ενότητα + ολοκλήρωση + E2E) και αυστηρό DDD κρατήστε τη μπάρα υψηλή, και το
  [Συμβάλλω οδηγό](./contributing.md) παραδίδει σας μέσα αυτό.
