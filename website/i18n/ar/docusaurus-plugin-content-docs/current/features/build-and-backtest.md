# إنشاء واختبار_backtest_لـ_cBots

إنشاء وتشغيل واختبار_backtest_لـ_cTrader_ cBots_(C#_و_Python،_كلاهما_.NET)_من_محرر_Monaco_في_المتصفح،_يعمل_على_صورة_`ghcr.io/spotware/ctrader-console`_الرسمية.

## الإنشاء

- صفحة **Builder** تستضيف محرر_Monaco؛ يقوم `CBotBuilder` بتجميع المشروع باستخدام
  `dotnet build` **في_حاوية_ذات_استخدام_واحد** (`AppOptions.BuildImage`،_عمل_المجلد_ bind-mount
  في_`/work`)،_لذا_لا_تصل_أهداف_MSBuild_غير_الموثوقة_إلى_المضيف._NuGet_مخ细_
  عبر_البناءات_عبر_مجلد_مشارك._مضيف_الويب_يحتاج_إلى_وصول_إلى_مقبس_Docker.
- قوالب_البدء_لـ_C#_و_Python_موجودة_في_`src/Nodes/Builder/Templates/`.

## التشغيل_واختبار_backtest

- **Instances** =_تسلسل_هرمي_للحالة_TPH_(`Run`/`Backtest` × `Pending`/`Scheduled`/`Starting`/
  `Running`/`Stopping`/`Stopped`/`Failed`)._الانتقال_يستبدل_الكيان_(تغيير_المعرف)،
  مع_الحفاظ_على_معرف_الحاوية.
- `NodeScheduler`_يختار_أقل_العقد_ تحميلاً_المؤهلة؛_`ContainerDispatcherFactory`_توجه_إلى
  وكيل_العقدة_البعيدة_عبر_HTTP_أو_مرسل_Docker_المحلي.
- مزامنة_إكمال_الاستطلاع_تؤكد_الحاويات_الخارجة_(حاويات_backtest_تنهي_ذاتها_عبر_
  `--exit-on-stop`)؛_التقرير_موجود_→_مكتمل_(يخزن_`ReportJson`)_،_مفقود_→_فاشل.
- سجلات_الحاوية_الحية_تتدفق_إلى_المتصفح_عبر_SignalR؛_منحنيات_حقبة_backtest_تُحلل_من_
  التقرير_وتُعرض_بيانياً.

## التشغيل من محرر الكود

يؤدي النقر على **تشغيل** في محرر الكود إلى فتح مربع حوار بدلاً من بدء تشغيل أعمى ومُبرمَج بشكل ثابت:

- **حساب التداول** (مطلوب) — حساب cTrader الذي يتصل به cBot.
- **مجموعة المعلمات** (اختياري) — اختر مجموعة موجودة، أو اتركها فارغة للتشغيل بـ**قيم المعلمات الافتراضية** الخاصة بـ cBot. زر **+** بجوار المُحدِّد يُنشئ مجموعة معلمات جديدة مباشرةً (انظر أدناه) ويختارها.
- **الرمز / الإطار الزمني** افتراضيًا `EURUSD` / `h1` ويمكن تغييرهما؛ **إلغاء** أو **تشغيل**.

عند **التشغيل**، يحفظ المحرر الكود المصدري الحالي ويبنيه، ويبدأ المثيل على الحساب المختار بالمعلمات المختارة، ثم يتتبع سجلات الحاوية المباشرة. (يُمرِّر تدفق السجلات ملف تعريف ارتباط المصادقة الخاص بالمستخدم المسجَّل إلى مركز SignalR `/hubs/logs`، لذا يتصل بدلاً من الفشل بـ `Invalid negotiation response received`.)

## مجموعات المعلمات

**مجموعة المعلمات** هي مجموعة مُسمّاة وقابلة لإعادة الاستخدام من تجاوزات معلمات cBot، مُخزَّنة ككائن JSON مسطَّح يُطابِق كل اسم معلمة بقيمة عددية، مثل `{"Period": 14, "Label": "trend"}`. عند التشغيل/الاختبار الرجعي تُحوَّل إلى ملف cTrader `params.cbotset` (`{ "Parameters": { … } }`). يمكنك إنشاء/تحرير مجموعة كـ JSON خام من مربع حوار **مجموعات المعلمات** الخاص بـ cBot أو مباشرةً من مربع حوار التشغيل.

يتم **التحقق** من JSON عند الحفظ: يجب أن يكون كائنًا مسطَّحًا واحدًا جميع قيمه عددية (سلسلة / رقم / منطقية). يُرفَض الجذر غير الكائن، أو المصفوفة، أو الكائن المتداخل، أو القيمة `null`، أو JSON المُشوَّه (خطأ واضح في مربع الحوار، `400 Bad Request` في واجهة برمجة التطبيقات). الكائن الفارغ `{}` مسموح ويعني "بدون تجاوزات".
