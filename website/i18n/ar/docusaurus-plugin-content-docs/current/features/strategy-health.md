# صحة_الاستراتيجية_و_انحلال_Alpha

كل_حافة_تنحل_—_البحث_صريح_بأن_نصف_حياة_استراتيجية_كمية_انهار_من_سنوات_
إلى_أشهر،_لذا *الاستفادة_تتفوق_الاكتشاف*._مراقب_صحة_الاستراتيجية_يخبرك،
من_تاريخ_عوائد_الاستراتيجية_نفسها،_ما_إذا_كانت_الحافة_لا_تزال_موجودة.

افتح **cBots → Strategy Health** (`/quant/health`).

## ماذا_يفعل

بالنظر_إلى_سلسلة_عوائد_(أو_منحنى_الحققة_،_الأقدم_أولاً)_،_إنه:

- يقسم_التاريخ_إلى_ **أقدم** و **أحدث** نصفي_ويقارن_نسب_Sharpe_الخاصة_بهما؛
- يشغل_ **مسح_نقطة_التغيير_CUSUM** _لتحديد_المراقبة_التي_انتقل_فيها_المتوسط_
  بأوضح_شكل_(انقطاع_النظام)_،_يُبلغ_فقط_عند_كون_الانحراف_إحصائياً_ملحوظاً؛
- يرجع_حكماً:

| الحكم | المعنى |
|---|---|
| **Healthy** | الأداء_الأخير_متوافق_مع_(أو_أفضل_من)_السجل_الأقدم. |
| **Degrading** | Sharpe_الأخيرة_أضعف_بشكل_جوهري_من_السجل_الأقدم_—_راقب_عن_كثب. |
| **Decayed** | الحافة_اختفت_بفعالية_في_النافذة_الأخيرة_—_فكر_في_الإيقاف. |
| **Unknown** | لا_يكفي_تاريخ_للفصل. |

