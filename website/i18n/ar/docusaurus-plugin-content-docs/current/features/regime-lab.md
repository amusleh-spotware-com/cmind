## ماذا_يفعل

بالنظر_إلى_سلسلة_عوائد_(أو_منحنى_الحققة_،_الأقدم_أولاً)_،_إنه:

- يحسب **التقلب_المحقق_المتتبع** _عند_كل_نقطة_ويقسّم_التاريخ_إلى_
  **Calm**،
  **Normal** و**Turbulent** _أنظمة_حسب_أثلاث_ذلك_التقلب؛
- يُبلغ **الأداء_لكل_نظام** —_المشاهدات_،_متوسط_العائد_،_التقلب_و_Sharpe_—
  لترا_أين_تعيش_الحافة؛
- يقدر **أس_ Hurst** _عبر_تحليل_المدى_المُعيد_التقسيم_(R/S):_فوق_~0.55_السلسلة_
  **متوجهة_/_مستدامة**،
  abaixo_~0.45_هي **عائدة_للوسط**،_وحول_0.5_هي_قريبة_من_المشي_العشوائي.
