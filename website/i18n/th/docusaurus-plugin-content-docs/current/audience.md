# cMind สำหรับใคร

เวอร์ชันสั้น: **ผู้คนที่มีเงินในตำแหน่งและต้องการควบคุม** cMind เป็นคอนโซล algorithmic-trading operations ที่เป็นไปตามหลัก - ไม่ใช่เครื่องมือ fintech ของผู้บริโภคที่ทำให้เกิดแอนิเมชันเพื่อเฉลิมฉลองเมื่อคุณทำเงิน $3 ไม่ว่าคุณจะเป็นผู้ค้า โบรกเกอร์ หรือผู้ให้บริการ มีเส้นทางด้านล่างสำหรับคุณ

## 📈 Traders — self-host desk ทั้งหมดของคุณ

คุณค้าบน cTrader และคุณเบื่อกับการนำ editor code backtester VPS และแท็บเบราว์เซอร์สามแท็บมารวมกัน Self-host cMind แล้วรับ authoring backtesting live execution copy trading และ monitoring ในคอนโซลที่ขับเคลื่อนด้วย AI เดียว - บนกล่องของคุณเอง ข้อมูลของคุณจะไม่ออกจากนั่น

**อ่าน:** [cMind สำหรับผู้ค้า cTrader →](./for-traders.md)

## 🏦 Brokers — white-label มันสำหรับลูกค้าของคุณ

คุณบริหารจัดการ brokerage cTrader และต้องการข้อความได้เปรียบเหนือคู่แข่งที่ไม่นำเสนออะไรนอก terminal cMind ส่ง cMind เป็นผลิตภัณฑ์ของคุณเอง: ให้ลูกค้า AI copy trading และ prop-firm challenges ภายใต้ชื่อแบรนด์ของคุณ จำกัด accounts ให้อยู่ในหนังสือของคุณ และเปิด revenue lines ใหม่

**อ่าน:** [cMind สำหรับโบรกเกอร์ cTrader →](./for-brokers.md) ·
[White-label สำหรับธุรกิจ →](./white-label-for-business.md)

## 🖥️ Cloud & VPS providers — ขายมันเป็น managed hosting

คุณให้เช่าคอมพิวต์ นำเสนอ managed cMind hosting และคุณ land high-value compute-hungry sticky workload — และ monetize subscription metered compute white-label และ AI

**อ่าน:** [cMind สำหรับผู้ให้บริการคลาวด์ & VPS →](./for-cloud-providers.md)

## 🧑‍💻 Contributors — ช่วยทำให้ดีขึ้น

cMind เป็น open source (MIT) และ community-shaped: C# 14 / .NET 10 strict DDD EF Core + PostgreSQL .NET Aspire MCP server และ three test tiers ทุกบุคลิกภาพด้านบนสามารถ - และควร - ผลักดันมันไปข้างหน้า:

- **Traders:** เพิ่มเทมเพลต cBot ขอคุณสมบัติ รายงานข้อบกพร่อง
- **Brokers:** มีส่วนร่วมในการรวม integrations และควบคุม desk ที่คุณต้องการ
- **Providers:** upstream การแก้ไขการทำงาน IaC และการปรับปรุง

ไฟล์ issues เพิ่มการแปล UI หรือ AI provider adapters และส่ง PRs — [Contributing guide](./contributing.md) และ [MCP server docs](./features/mcp.md) ได้รับคุณไป

## cMind *ไม่* สำหรับคุณหรือ

ตรวจสอบตัวเองอย่างซื่อสัตย์ cMind อาจมากเกินไปถ้า:

- คุณไม่เคยสัมผัส terminal และไม่ต้องการเริ่มต้น — แม้ว่า [cloud/VPS provider สามารถ host มันให้คุณ](./for-cloud-providers.md)
- คุณเพียงแค่ค้าด้วยตนเองและไม่สนใจเกี่ยวกับ bots backtests หรือ copy trading
- คุณต้องการ hosted SaaS ที่มี support hotline และคุณไม่ต้องการให้ใครบุคคลใดเป็น self-hosted — cMind เป็น self-hosted by design (มันพยายามจริงๆเพื่อให้เรียบง่าย)

ยังคงพยักหน้าตรงกันไหม ยินดีต้อนรับ → [รัน locally](./deployment/local.md)
