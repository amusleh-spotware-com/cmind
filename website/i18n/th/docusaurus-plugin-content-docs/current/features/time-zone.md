# เขตเวลา

เวลาทุกค่าที่แอปแสดงจะถูกเรนเดอร์ในเขตเวลาของคุณเอง ไม่ใช่ของเซิร์ฟเวอร์ ตัวเลือกของคุณจะถูกบันทึกในโปรไฟล์และติดตามคุณข้ามอุปกรณ์

ในการเข้าครั้งแรก แอปจะรับเขตเวลาของเบราว์เซอร์โดยอัตโนมัติ คุณเปลี่ยนได้ทุกเมื่อที่ การตั้งค่า → เขตเวลา ค่าเริ่มต้นของการปรับใช้คือตัวเลือก white-label App:Branding:DefaultTimeZone (ค่าเริ่มต้น UTC) เวลาจะถูกจัดเก็บและส่งกลับจาก API เป็น UTC เสมอ — แปลงเฉพาะการแสดงผลเท่านั้น

- ลำดับการกำหนด: เขตเวลาโปรไฟล์ จากนั้นคุกกี้ จากนั้นค่าเริ่มต้นการปรับใช้ จากนั้น UTC
- การตรวจจับทำงานครั้งเดียวและไม่เคยเขียนทับเขตเวลาที่คุณเลือก
- การจัดรูปแบบเป็นไปตามภาษาของคุณ ป้ายกำกับสัมพัทธ์เช่น «2 นาทีที่แล้ว» ไม่ได้รับผลกระทบ
