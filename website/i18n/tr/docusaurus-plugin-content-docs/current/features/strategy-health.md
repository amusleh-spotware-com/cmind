# Strateji Sağlığı Monitörleme

Canlı çalışan bot'ları izleyin — Sharpe, MDD, kazanç faktörü, anomaliler.

## Metrikler

- **Sharpe Oranı**: Risk-düzeltilmiş getiri
- **Maksimum Düşüş**: En büyük cüzdan düşüşü
- **Kazanç Faktörü**: (Kazançlı trades) / (Kaybı trades)
- **Win Rate**: Kazananlar %

## Uyarılar

Eşik aşıldığında:

- Sharpe < 0.5 → Düşük performans uyarısı
- MDD > 20% → Yüksek kayıp uyarısı
- Win Rate < 30% → Kaybedici uyarısı

Pano veya e-posta ile bildir.

## Anomali Tespiti

AI makine öğrenmesi:

- Anormal kazanç düzeni
- Bağlantı sorunları
- Kaymış fırlatma
