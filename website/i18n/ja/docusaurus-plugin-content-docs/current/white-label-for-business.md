# ビジネス向けホワイトラベル cMind 🏢

Prop ファーム、ブローカーデスク、またはコピートレード サービスを運用していますか？cMind は初日から**あなた自身の製品として再販される**ように構築されました。すべてのサーフェス — 名前、ロゴ、ファビコン、色、さらには インストール可能な電話アプリ — あなたのブランドに曲がります。あなたの顧客は*あなたの*会社を見ます。コード変更、フォークなし、設定のみ。

:::tip TL;DR
`App:Branding` をあなたの名前、色、ロゴを指してください。再起動してください。完了。完全な技術リファレンスは [ホワイトラベル機能ドキュメント](./features/white-label.md)に存在します。
:::

## 再ブランド化できるもの

| サーフェス | 何が変わる |
|---|---|
| **製品名** | アプリバーテキスト + ブラウザタブタイトル |
| **ロゴ & ファビコン** | ブラウザタブを含むあなたのマーク |
| **色** | 完全なパレット — プライマリ、サーフェス、ステータス色 — 全体 UI *と* アプリ自身の CSS を通じてデザイントークン経由で流れます |
| **インストール可能なアプリ（PWA）** | ホーム画面追加名、アイコン、スプラッシュはあなたのブランドを使用 |
| **メタ / SEO** | 説明とサポート URL はあなたのもの |
| **カスタム CSS** | 最後の 5% のポーランドのための独自の注入 |

すべてがストック cMind アイデンティティにデフォルト設定されるため、気になるものだけをオーバーライドします。

## 60 秒の再ブランド化

デプロイメントにこれらを設定します（JSON 設定または環境変数）:

```json
{
  "App": {
    "Branding": {
      "ProductName": "AcmeFX",
      "CompanyName": "Acme Markets Ltd",
      "SupportUrl": "https://support.acme.example",
      "LogoUrl": "/branding/acme-logo.svg",
      "FaviconUrl": "/branding/acme.ico",
      "PrimaryColor": "#2D7FF9",
      "SecondaryColor": "#1E63C8",
      "ShowSiteLink": false
    }
  }
}
```

環境変数フォーム: `App__Branding__ProductName=AcmeFX`。色はスタートアップで検証されます —
不正な 16 進値はクリアなメッセージでブートに失敗する代わりに壊れたページをレンダリングします。良くして
大きく、ちょうど あなたが望むとき。

## 「Powered by cMind」リンク

**デフォルトで**、ダッシュボードは小さな、趣味の良い **「Powered by cMind」** リンクを表示します。このサイトを指しています。デフォルトで有効なのは、プロジェクトを誇りに思っていて、訪問者が見つけるのを助けるからです — しかし、**あなたの呼び出し**です。

- **それを保ちます**（デフォルト）: ダッシュボード上の微妙なクレジットリンク。コスト なし、プロジェクトを支援します。
- **それを隠します**: `App__Branding__ShowSiteLink=false` を設定すると完全に消えます — 製品が紛れもなく*あなたの*完全にホワイトラベル化されたデプロイメントのための完璧です。

[ホワイトラベル機能ドキュメント](./features/white-label.md#powered-by-link)でそれがレンダリングされる場所を正確に参照してください。

## マルチテナント、顧客ごとのブランディング

ブランディングは単なるデプロイメント設定なので、各テナントデプロイメントは独自のアイデンティティを実行できます。顧客あたり 1 つの別々のインスタンスを実行するか、独自の制御プレーンからブランディングを駆動してください — アプリは `IOptionsMonitor` から読み取るため、オプションが変更されるときにテーマさえライブで再構築できます。

これをペアリング:

- **[機能トグル](./features/feature-toggles.md)** — 各テナントが見る機能を決定します。
- **[Prop-firm ルール](./features/prop-firm.md)** — ライブエクイティトレッキングでチャレンジルールを実施します。
- **[パフォーマンス手数料](./features/copy-performance-fees.md)** + **[プロバイダーマーケットプレイス](./features/copy-provider-marketplace.md)** — コピートレードを現金化します。
- **[コンプライアンス](./features/compliance.md)** — あなたの規制当局が尋ねるであろう監査証跡を保ちます。

[Kubernetes](./deployment/kubernetes.md)、[Azure](./deployment/cloud-azure.md)、または
