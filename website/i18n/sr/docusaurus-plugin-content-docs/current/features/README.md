# Функције — пунина турнеја 🧭

Добро дошао на велику турнеју. cMind паковки *много* у једну апликацију, тако ево је мапа. Свака способност
има своју дубоко-нурење док — кликни кроз шта год ти свраб grebanje.

## 🔁 Копирање трговања

Круна драгуља. Огледај главни налог на много, и задржи их синхронизирана чак и када интернет
зла.

- **[Копирање трговања](./copy-trading.md)** — језгро: огледање, врсте налога, SL/TP, скорњаж, desync/resync.
- **[Извршена транспарентност](./copy-execution-transparency.md)** — види управо шта је копирано, када, и зашто.
- **[Перформансне таксе](./copy-performance-fees.md)** — наплати своју сигнал, високо-водени-znamenka стил.
- **[Добављач маркетплатформе](./copy-provider-marketplace.md)** — дозволи трговцима да открију и слече добављаче.
- **[Обавештења](./copy-notifications.md)** — бити рече када нешто те требе.
- **[AI копирање препоручивач](./ai-copy-recommender.md)** — дозволи AI предложи ко копирај.
- **[Open API жетон животни циклус](./token-lifecycle.md)** — како cMind задржава управо један валидан жетон по cID.

## 📊 Твој домаћин база

- **[Контролна плоча](./dashboard.md)** — живо, мобилна-прва командни центру: KPI са sparklines, активност шарта, статус прстен, живо хранити, и (за админе) кластер здравља. То себе освежава.

## 🧠 AI језгро

Не кутија разговора причвршћена на страну — AI која заиста *ради посла*.

- **[AI асистент, агент, заштита ризика & оповештења](./ai.md)** — генерисање стратегије, самоправљајуће градње, позадина заштита ризика која може аутоматски зауставити ботове, и паметне оповештења.

## 🛠️ Грађење & покрени

- **[Грађење & тестирање cBots](./build-and-backtest.md)** — у-прегледач Monaco IDE, C#/Python шаблони, песничке градње, и живе криве акција.
- **[MCP сервер](./mcp.md)** — експонирај cMind алати над HTTP + SSE тако AI клијенти могу га покренути.

## 🏢 Покрени као бизнис

- **[Белоетиетна / брендирање](./white-label.md)** — прерадити свака површина преко конфигурације.
- **[Prop-firm изазов симулација](./prop-firm.md)** — примени дневна-губитак, drawdown, и циљној правила са живим акцијама.
- **[Функција приклјући](./feature-toggles.md)** — одлучи шта сваки развој/тенант види.
- **[Усклађивање / правна](./compliance.md)** — вођење трага и правна површина.

## 📱 Искуство
