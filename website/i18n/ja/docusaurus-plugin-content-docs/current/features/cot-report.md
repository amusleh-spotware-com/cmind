# Commitment of Traders (COT)

cMindは組み込みの**Commitment of Traders**レポート — 米国先物市場での買い建て・売り建ての週次CFTC内訳（商業的ヘッジャー、大型スペキュレーター、ファンド）、インタラクティブな履歴チャート、正規化された**COTインデックス**、cBot向けの認証されたREST API、およびAIクライアント向けのMCPツールを提供します。データは**CFTC公開Socrataデータセット**から直接取得されます — APIキーなし、集約者なし。経済カレンダーと同様、これはデカップルされたモジュールで、トレーディングコアに影響を与えることなく無効にできます。

## What it gives you

- **3つのレポート形式すべて、先物のみと先物+オプション結合:**
  - **Legacy** — 非商業（大型スペキュレーター）、商業（ヘッジャー）、報告不可能。
  - **Disaggregated** — 生産者/マーチャント、スワップディーラー、マネージドマネー、その他報告対象。
  - **Traders in Financial Futures (TFF)** — ディーラー、資産マネージャー、レバレッジファンド、その他報告対象。
- **キュレーションされた市場カタログ** — FXメジャー、金/銀/銅、原油・天然ガス、国債、株価指数、暗号通貨、および主要な穀物/ソフトコモディティ — 各々が安定したCFTC契約コードにマップされ、明確な場合は取引可能なシンボルにマップされます（例：Euro FX → `EURUSD`、Gold → `XAUUSD`）。
- **COTインデックス（0–100）** — 現在のスペキュレーター純ポジションが履歴範囲内のどこに位置するか（デフォルト~3年のルックバック）。極値に近い読み値は、しばしば反転前となる混雑したポジショニングをフラグします；レポートは**長期極値**（≥80）または**短期極値**（≤20）とラベル付けします。
- **特定時点の正確性。** 週次レポートは火曜日に測定されますが、翌金曜日にのみ公開になります；すべての読み取りがそのリリース瞬間を尊重するため、バックテストされたポジショニング信号は公開前にレポートを見ることはありません（ルックアヘッドなし）。

## Using the page

左ナビゲーションから**Commitment of Traders**を開きます。**市場**と**レポートタイプ**（Legacy /
Disaggregated / Financial）を選択し、**Futures + options**を切り替えて先物のみと結合版の間で切り替えます。ページは以下を表示します：

- **時系列での純ポジショニング** — 履歴ウィンドウ全体の各トレーダーカテゴリの純ポジション（買い−売り）のインタラクティブ折れ線グラフ。
- **COTインデックス** — 0–100インデックスの折れ線グラフ、最新の読み値とその極値ラベル付き。
- **最新スナップショット** — トレーダーカテゴリごとの買い/売り/純/建玉%の表、プラス総建玉とレポート日付。

## How the data flows

週次インジェスションワーカーは追跡対象市場の6つのCFTCデータセットを抽出し、市場カタログをアップサート、各新規レポートを**べき等的に**追加します（再実行はスナップショットを複製しません）。最初の実行は数年の履歴をバックフィルします；後続の実行は最新数週間を再同期して遅延改訂をキャッチします。
すべてはキーなしでboxで実行されます；オプションのSocrataアプリトークンはレート制限を上げるだけです。

## Configuration

すべてのキーは`App:Cot`の下に存在します（[feature toggles](./feature-toggles.md)および
[white-label owner settings](./white-label-owner-settings.md)を参照）：

| Key | Default | Purpose |
|-----|---------|---------|
| `IngestionEnabled` | `true` | 週次インジェスションワーカーが実行されるかどうか。 |
| `PollInterval` | `6h` | ワーカーがCFTCデータセットをポーリングする頻度。 |
| `BackfillYears` | `5` | 最初の実行時に抽出される履歴の年数。 |
| `ReconcileLookbackWeeks` | `4` | 各サイクルで改訂をキャッチするために再同期される最新週数。 |
| `SocrataAppToken` | — | 匿名レート制限を上げるオプショナルトークン。 |
| `CotIndexLookbackWeeks` | `156` | COTインデックス範囲として使用される週次レポート（~3年）。 |

## Gating

可視性は経済カレンダーと同じ、2段階ゲート：ホワイトラベルハードゲート
`App:Branding:EnableCot`（ビルドレベル）**および**ランタイム機能トグル`App:Features:Cot`。どちらかオフの場合、ナビゲーションリンク、ページ、REST API、MCPツールはすべて消えます（APIは`404`を返します）。データソースはキーレスであるため、データソースキーゲートはありません — 有効は表示可能を意味します。

## For developers

- Domain: `Core.Cot` — `CotMarket`および`CotReport`集約、`CotPositions`値オブジェクト、
  `CotIndexCalculator`ドメインサービス、および`ICotReports` / `ICotSource`ポート。
- Infrastructure: `Infrastructure.Cot` — `CftcSocrataSource`アンチコラプション解析、レートゲート、
  追記専用書き込みサービス、読み取り側、および週次インジェスションワーカー（EFスキーマ`cot`）。
- cBot & AIアクセス：[COT cBot API](./cot-cbot-api.md)（REST、`market:read` JWT）およびMCPツール
  `CotMarkets`、`CotLatest`、`CotHistory`、`CotHealth`。
